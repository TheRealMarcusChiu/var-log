# Human and Animal Language

- both animals and humans use <strong>signals</strong> (sounds and gestures) to communicate
- animals communicate with:
  - odor - <em>mice</em>
  - complex gestures - <em>bees' waggle-dance [video](https://www.youtube.com/watch?v=LU_KD1enR3Q)</em>
  - songs - <em>used by birds and whales</em>
  - lexigram - a symbol representing a word/lexeme - <em>used by chimpanzees and baboons</em>
- human language is unique because it is:
  - <strong>recursive</strong> - used to describe a language with units (such as sentences or phrases) that can contain themselves (such as sentences within sentences or phrases within phrases).
  - <strong>generative</strong> - used to describe a language that can convey an infinite number of ideas based on different combinations of words or symbols.
  - <strong>displacement</strong> - the ability of a language to describe things that are not present
