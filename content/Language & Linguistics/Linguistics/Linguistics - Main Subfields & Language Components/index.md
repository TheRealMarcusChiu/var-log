# <strong>Language Components</strong>

every language contains a <em>lexicon</em>, a <em>set of phonemes</em>, and a <em>grammar</em> that acts on these sets
- <strong>lexicon/vocabulary</strong> - the sum total of all the lexemes (all possible combinations of word-stems and [[Morphology (Word Forms／Inflexions)|affixes]] in a language)
	- <strong>lexeme</strong> - is the set of inflected forms of a single lemma ([[Morphology (Word Forms／Inflexions)|inflectional morphemes]] applied to the single word) a set of lexical forms having the same stem, the same part of speech, and the same word sense
	- <strong>lexeme (word sense)</strong> - refers to a particular meaning of a particular lexeme
	- <strong>lemma</strong> or <strong>word stem</strong>
	- lexeme vs lemma example:
		- run, runs, ran and running are [[Morphology (Word Forms／Inflexions)|inflectional forms]] of the same lemma run
		- the word runner (a derived term—it has a [[Morphology (Word Forms／Inflexions)|derivational morpheme]] attached) does not belong to the same lexeme run
	- <strong>set of phonemes</strong> - auditory representations of words
	- <strong>set of graphemes and diacritics</strong> - visual written representations of words
	- <strong>set of signed language</strong> - visual gesture representations of words
- <strong>grammar</strong> - is a set of rules a language obeys for creating words and sentences (i.e. morphology and syntax)
