###### Written Language
````excerpt
- <strong>character level</strong>
	- <strong>grapheme (letter/character)</strong> - the smallest meaningful unit in a writing system
	- [[diacritical marks - diacritic|diacritic]] - a glyph added to a grapheme
- <strong>word level</strong>
	- <strong>lexigram</strong> - a symbol representing a word
	- <strong>pictograph </strong>- a pictorial symbol for a word
````
^excerpt
