###### Meter (M)

```excerpt
- is the [[Metric System - International System of Units (SI Units)|SI]] unit of length
- it is defined by taking the fixed numerical value of the speed of light in vacuum 𝑐 to be 299,792,458 when expressed in the unit 𝑚·𝑠<sup>–1</sup>, where the second (𝑠) is defined in terms of the caesium frequency (Δ𝑣<sub>𝐶𝑠</sub>). This definition implies the exact relation c = 299,792,458𝑚·𝑠<sup>–1</sup>
```
