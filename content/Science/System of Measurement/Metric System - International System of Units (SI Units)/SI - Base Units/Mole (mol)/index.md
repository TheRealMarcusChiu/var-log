###### Mole (mol)
````excerpt
- is the [[Metric System - International System of Units (SI Units)|SI]] unit of the amount of substance
- one mole contains exactly 6.02214076x10<sup>23</sup> elementary entities. This number is the fixed numerical value of the Avogadro constant (𝑁<sub>𝐴</sub>) when expressed in the unit 𝑚𝑜𝑙<sup>–1</sup> and is called the Avogadro number
````
