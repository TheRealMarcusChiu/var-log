retrieval is the process of getting information out of memory

ways of retrieval:
- blind retrieval - the process of retrieval willfully
- retrieval cues - are external and/or internal stimuli that helps the process of retrieval
	- associations
	- contextual
	- mood

3 main types of recall:
- free recall
- cued recall
- serial recall

how we recall:
- direct retrieval - information linked directly with cue
- sequential scan - systematic search through entire contents
- hierarchical inference - cue is linked to a subset of information

the brain is able to determine in advance whether there is any point in searching memory for a particular fact
- instantly recognizes the question "what is socrates' phone number?" as absurd

2 main ways of accessing memory:
- recognition
	- example, recognizing face
	- largely unconscious process
- recall
	- example, recalling name of face recognized
	- largely a conscious process
