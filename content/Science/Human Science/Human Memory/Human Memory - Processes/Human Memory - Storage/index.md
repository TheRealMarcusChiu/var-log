# Memory Storage Types
- <strong>sensory memory</strong> - the shortest-term element of memory
	- <strong>echoic memory</strong> - auditory sensory memory
		- phonological loop -
	- <strong>iconic memory</strong> - visual sensory memory
		- visuospatial sketchpad -
	- neocortical sensory and motor buffers -
- <strong>working memory</strong> - considered short-term memory that is active instead of temporary storage system
- <strong>short-term memory</strong> - temporary information storage system that holds about 7 items
	- a method called chunking can help to increase the capacity of short-term memory
- <strong>long-term memory</strong> - can store a seemingly <em>unlimited</em> amount of information almost <em>indefinitely</em>
	- <strong>implicit</strong> - unconsciously recalled
		- <strong>procedural</strong> - skills on how to do things
		- <strong>priming</strong> -
		- <strong>[[Conditioning Behavior (Classical Conditioning & Operant Conditioning)|classical conditioning]]</strong> -
	- <strong>explicit</strong> - consciously recalled
		- <strong>declarative</strong> - facts and events
			- <strong>contextual</strong> or <strong>associative</strong>
				- <strong>episodic</strong> or <strong>autobiographical</strong> -
				- <strong>retrospective</strong> - what happened in the past
				- <strong>prospective</strong> - things to remember in the future
			- <strong>non-contextual </strong>or<strong> non-associative</strong>
				- <strong>semantic</strong> - more structured: abstract and relational
				- <strong>familiarity</strong> -

# Transitions Between Memory Storage Types
- SM can become STM through the process of attention
- STM can become LTM through the process of consolidation:
	- rehearsal
	- meaningful association
	- Motivation
- LTM to STM is a process of [[Human Memory - Retrieval／Recall|retrieval]]
