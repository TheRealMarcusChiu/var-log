<strong>Amino Acids</strong>

all <strong>amino acids</strong> contain:
- a central carbon atom (C)
- an <strong>[[Functional Groups|amino]] </strong>group (NH<sub>2</sub>)
- a <strong>[[Functional Groups|carboxylic acid]] </strong>group (COOH)
- a <strong>side chain </strong>(R)

###### Amino Acid - General Structure
<span style="white-space: pre-wrap"><code>HH O </code><br><code>|| ‖ </code><br><code>N - C - C </code><br><code>|| | </code><br><code>HR O-H</code></span>

###### Amino Acids - Chemical Formula
 R-CH(NH<sub>2</sub>)COOH

the properties of amino acids are determined by the functional substituents linked on the <strong>side chains</strong> (side chains also referred to as <strong>R groups)</strong>

there are 20 standard side chains/R groups/residues found in nature
### Peptide Bond Formation

2 amino acids bond together via dehydration synthesis/condensation reaction
<span style="white-space: pre-wrap"><code>H   H   O        H   H   O          H   H   O       H   O</code><br><code>|   |   ‖ || ‖          |   |   ‖       |   ‖</code><br><code>N - C - C    +   N - C - C     =    N - C - C - N - C - C    +   H2O</code><br><code>|   |   |        |   |   |          |   |       |   |   |</code><br><code>H   R1  O-H      H   R2  O-H        H   R1      H   R2  O-H</code></span>
