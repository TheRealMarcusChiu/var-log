###### Splash

```excerpt
- a real-time fluid simulation implemented using [[JavaScript - WebGPU|WebGPU]]
- this project uses a hybrid MLS-MPM (Moving Least Squares Material Point Method) approach for simulation, allowing for the real-time simulation of approximately 100,000 particles even on integrated graphics
- for rendering, it employs Screen-Space Fluid Rendering combined with the Narrow-Range Filter, which produces high-quality reflections, refractions, and smooth fluid surfaces without constructing explicit meshes
```

^excerpt
