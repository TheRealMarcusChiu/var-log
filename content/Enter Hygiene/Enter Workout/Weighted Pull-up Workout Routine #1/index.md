# Core Principle

two exposures per week:
- heavy neural expression
- mid-range force development

# Progression Rules

progress heavy singles like this:
```
week 1: +115
Week 2: +117
Week 3: +118-119
Week 4: +120
```

When to increase +110 volume.

Once you can do either:
- 4x2 @ +110
- 3x3 @ +110

# Monthly Structure

Weeks 1-3
- run plan as written
- no max testing

Week 4 (Deload / Neural Test)
- one heavy day only
- work up to 1 clean single
- stop as soon as speed drops

# Expected Timeline

If executed cleanly:
- +115-118: 2-4 weeks
- +120: 6-8 weeks
- +125: 3-5 months
- +130: 6-9 months
- +135: 9-18 months

# Things that will kill this plan
- training pull-ups more than 2x a week
- grinding reps
- skipping isometrics
- adding random volume
- testing max weekly
