# Progression

heavy singles progress like

```
Week 1: +105
Week 2: +107
Week 3: +108-109
Week 4: +110
Week 5: +112
Week 6: +115
```

when to raise base weight. once you can do either:

- 4x2 @ +100
- 3x3 @ +100

# Monthly Structure

Weeks 1-3

- run plan exactly
- no max testing
- focus on clean reps and pauses

Week 4 (Neural Test Week)

- Day 1 only
- work up to 1 clean single
- stop when speed drops
