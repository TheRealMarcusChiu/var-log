###### Mathematical Structures
````excerpt
- is a set endowed with some additional features/meaning/significance on the set (e.g. an [[Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms|operation]], [[Finitary Relation (k-ary relation)|relation]], [[Distance Measures／Metrics／Semi-Metrics／Functions|metric]], or [[Topology|topology]])
- a partial list of possible structures are; [[Measure／Measures|measures]], [[Algebraic Structures|algebraic structures]] (groups, fields, etc.), [[Topology|topologies]], metric structures (geometries), orders, events, [[Equivalence Relations|equivalence relations]], differential structures, and categories
- the formalization of mathematical structures is called [[Category Theory]]
````
