###### NP (Non-Deterministic Polynomial Time)
````excerpt
- a type of [[Complexity Classes|complexity class]] that is the set of all [[Decision Problems - Optimization Problems|decision problems]] where correct answers can be VERIFIED in polynomial time by a [[Deterministic Turing Machine (DTM)|deterministic Turing machine]]
- or alternatively the set of problems that can be solved in polynomial time by a [[Non-Deterministic Turing Machine (NTM)|nondeterministic Turing machine]]
````
