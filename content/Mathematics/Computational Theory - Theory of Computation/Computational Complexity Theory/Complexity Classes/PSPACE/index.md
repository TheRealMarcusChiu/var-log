###### PSPACE
````excerpt
- a type of [[Complexity Classes|complexity class]] that is the set of all [[Decision Problems - Optimization Problems|decision problems]] that can be solved by a [[Deterministic Turing Machine (DTM)|deterministic Turing machine]] using a polynomialamount of space/memory
````
