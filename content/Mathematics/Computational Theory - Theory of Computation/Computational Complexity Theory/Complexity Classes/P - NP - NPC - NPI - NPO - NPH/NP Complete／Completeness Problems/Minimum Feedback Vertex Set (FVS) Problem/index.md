###### Minimum Feedback Vertex Set (FVS) Problem
````excerpt
- in [[Graph Theory|graph theory]], a feedback vertex set (FVS) of a graph is a set of vertices whose removal leaves a graph without cycles ("removal" means deleting the vertex and all edges adjacent to it)
	- Equivalently, each FVS contains at least one vertex of any cycle in the graph
- The <strong>minimum feedback vertex set problem</strong> is an [NP-complete](https://en.wikipedia.org/wiki/NP-complete) problem; it was among the [first problems shown to be NP-complete](https://en.wikipedia.org/wiki/Karp%27s_21_NP-complete_problems). It has wide applications in [[Operating Systems (OS)|operating systems]], [[Database Management Systems (DBMS)|database systems]], and [[Very Large-Scale Integration (VLSI)|VLSI]] chip design
````
