###### Turing Completeness/Complete - Computationally Universal
````excerpt
- in [[Computability Theory - Recursion Theory|computability theory]], a system of data-manipulation rules (such as a [[Model of Computation - Computational Model|model of computation]], a computer's [[Instruction Set Architecture (ISA)|instruction set]], a [[Computer／Programming Languages|programming language]], or a cellular automaton) is said to be <strong>Turing-complete</strong> or <strong>computationally universal</strong> if it can be used to simulate any [[Turing Machine (TM)|Turing machine]]
````
^excerpt

# From UT MSDS Online

If it contains the following then it is Turing complete:
- assignment statements
- arithmetic
- conditionals
- loops
