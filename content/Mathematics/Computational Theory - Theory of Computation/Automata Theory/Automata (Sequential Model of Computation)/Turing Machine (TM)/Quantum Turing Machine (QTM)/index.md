A <strong>quantum [[Turing Machine (TM)|Turing machine]]</strong> (<strong>QTM</strong>), also a <strong>universal quantum computer</strong>, is an abstract machine used to model the effect of a quantum computer. It provides a very simple model which captures all of the power of quantum computation. Any quantum algorithm can be expressed formally as a particular quantum Turing machine.
