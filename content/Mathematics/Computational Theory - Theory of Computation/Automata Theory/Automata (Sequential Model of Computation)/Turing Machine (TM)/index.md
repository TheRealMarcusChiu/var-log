###### Turing Machine (TM)
````excerpt
- is a [[Mathematical Model|mathematical model]] of a hypothetical computing machine which can use a predefined set of rules to determine a result from a set of input variables
````
^excerpt

# TM - 4 Main Components
- Infinite length magnetic tape
- Read/Write head
- Rules table / FSM
- State register
