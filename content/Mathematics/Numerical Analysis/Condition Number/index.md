###### Condition Number
````excerpt
- the condition number of a [[Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms|function]] measures how much the output value of the function can change for a small change in the input argument
- is derived from the theory of propagation of uncertainty, and is formally defined as the value of the asymptotic worst-case relative change in output for a relative change in input
````
