###### Level Sets
````excerpt
- a <em>level set</em> of a real-valued function 𝑓 of 𝑛 real variables is a set where the function takes on a given constant value 𝑐:
	- 𝐿<sub>𝑐</sub>(𝑓) = {(𝑥<sub>1</sub>, ..., 𝑥<sub>𝑛</sub>) | 𝑓(𝑥<sub>1</sub>, ..., 𝑥<sub>𝑛</sub>) = 𝑐}
	- <strong>level curve</strong> - <strong>contour line</strong> - <strong>isoline</strong> - is a <em>level set</em> whose function 𝑓 has two independent variables
	- <strong>level surface</strong> - <strong>isosurface</strong> - is a <em>level set</em> whose function 𝑓 has three independent variables
	- <strong>level hypersurface</strong> - is a <em>level set</em> whose function 𝑓 has 4 or more independent variables
- a level set is a special case of a [fiber](https://en.wikipedia.org/wiki/Fiber_(mathematics))
````
^excerpt

# Level Sets - Examples

Consider the 2D <em>[[L2／Euclidean Distance Metric|Euclidean distance metric]]</em>:
- $d(x,y) = \sqrt{x^2 + y^2}$

A level set 𝐿<sub>𝑟</sub>(𝑑) of the <em>Euclidean distance metric</em>, consists of all points that lie at a distance 𝑟 from the origin.

For example:
- (3,4) ∊ 𝐿<sub>5</sub>(𝑑)
