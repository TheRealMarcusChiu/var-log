###### Linear Extension Theorem
````excerpt
- Let:
	- 𝛽 =⟨𝑏<sub>1</sub>, 𝑏<sub>2</sub>, ..., 𝑏<sub>𝑛</sub>⟩be the [[Basis Vectors|basis]] for [[Vector Spaces - Linear Spaces|vector space]] 𝑉
	- {𝑤<sub>1</sub>, 𝑤<sub>2</sub>, ..., 𝑤<sub>𝑛</sub>} be arbitrary vectors in vector space 𝑊
- then there exists a UNIQUE linear transformation 𝑇:𝑉→𝑊 such that 𝑇(𝑣<sub>𝑖</sub>) = 𝑤<sub>𝑖</sub> for all 𝑖
````
^excerpt
