###### Unitary Matrix

a complex [[Square Matrix|square matrix]] 𝑈 is <strong>unitary</strong> if its [[Transpose Matrix - Complex-Conjugate／Hermitian／Adjoint／Transjugate／Bedaggered Transpose Matrix|conjugate transpose]] 𝑈\* is also its inverse, that is, if:

- 𝑈\*𝑈 = 𝑈𝑈\* = 𝐼

where:

- 𝐼 - [[Identity Matrix|identity matrix]]
