###### Diagonal Matrix
````excerpt
- a [[Square Matrix|square matrix]] in which the entries outside the main diagonal are all zero
- an [[Identity Matrix|identity matrix]] of any size, or any multiple of it (a scalar matrix), is a diagonal matrix
````
^excerpt
