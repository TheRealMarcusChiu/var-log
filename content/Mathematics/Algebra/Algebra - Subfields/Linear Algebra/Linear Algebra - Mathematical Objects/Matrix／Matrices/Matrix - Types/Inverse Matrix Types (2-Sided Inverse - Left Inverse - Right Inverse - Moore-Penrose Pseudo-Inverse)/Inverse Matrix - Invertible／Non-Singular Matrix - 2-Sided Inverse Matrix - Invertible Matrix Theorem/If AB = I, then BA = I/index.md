Given 𝐴𝐵 = 𝐼, prove 𝐵𝐴 = 𝐼

###### PROOF

- 𝐵 = 𝐵𝐼
- 𝐵 = 𝐵(𝐴𝐵)
- 𝐵 = (𝐵𝐴)𝐵
- 𝐼𝐵 = (𝐵𝐴)𝐵
- 𝐼𝐵 - (𝐵𝐴)𝐵 = 0
- (𝐼 - 𝐵𝐴)𝐵 = 0

Since 𝐵 is full rank, (𝐼 - 𝐵𝐴) is 0 on all vectors.

This means (𝐼 - 𝐵𝐴) must be the 0 matrix.

So 𝐼 = 𝐵𝐴
