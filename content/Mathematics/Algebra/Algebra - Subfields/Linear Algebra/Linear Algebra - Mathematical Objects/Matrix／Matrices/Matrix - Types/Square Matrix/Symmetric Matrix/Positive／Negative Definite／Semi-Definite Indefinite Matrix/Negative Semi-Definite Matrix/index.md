###### Negative Semi-Definite Matrix

```excerpt
- is a matrix 𝐴 where:
	- 𝐴's [[Eigenvectors (Characteristic Vectors) - Eigenvalues (Spectrum)|eigenvalues]] ≤ 0
- if 𝐴's eigenvalues ≤ 0, then 𝐴 is [[Negative Definite Matrix|negative definite]]
```

^excerpt
