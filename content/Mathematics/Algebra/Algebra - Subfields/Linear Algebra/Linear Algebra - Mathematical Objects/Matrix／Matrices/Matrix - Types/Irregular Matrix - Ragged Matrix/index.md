###### Irregular Matrix - Ragged Matrix

```excerpt
- is a [[Matrix／Matrices|matrix]] that has a different number of elements in each row
- is not used in [[Linear Algebra|linear algebra]], since standard matrix transformations cannot be performed on them, but they are useful in computing as arrays which are called [jagged arrays](https://en.wikipedia.org/wiki/Jagged_array)
- irregular matrices are typically stored using [Iliffe vectors](https://en.wikipedia.org/wiki/Iliffe_vector)
```

^excerpt
