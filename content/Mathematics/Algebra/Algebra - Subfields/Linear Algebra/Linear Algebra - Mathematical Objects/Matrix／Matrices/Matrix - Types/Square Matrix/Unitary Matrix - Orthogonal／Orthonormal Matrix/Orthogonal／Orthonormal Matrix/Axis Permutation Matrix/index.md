###### Axis Permutation Matrix

- is an [[Orthogonal／Orthonormal Matrix|orthogonal matrix]] obtained by permuting the rows of an nxn identity matrix 𝐼 according to some permutation of the numbers 1 to n
- every row and column therefore contains precisely a single 1 with 0s everywhere else
- every permutation corresponds to a unique permutation matrix
- is [[Determinants - Invertible／Non-Singular - Not-Invertible／Singular|nonsingular]]
- its [[Determinants - Invertible／Non-Singular - Not-Invertible／Singular|determinant]] is always +/-1
  - \= 1, when number of row exchanges is even
  - \= 0, when number of row exchanges is odd
- a permutation matrix 𝐴 satisfies: 𝐴<sup>𝑇</sup>𝐴 = 𝐴𝐴<sup>𝑇</sup> = 𝐼
