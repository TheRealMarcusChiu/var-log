###### Hermitian Matrix - Self-Adjoint Matrix

```excerpt
- is a complex [[Square Matrix|square matrix]] that is equal to its [[Transpose Matrix - Complex-Conjugate／Hermitian／Adjoint／Transjugate／Bedaggered Transpose Matrix|conjugate transpose]]—that is, the element in the ith row and jth column is equal to the [[Complex Conjugate／Conjugation|complex conjugate]] of the element in the jth row and ith column, for all indices i and j
```

^excerpt

# Hermitian Matrix - Syntax

Hermitian Matrix of matrix 𝐴 is denoted as 𝐴<sup>H</sup>
