###### Positive Semi-Definite Matrix

```excerpt
- is a matrix 𝐴 where:
	- 𝐴's [[Eigenvectors (Characteristic Vectors) - Eigenvalues (Spectrum)|eigenvalues]] ≥ 0
	- 𝐴's [[Eigenvectors (Characteristic Vectors) - Eigenvalues (Spectrum)|eigenvectors]] are pairwise orthogonal
- if 𝐴's eigenvalues \> 0 strictly greater than zero, then 𝐴 is [[Positive Definite Matrix|positive definite]]
```

^excerpt

see: [[Positive／Negative Definite／Semi-Definite Indefinite Matrix|Positive/Negative Definite/Semi-Definite Indefinite Matrix]]

# Positive Semi-Definite Matrix - Properties

- since at least 1 of 𝐴's eigenvalues is equal to 0, then:
  - 𝐴 is [[Determinants - Invertible／Non-Singular - Not-Invertible／Singular|singular]] (i.e. its [[Determinants - Invertible／Non-Singular - Not-Invertible／Singular|determinant]] is equal to zero)
  - thus 𝐴 is not [[Inverse Matrix - Invertible／Non-Singular Matrix - 2-Sided Inverse Matrix - Invertible Matrix Theorem|invertible]]

# Positive Semi-Definite Matrix - Applications

- used in [[Statistics|statistics]] (e.g. [[Covariance & Correlation Matrix|covariance matrices, correlation matrices]], [[Cross-Product Matrix|cross-product matrices]] are all <em>positive semi-definite matrices</em>) these matrices are also symmetric
