<strong>Orthogonal Eigen-Decomposition/Diagonalization </strong>is the [[Eigen Decomposition／Factorization - Diagonalization／Diagonalizing／Diagonalize - Diagonalizable／Non-Defective Matrix - Non-Diagonalizable／Defective Matrix|eigen-decomposition/diagonalization]] of a [[Symmetric Matrix|symmetric matrix]] 𝐴 which is defined as:

- 𝐴 = 𝑃𝐷𝑃 <sup>-1</sup>= 𝑃𝐷𝑃 <sup>𝑇</sup>

where:

- 𝑃 is a [[Orthogonal／Orthonormal Matrix|square orthonormal matrix]]; columns and rows of 𝑃 are the orthogonal [[Eigenvectors (Characteristic Vectors) - Eigenvalues (Spectrum)|eigenvectors]] of 𝐴 (thus: 𝑃 <sup>𝑇</sup>𝑃 = 𝑃𝑃 <sup>𝑇</sup>= 𝐼)
- 𝐷 is a [[Diagonal Matrix|diagonal matrix]]
