###### Linear Independence - Linearly Independent - Linear Dependence - Linearly Dependent

```excerpt
- a set of [[Vectors|vectors]] is said to be <strong>linearly independent</strong> if there exists NO non-trivial linear combination of the vectors that equals the zero vector
- a set of vectors is said to be <strong>linearly dependent</strong> if there exists a non-trivial linear combination of the vectors that equals the zero vector
```

^excerpt

# Linear Independence - Definition

A set of vectors {𝑣<sub>1</sub>, 𝑣<sub>2</sub>, ..., 𝑣<sub>𝑘</sub>} from a vector space 𝑉 is <strong>linearly independent</strong> if there do NOT exist scalars {𝛼<sub>1</sub>, 𝛼<sub>2</sub>, ..., 𝛼<sub>𝑘</sub>}, not all zero such that:

- 𝛼<sub>1</sub>𝑣<sub>1</sub> + 𝛼<sub>2</sub>𝑣<sub>2</sub> + ... + 𝛼<sub>𝑘</sub>𝑣<sub>𝑘</sub> = zero vector
