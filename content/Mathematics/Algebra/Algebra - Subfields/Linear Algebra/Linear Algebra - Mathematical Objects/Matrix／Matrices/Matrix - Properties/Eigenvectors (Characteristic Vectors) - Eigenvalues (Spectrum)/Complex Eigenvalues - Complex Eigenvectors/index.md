# Other

Let 𝐴 be a matrix with REAL entries. If
- 𝜆 is a <em>complex eigenvalue</em> with <em>complex eigenvector</em> 𝑣, then
- 𝜆̅ is a <em>complex eigenvalue</em> with <em>complex eigenvector</em> 𝑣̅

In other words, both eigenvalues and eigenvectors come in conjugate pairs.
