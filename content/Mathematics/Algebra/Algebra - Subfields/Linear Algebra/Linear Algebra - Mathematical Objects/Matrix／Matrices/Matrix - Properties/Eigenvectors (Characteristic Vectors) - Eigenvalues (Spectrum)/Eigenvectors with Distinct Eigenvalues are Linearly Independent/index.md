# Theorem

Let 𝑣<sub>1</sub>, 𝑣<sub>2</sub>, ..., 𝑣<sub>𝑘</sub> be eigenvectors of a matrix 𝐴, and suppose that the corresponding eigenvalues 𝜆<sub>1</sub>, 𝜆<sub>2</sub>, ..., 𝜆<sub>𝑘</sub> are distinct (all different from each other).

Then {𝑣<sub>1</sub>, 𝑣<sub>2</sub>, ..., 𝑣<sub>𝑘</sub>} is linearly independent
