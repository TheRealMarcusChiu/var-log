###### Orthonormal/Orthonormality

```excerpt
- given an [[Inner Product Spaces|inner product space (𝑉,𝐹,⟨·,·⟩)]]:
	- vectors 𝑣<sub>𝑖</sub>,𝑣<sub>𝑗</sub>∊𝑉 are <em>orthonormal</em> if the [[Inner Products|inner product (⟨·,·⟩)]] of 𝑣<sub>𝑖</sub> and 𝑣<sub>𝑗</sub>either equals:
		- 0 if 𝑖≠𝑗 (i.e. 𝑣<sub>𝑖</sub> and 𝑣<sub>𝑗</sub>are [[Orthogonal／Orthogonality - Orthogonal Sets - Orthogonal Complement|orthogonal]])
		- 1 if 𝑖=𝑗 (i.e. 𝑣<sub>𝑖</sub> has unit length)
- given a [[Vector Spaces - Linear Spaces|vector space (𝑉,𝐹)]] with [[Bilinear Functionals - Bilinear Forms - 2／Two Forms|bilinear form (𝐵)]]:
	- vectors 𝑣<sub>𝑖</sub>,𝑣<sub>𝑗</sub>∊𝑉 are <em>orthonormal</em> if the 𝐵(𝑣<sub>𝑖</sub>,𝑣<sub>𝑗</sub>) either equals:
		- 0 if 𝑖≠𝑗 (i.e. 𝑣<sub>𝑖</sub> and 𝑣<sub>𝑗</sub>are [[Orthogonal／Orthogonality - Orthogonal Sets - Orthogonal Complement|orthogonal]])
		- 1 if 𝑖=𝑗 (i.e. 𝑣<sub>𝑖</sub> has unit length)
```

^excerpt

###### Orthonormal Sets

- a set of vectors form an <em>orthonormal set</em> if all vectors in the set are mutually orthonormal
- an <em>orthonormal set</em> which forms a basis is called an [[Orthonormal Basis Vectors|orthonormal basis]]
- can be represented as an [[Orthogonal／Orthonormal Matrix|orthonormal matrix]]
