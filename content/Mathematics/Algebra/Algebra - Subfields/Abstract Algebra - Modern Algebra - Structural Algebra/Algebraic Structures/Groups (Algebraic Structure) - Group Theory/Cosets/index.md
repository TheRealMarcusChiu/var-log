# Cosets - Examples

> [!expand-ui]- Vectors
> The elements ([[Vectors|vectors]]) of a [[Vector Spaces - Linear Spaces|vector space]] form an [[Abelian Groups (Algebraic Structure)|abelian group]] under [vector addition](https://en.wikipedia.org/wiki/Vector_addition). The [[Linear Subspaces - Vector Subspaces|subspaces]] of the vector space are [subgroups](https://en.wikipedia.org/wiki/Subgroups) of this group. For a vector space 𝑉, a subspace 𝑆, and a fixed vector 𝑣∊𝑉, the set
>
> 1. \[𝑣] = 𝑣 + 𝑆
> 2. \[𝑣] = {𝑣+𝑠 : 𝑠∊𝑆}
> 3. \[𝑣] = {𝑥∈𝑉 : 𝑥=𝑣+𝑠, 𝑠∈𝑆 } <font style="color: rgb(122,134,154);"># all three are equivalent</font>
>
> is called an [[Affine Subsets - Affine Subspaces|affine subspace]], and is a <strong>coset</strong> of 𝑣 often denoted as \[𝑣] (both left and right, since the group is [[Abelian Groups (Algebraic Structure)|abelian]]).
>
> ## Example
>
> If:
>
> - <font style="color: rgb(51,102,255);">𝑆 = { (𝑥,2𝑥)∊ℝ<sup>2</sup> : 𝑥∊ℝ }</font>
> - 𝑣∊𝑉
>
> Then the<font style="color: rgb(255,0,0);"> coset of 𝑣 </font>is parallel to <font style="color: rgb(51,102,255);">𝑆</font>
>
> - <font style="color: rgb(51,102,255);"><font style="color: rgb(255,0,0);">\[𝑣] = { 𝑥∈𝑉 ∣ 𝑥 = 𝑣+𝑠, 𝑠∈𝑆 }</font></font>
>
