# Abelian Groups - Definition

An <em>abelian group</em> is a set 𝐺 with one [binary operation](http://confluence.marcuschiu.com/pages/viewpage.action?pageId=1023131) on 𝐺 satisfying the<em> abelian group</em> axioms (𝑎, 𝑏, 𝑐 are arbitrary elements of the <em>abelian group</em> 𝐺)

A binary operation on 𝐺 is a mapping 𝐺×𝐺 → 𝐺, that is, a [[Binary Relation (Correspondence ／ Dyadic Relation ／ 2-Place Relation)|correspondence]] that associates with each ordered pair of elements of 𝐺 a uniquely determined element of 𝐺.
