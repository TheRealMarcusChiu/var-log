# Definition

A <strong>mixed moment</strong> of order 𝑘 (where 𝑘 = 𝑘<sub>1</sub> + ... + 𝑘<sub>𝑛</sub>) is defined as:
- $E[X_1^{k_1}...X_n^{k_n}]$

A <strong>central mixed moment</strong> of order 𝑘 (where 𝑘 = 𝑘<sub>1</sub> + ... + 𝑘<sub>𝑛</sub>) is defined as:
- $E[(X_1 - E[X_1])^{k_1}...(X_n- E[X_n])^{k_n}]$
