# Associativity/Associative - Definition

An [operation ∗](http://confluence.marcuschiu.com/pages/viewpage.action?pageId=1023111) on a [set 𝑆](http://confluence.marcuschiu.com/display/NOT/Set+Theory) is called <em>associative</em> if:
- (𝑥 [∗](http://confluence.marcuschiu.com/pages/viewpage.action?pageId=1023111) 𝑦) [∗](http://confluence.marcuschiu.com/pages/viewpage.action?pageId=1023111) 𝑧 = 𝑥 [∗](http://confluence.marcuschiu.com/pages/viewpage.action?pageId=1023111) (𝑦 [∗](http://confluence.marcuschiu.com/pages/viewpage.action?pageId=1023111) 𝑧)       for all 𝑥,𝑦,𝑧 ∊ 𝑆

An operation that does not satisfy the above property is called <em>non-associative</em>.
