# Distributivity/Distributive - Definition

Given two [[Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms|operators]] ∗ and + on [[Set Theory|set 𝑆]]

- the operation ∗ is <em>left-distributive</em> over (or with respect to) +;  if 𝑥 ∗ (𝑦 + 𝑧) = (𝑥 ∗ 𝑦) + (𝑥 ∗ 𝑧) for all 𝑥,𝑦,𝑧 ∊ 𝑆
- the operation ∗ is <em>right-distributive</em> over (or with respect to) +; if (𝑦 + 𝑧) ∗ 𝑥 = (𝑦 ∗ 𝑥) + (𝑧 ∗ 𝑥) for all 𝑥,𝑦,𝑧 ∊ 𝑆
- the operation ∗ is <em>distributive</em> over (or with respect to) +; if it is both left-distributive and right-distributive

If ∗ is [[Commutativity／Commutative|commutative]], then the three conditions above are logically equivalent.
