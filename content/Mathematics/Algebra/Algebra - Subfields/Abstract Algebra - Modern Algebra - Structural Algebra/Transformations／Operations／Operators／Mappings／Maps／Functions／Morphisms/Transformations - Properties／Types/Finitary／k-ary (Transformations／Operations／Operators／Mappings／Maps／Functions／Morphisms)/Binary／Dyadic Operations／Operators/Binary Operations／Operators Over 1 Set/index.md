###### Binary Operator Over 1 Set (𝑆×𝑆 → 𝑆 or 𝑆×𝑆×𝑆)
````excerpt
- is a <em>[[Binary／Dyadic Operations／Operators|binary operation]]</em> whose two [domains](http://confluence.marcuschiu.com/pages/viewpage.action?pageId=1022702) and the [codomain](http://confluence.marcuschiu.com/pages/viewpage.action?pageId=1022702) are the same set
- is a <em>binary operation</em> 𝑓 on a set 𝑆 may be viewed as a ternary [relation](http://confluence.marcuschiu.com/pages/viewpage.action?pageId=1022533) on 𝑆, that is, the set of triples (𝑎, 𝑏, 𝑓(𝑎,𝑏)) in 𝑆×𝑆×𝑆 for all 𝑎 and 𝑏 in 𝑆
````
