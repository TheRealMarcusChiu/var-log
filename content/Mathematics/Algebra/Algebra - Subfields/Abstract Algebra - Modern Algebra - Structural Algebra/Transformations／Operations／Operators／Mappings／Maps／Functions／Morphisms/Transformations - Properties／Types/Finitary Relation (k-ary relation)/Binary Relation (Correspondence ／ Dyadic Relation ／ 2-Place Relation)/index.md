###### Binary Relation (Correspondence / Dyadic Relation / 2-Place Relation)
````excerpt
- a binary relation relates 2 [[Mathematical Object|mathematical objects]] together, for example:
	- ≤ is a binary relation: ≤ relates numbers 1≤4
	- ⊆ is a binary relation: ⊆ relates sets {𝑥}⊆{𝑥,𝑦,𝑧}
- a binary relation is a type of [[Binary／Dyadic Operations／Operators|binary operation]] that outputs a boolean value: true or false
	- + is not a binary relation because 1+4 does not result in a boolean value but a scalar value of 5
````
^excerpt
