###### Analytic Functions

```excerpt
- an analytic function is a function that is locally given by a convergent power series
- a function is analytic if and only if its Taylor series about 𝑥<sub>0</sub> converges to the function in some neighborhood for every 𝑥<sub>0</sub> in its domain
```

^excerpt
