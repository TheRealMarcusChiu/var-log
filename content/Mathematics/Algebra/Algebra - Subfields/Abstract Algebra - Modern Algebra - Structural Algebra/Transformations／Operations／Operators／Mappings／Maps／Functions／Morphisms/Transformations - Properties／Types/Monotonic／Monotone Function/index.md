###### Monotonic/Monotonic Function
````excerpt
- is a [[Functions (Domain - Codomain - Preimage - Image - Range)|function]] that is strictly non-decreasing or non-increasing
- is a function between [ordered sets](https://en.wikipedia.org/wiki/List_of_order_structures_in_mathematics) that preserves or reverses the given order
````
