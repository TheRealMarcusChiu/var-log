###### Unit Step Function - Heavyside Step Function

```excerpt
- usually denoted by 𝐻 or 𝜃 (but sometimes 𝑢, 𝟏 or 𝟙)
- the value of which is zero for negative arguments and one for positive arguments
	- $H(t) = \begin{cases} 0 & t < 0 \\ 1 & \text{otherwise} \\ \end{cases}$
```
