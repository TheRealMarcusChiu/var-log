<strong>inverse function</strong> (or <strong>anti-function</strong>) is a [[Functions (Domain - Codomain - Preimage - Image - Range)|function]] that "reverses" another function

<em>𝑓(𝑥)</em> = 𝑦[if and only if](https://en.wikipedia.org/wiki/If_and_only_if)<em>𝑔(𝑦)</em> = <em>𝑥</em>

Not all functions have inverse functions; those which do are called <strong>invertible</strong>

In order for a function <em>𝑓</em>: <em>𝑋</em> → <em>𝑌</em> to have an inverse, it must be [[Function - Injective／Injection／One-to-one - Surjective／Surjection／Unto／Onto - Bijective／Bijection／One-to-one Correspondence|bijective]] (𝑖.𝑒. for every𝑦 in 𝑌 there must be [one, and only one](https://en.wikipedia.org/wiki/Uniqueness_quantification)𝑥 in 𝑋 so that <em>𝑓(𝑥)</em> = 𝑦)
