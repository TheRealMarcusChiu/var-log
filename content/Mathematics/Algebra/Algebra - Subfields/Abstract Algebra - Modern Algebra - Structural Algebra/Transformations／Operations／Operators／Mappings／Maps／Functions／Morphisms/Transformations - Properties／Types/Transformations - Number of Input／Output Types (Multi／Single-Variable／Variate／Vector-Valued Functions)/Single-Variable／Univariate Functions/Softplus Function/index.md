<strong>Softplus Function</strong>

- the name comes from the fact that it is a smoothed or “softened” version of: 𝑥<sup>+</sup> = 𝑚𝑎𝑥(0, 𝑥)
- can be used as an [[Activation Functions (AF)|activation function]]

# Formula

- 𝜁(𝑥) = 𝑙𝑜𝑔(1 + 𝑒𝑥𝑝(𝑥))
