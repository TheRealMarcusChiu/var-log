###### Gamma Function (𝛤)

```excerpt
- is one commonly used extension of the factorial function to [[Numbers Classes／Classification of Numbers|complex numbers]]
- is defined for all complex numbers except the non-positive integers
- for any positive integer 𝑛:
	- 𝐺𝑎𝑚𝑚𝑎(𝑛) = 𝛤(𝑛) = (𝑛 - 1)!
```
