###### Degenerate Bilinear Functionals - Degenerate Bilinear Forms
````excerpt
- is a [[Bilinear Functionals - Bilinear Forms - 2／Two Forms|bilinear functional (𝐵 : 𝑉⨯𝑉→𝐹)]] such that 𝐵 is not an [[Morphisms - Homomorphic／Homomorphisms (Isomorphic／Isomorphisms - Monomorphic／Monomorphisms／Injective - Epimorphic／Epimorphisms／Surjective - Endomorphic／Endomorphisms - Automorphic／Automorphisms)|isomorphism]]
- if 𝑉 is finite-dimensional, then there exists some non-zero 𝑣∊𝑉 such that:
	- 𝐵(𝑣,𝑢) = 0 for all 𝑢∊𝑉
- opposed to [[Non-Degenerate Bilinear Functionals - Non-Degenerate Bilinear Forms|non-degenerate bilinear functionals]]
````
