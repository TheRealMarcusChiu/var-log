###### Kronecker Delta Function/Symbol (𝛿<sub>𝑖𝑗</sub>)
````excerpt
- is a function of two variables that outputs 1 if the variables are equal, and 0 otherwise
	- $𝛿_{ij} = \begin{cases} 1 & if \quad i = j \\ 0 & otherwise \\ \end{cases}$
- for example: 𝛿<sub>12</sub> = 0 and 𝛿<sub>33</sub> = 1
- also denoted as:
	- $𝛿_{ij} = 𝛿^i_j = 𝛿(i,j) = [i = j] = v^i·v_j$
````
^excerpt
