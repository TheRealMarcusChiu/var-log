###### Indicator Function - Characteristic Function - Membership Function
````excerpt
- the <strong>indicator/characteristic/membership function</strong> of a subset 𝑆 of a set 𝑋 is a [[Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms|function]] 1<sub>𝐴</sub>: 𝑋 → {0,1} defined as:
	- $1_A(x) := \begin{cases} 1 & \text{if } x \in A, \\ 0 & \text{if } x \notin A \\ \end{cases}$
````
