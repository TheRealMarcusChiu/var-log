<strong>Multivariate Beta Function</strong> is a generalization of the [[Beta Function]]
- 𝐵(𝛼<sub>1</sub>, ..., 𝛼<sub>𝑘</sub>) = \[𝛱<sub>1≤𝑖≤𝑘</sub>𝛤(𝛼<sub>𝑖</sub>)\] / 𝛤(𝛴<sub>1≤𝑖≤𝑘</sub>𝛼<sub>𝑖</sub>)

where:
- 𝛤 - [[Gamma Function|gamma function]]
