## Incomplete Beta Function

The <strong>incomplete beta function</strong>, a generalization of the [[Beta Function|beta function]], is defined as

![[Mathematics/Algebra/Algebra - Subfields/Abstract Algebra - Modern Algebra - Structural Algebra/Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms/Transformations - Properties／Types/Transformations - Number of Input／Output Types (Multi／Single-Variable／Variate／Vector-Valued Functions)/Multi-Variable／Multivariable／Multivariate Functions/Beta Function/Incomplete Beta Function - Regularized Incomplete Beta Function/incomplete-beta-distribution-1.png|301]]

For 𝑥 = 1, the incomplete beta function coincides with the complete beta function. The relationship between the two functions is like that between the [[Gamma Function|gamma function]] and its generalization the [incomplete gamma function](https://en.wikipedia.org/wiki/Incomplete_gamma_function).
