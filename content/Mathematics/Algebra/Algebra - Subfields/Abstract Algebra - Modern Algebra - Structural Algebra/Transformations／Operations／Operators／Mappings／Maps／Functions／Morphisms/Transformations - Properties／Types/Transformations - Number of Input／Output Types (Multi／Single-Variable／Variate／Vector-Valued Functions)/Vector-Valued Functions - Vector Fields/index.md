###### Vector Fields
- is a vector-valued function whose number of outputs equals the number of inputs
- a <strong>vector field</strong> is an assignment of a [[Vectors|vector]] to each point in a subset of space
- helps to visualize [[Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms|functions]] that have the same number of dimensions in the input as the output
- a 1D vector field is a [[Scalar-Valued Functions - Scalar Fields|scalar field]]
