###### Hermitian Adjoint/Conjugate (Transformations/Operations/Operators/Mappings/Maps/Functions) (𝑇<sup>†</sup>,𝑇<sup>\*</sup>)

```excerpt
- specifically in [operator theory](https://en.wikipedia.org/wiki/Operator_theory), each [[Linear (Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms) - Vector Space Homomorphism|linear operator (𝑇)]] on an [[Inner Product Spaces|inner product space (𝑉,𝐹,⟨·,·⟩)]] defines a <strong>Hermitian adjoint</strong> (or <strong>adjoint</strong>) operator 𝑇<sup>∗</sup> or 𝑇<sup>†</sup>on that space according to the rule:
	- ⟨𝑇𝑥,𝑦⟩ = ⟨𝑥,𝑇<sup>∗</sup>𝑦⟩ = ⟨𝑥,𝑇<sup>†</sup>𝑦⟩
- corresponding matrix representation: [[Transpose Matrix - Complex-Conjugate／Hermitian／Adjoint／Transjugate／Bedaggered Transpose Matrix|Hermitian Matrix]]
```
