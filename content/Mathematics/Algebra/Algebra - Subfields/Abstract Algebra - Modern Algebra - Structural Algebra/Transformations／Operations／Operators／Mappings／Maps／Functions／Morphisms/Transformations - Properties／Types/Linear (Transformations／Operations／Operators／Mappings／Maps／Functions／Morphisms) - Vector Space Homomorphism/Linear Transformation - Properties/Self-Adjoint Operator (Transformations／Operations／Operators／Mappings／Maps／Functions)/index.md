###### Self-Adjoint Operator (Transformations/Operations/Operators/Mappings/Maps/Functions) (𝑇<sup>†</sup>𝑇)

```excerpt
- given a[[Linear (Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms) - Vector Space Homomorphism|linear operator (𝑇)]] the <em>self-adjoint</em> of 𝑇 is defined as:
	- 𝑠𝑒𝑙𝑓-𝑎𝑑𝑗𝑜𝑖𝑛𝑡(𝑇) = 𝑇<sup>†</sup>𝑇
- where 𝑇<sup>†</sup> is the [[Hermitian Adjoint／Conjugate (Transformations／Operations／Operators／Mappings／Maps／Functions)|Hermitian adjoint]]
```
