###### Multilinear Map - k-Linear Map
````excerpt
- is a [[Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms|function]] of several variables that is linear separately in each variable
	- 𝑓: 𝑉<sub>1</sub> ⨯ ... ⨯ 𝑉<sub>𝑛</sub> → 𝑋
- a multilinear map of:
	- one variable is a [[Linear (Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms) - Vector Space Homomorphism|linear map]] 𝑓: 𝑉<sub>1</sub> → 𝑋
	- two variables is a [[Bilinear (Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms)|bilinear map]] 𝑓: 𝑉<sub>1</sub> ⨯ 𝑉<sub>2</sub> → 𝑋

- a multilinear map of:
	- k copies of the same variable and with a [[Scalar-Valued Functions - Scalar Fields|scalar field]] 𝐹 as its [[Functions (Domain - Codomain - Preimage - Image - Range)|codomain]] is a [[Multilinear Functionals - Multilinear Forms - k Forms - k-forms|multilinear form]] 𝑓: 𝑉 ⨯ ... ⨯ 𝑉 → 𝐹
````

# Multilinear Map - Definition

A <em>multilinear map</em> is a function:
- 𝑓: 𝑉<sub>1</sub> ⨯ ... ⨯ 𝑉<sub>𝑛</sub> → 𝑋

such that:
- 𝑉<sub>1</sub>, ..., 𝑉<sub>𝑛</sub>, and 𝑋 are [[Vector Spaces - Linear Spaces|vector spaces]]
- for each 𝑖, if all the variables but 𝑣<sub>𝑖</sub> are held constant then 𝑓(𝑣<sub>1</sub>, ..., 𝑣<sub>𝑖</sub>, ..., 𝑣<sub>𝑛</sub>) is a [[Linear (Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms) - Vector Space Homomorphism|linear function]] of 𝑉<sub>𝑖</sub> to 𝑋
