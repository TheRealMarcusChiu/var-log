###### Generalized Inverse - G-Inverse Element
````excerpt
- the <strong>g-inverse</strong> of an element 𝑥 is an element 𝑦 that has some properties of an [[Invertibility／Invertible／Inverse Element - Right／Left Inverse Element|inverse element]] but does not necessarily have all of them
- <strong>g-inverses</strong> can be defined in any [[Mathematical Structures|mathematical structure]] that involves [[Associativity／Associative|associative]] multiplication (i.e. in a [[Algebraic Structures|semigroup]])
````
