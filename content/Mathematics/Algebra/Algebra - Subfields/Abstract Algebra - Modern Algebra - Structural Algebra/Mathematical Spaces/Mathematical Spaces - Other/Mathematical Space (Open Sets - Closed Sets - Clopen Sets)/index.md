# Open Set
- is a generalization of an [[Real Intervals (Empty／Degenerate／Proper／Bounded／Finite／Half-Bounded／Unbounded／Left／Right／Open／Closed Intervals)|open interval]] in the real line
- an <em>open set</em> is identical to its [[Mathematical Space (Interior - Exterior)|interior]]

# Closed Set
- is a generalization of a [[Real Intervals (Empty／Degenerate／Proper／Bounded／Finite／Half-Bounded／Unbounded／Left／Right／Open／Closed Intervals)|closed interval]] in the real line
- a <em>closed set</em> contains all its [[Mathematical Space (Limit Points - Accumulation Points - Cluster Points)|limit points]]

# Clopen Set
- a <em>clopen set</em> is both open and closed
