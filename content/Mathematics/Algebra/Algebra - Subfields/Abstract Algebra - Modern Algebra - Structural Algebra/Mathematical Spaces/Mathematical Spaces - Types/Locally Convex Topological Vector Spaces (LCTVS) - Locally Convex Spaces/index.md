###### Locally Convex Topological Vector Spaces (LCTVS) - Locally Convex Spaces
````excerpt
- is a type of [[Mathematical Spaces|mathematical space]]
- is a type of [[Topological Vector Spaces (TVS) - Linear Topological Spaces|topological vector space (𝑉,𝐹,𝜏)]] that is <em>locally convex</em>
- is a type of [[Topological Vector Spaces (TVS) - Linear Topological Spaces|topological vector space (𝑉,𝐹,𝜏)]] that generalizes [[Normed Vector Spaces - Normed Spaces|normed vector spaces (𝑉,𝐹,||·||)]]

	- a [[Topological Vector Spaces (TVS) - Linear Topological Spaces|topological vector space (𝑉,𝐹,𝜏)]] whose [[Topology|topology (𝜏)]] is generated by translations of [balanced](https://en.wikipedia.org/wiki/Balanced_set), [absorbent](https://en.wikipedia.org/wiki/Absorbent_set), [convex sets](https://en.wikipedia.org/wiki/Convex_set)
	- a vector space with a family of [[Norms - Semi-Norms|semi-norms]], and a [[Topology|topology (𝜏)]] defined in terms of that family
````

# LCTVS - Definition via Convex Sets

A [[Topological Vector Spaces (TVS) - Linear Topological Spaces|topological vector space (𝑉,𝐹,𝜏)]] is called locally convex if it has a [neighborhood basis](https://en.wikipedia.org/wiki/Neighborhood_basis) at the origin consisting of balanced, convex sets.

A subset 𝑆 in 𝑉 is called:
- <strong>convex</strong> if for all 𝑥,𝑦∊𝑆 and 0≤𝑡≤1 : 𝑡𝑥 + (1-𝑡)𝑦 ∊ 𝑆. In other words, 𝑆 contains all line segments between points in 𝑆.
- <strong>circled</strong> if for all 𝑥∊𝑆 and scalars 𝛼 : if |𝛼|=1 then 𝛼𝑥∊𝑆
- <strong>balanced</strong> if for all 𝑥∊𝑆 and scalars 𝛼 : if |𝛼|≤1 then 𝛼𝑥∊𝑆
- a <strong>cone</strong> (when the underlying [[Fields (Algebraic Structure) - Field Theory|field (𝐹)]] is ordered) if for all 𝑥∊𝑆 and t≥0 : 𝑡𝑥∊𝑆
- <strong>absorbent/absorbing</strong> if for all 𝑥∊𝑆, there exists 𝑟\>0 such that 𝑥∊𝑡𝑆 for all 𝑡∊𝐹 satisfying |𝑡|\>𝑟. The set 𝑆 can be scaled out by any "large" value to absorb every point in the space
	- in any topological vector space, every [[Topological Spaces (Neighborhoods of a Point／Set)|neighborhood]] of the origin is absorbent
- <strong>absolutely convex</strong> or a <strong>disk</strong> if it is both <em>balanced</em> and <em>convex</em>. This is equivalent to it being closed under linear combinations whose coefficients absolutely sum to ≤ 1; such a set is absorbent if it spans all of 𝑉
