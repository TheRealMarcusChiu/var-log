###### Basis Functions

```excerpt
- is an element of a particular [[Mathematical Space (Basis)|basis]] for a [[Function Spaces|function space]]
- every [[Functions (Domain - Codomain - Preimage - Image - Range)|function]] in the [[Function Spaces|function space]] can be represented as a linear combination of <em>basis functions</em>, just as every [[Vectors|vector]] in a [[Vector Spaces - Linear Spaces|vector space]] can be represented as a linear combination of [[Basis Vectors|basis vectors]]
	- i.e. [[Function Spaces|function space]] is to the <em>basis function</em> as [[Vector Spaces - Linear Spaces|vector space]] is to the [[Basis Vectors|basis vector]]
```

^excerpt

# Basis Functions - Examples

Given a [[Finite-Dimensional Vector Spaces|finite-dimensional vector space (𝑉,𝐹)]] where 𝑉 is a set of all polynomial functions of degree 𝑘 or less

- example finite basis #1 {1, 𝑥, 𝑥<sup>2</sup>, ..., 𝑥<sup>𝑘</sup>} <font style="color: rgb(122,134,154);"># "standard basis"</font>
- example finite basis #2 {4, 6𝑥, 1𝑥<sup>2</sup>, ..., 2𝑥<sup>𝑘</sup>}

Given an [[Infinite-Dimensional Vector Spaces|infinite-dimensional vector space (𝑉,𝐹)]] where 𝑉 is a set of all polynomial functions of any degree

- example infinite basis #1 {1, 𝑥, 𝑥<sup>2</sup>, ... } <font style="color: rgb(122,134,154);"># "standard basis"</font>
- example infinite basis #2 {4, 6𝑥, 1𝑥<sup>2</sup>, ... }

Given an [[Infinite-Dimensional Vector Spaces|infinite-dimensional vector space (𝑉,𝐹)]] where 𝑉 is a set of functions, where each function (𝑓) outputs a [[Fields (Algebraic Structure) - Field Theory|field (𝐹)]] (i.e. 𝑓: 𝑋 → 𝐹)

- example infinite basis #1 {1, 𝑥, 𝑥<sup>2</sup>, ..., 𝑐𝑜𝑠(𝑥), ..., 5<sup>𝑥</sup>, ... }
