###### Neighborhood 𝑁<sub>𝑑</sub>(·,𝜖) <sup>[[Mathematical Space (Neighborhoods)|(general definition)]]</sup>
````excerpt
- given a [[Metric Spaces|metric space (𝑋,𝑑)]] where:
	- 𝑋 is a set of elements
	- 𝑑(·,·) is a [[Distance Measures／Metrics／Semi-Metrics／Functions|distance metric]]
- Let 𝑎∈𝑋 and 𝜖\>0. Then the <strong>𝜖-neighborhood 𝑁<sub>𝑑</sub>(𝑎)</strong> is a set of elements defined as:
	- 𝑁<sub>𝑑</sub>(𝑎) = { 𝑏∈𝑋 : 𝑑(𝑎,𝑏)\<𝜖 }
````
