###### Discrete Topology (𝜏)
````excerpt
- is a type of [[Topology|topology (𝜏)]] on a set 𝑋 where it contains every possible subset of 𝑋 (𝜏 = powerset of 𝑋):
	- 𝜏 = { 𝑆 : 𝑆 ⊂ 𝑋 }
- when paired with its elements (𝑋) it forms a [[Discrete Topological Spaces|discrete topological space (𝑋,𝜏)]]
````
^excerpt

# Discrete Topology - Example
- Let 𝑋 = {𝑎, 𝑏, 𝑐}, then a <em>discrete topology (𝜏)</em> = {⦰, {𝑎}, {𝑏}, {𝑐}, {𝑎, 𝑏}, {𝑏, 𝑐}, {𝑎, 𝑐}, {𝑎, 𝑏, 𝑐}}
