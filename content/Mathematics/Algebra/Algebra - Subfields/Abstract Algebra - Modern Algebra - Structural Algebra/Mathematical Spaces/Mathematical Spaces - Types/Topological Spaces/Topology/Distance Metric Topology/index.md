###### Distance Metric Topology (<em>𝜏<sub>𝑑</sub></em>)
````excerpt
- is a type of [[Topology|topology (𝜏)]] induced by a [[Distance Measures／Metrics／Semi-Metrics／Functions|distance metric (𝑑)]] defined on a [[Metric Spaces|metric space (𝑋,𝑑)]]
- given a [[Distance Measures／Metrics／Semi-Metrics／Functions|distance metric (𝑑)]] the set { 𝐵<sub>𝑑</sub>(𝑥,𝜖) | 𝑥∈𝑋 and 𝜖\>0 } forms the BASIS of a <em>distance metric topology (</em><em>𝜏<sub>𝑑</sub></em><em>)</em>, where 𝐵<sub>𝑑</sub>(𝑥,𝜖) is an [[Metric Spaces (Epsilon Balls - Open Sets - Boundary Points／Sets - Closed Sets - Closures - Bounded Sets ／ Boundedness - Unbounded Sets ／ Unboundedness - Totally Bounded Sets ／ Totally Boundedness)|epsilon-ball/open-set]] defined as 𝐵<sub>𝑑</sub>(𝑥,𝜖) = { 𝑦∈𝑋 : 𝑑(𝑥,𝑦) \< 𝜖 }
- the <em>distance metric topology</em> makes 𝑋 a [T2-space](https://mathworld.wolfram.com/T2-Space.html)
````
^excerpt
