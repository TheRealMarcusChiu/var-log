###### Quotient Topology (𝜏<sub>𝑋/∼</sub>)
````excerpt
- is a type of [[Topology|topology (𝜏)]] on a set 𝑋
- a set 𝑋 endowed with a <em>quotient topology</em> is called a [[Quotient Spaces (topology)|quotient space]]
- If ∼ is an [[Equivalence Relations|equivalence relation]] on a [[Topological Spaces|topological space]] 𝑋, then the [[Quotient Spaces (topology)|quotient space]] 𝑋/\~ = {\[𝑥\] : 𝑥∈𝑋} and the <em>quotient topology</em> on 𝑋/∼ is defined to be:
	- 𝜏<sub>𝑋/∼</sub> = { 𝐴⊆𝑋/\~ | 𝑞<sup>−1</sup>(𝐴) is [[Topological Spaces (Open Sets - Closed Sets - Clopen Sets - Closures)|open]] in 𝑋 }
- where:
	- 𝑞: 𝑋 → 𝑋/∼ defined by 𝑝 ↦ \[𝑝\] is the quotient map
	- \[𝑥\] is the equivalence class (or in this case the [[Cosets|coset]]) of 𝑥
````
^excerpt

# Theorems

> [!expand-ui]- The quotient topology is a topology
> To prove the quotient topology on a set 𝑋 is a [[Topology|topology]], note that:
> - 𝑞<sup>−1</sup>(∅) = ∅ ∈ 𝜏<sub>𝑋</sub>
> - 𝑞<sup>−1</sup>(𝑋/∼) = 𝑋 ∈ 𝜏<sub>𝑋</sub>
> - and for all 𝐴<sub>1</sub>,𝐴<sub>2</sub>∈ 𝑋/∼:
> 	- 𝑞<sup>−1</sup>(𝐴<sub>1</sub>∩𝐴<sub>2</sub>) = 𝑞<sup>−1</sup>(𝐴<sub>1</sub>) ∩ 𝑞<sup>−1</sup>(𝐴<sub>2</sub>) ∈ 𝜏<sub>𝑋</sub>
> 	- 𝑞<sup>−1</sup>(∪<sub>𝐴∈𝒜</sub>𝐴) = ∪<sub>𝐴∈𝒜</sub>𝑞<sup>−1</sup>(𝐴) ∈ 𝜏<sub>𝑋</sub>

> [!expand-ui]- The quotient map 𝑞: 𝑋 → 𝑋/∼ is continuous, and a function 𝑓: 𝑋/∼ → 𝑍 is continuous if and only if 𝑓 ◦ 𝑞 is continuous
> It is clear that 𝑞 is [[Continuous Functions／Transformations Between Topological Spaces (Topological Continuity - Topologically Continuous)|continuous]] and that 𝑓 [[Continuous Functions／Transformations Between Topological Spaces (Topological Continuity - Topologically Continuous)|continuous]] implies 𝑓 ◦ 𝑞 is also [[Continuous Functions／Transformations Between Topological Spaces (Topological Continuity - Topologically Continuous)|continuous]].
>
> So suppose 𝑓 ◦ 𝑞 is [[Continuous Functions／Transformations Between Topological Spaces (Topological Continuity - Topologically Continuous)|continuous]] and let 𝐴 ∈ 𝜏<sub>𝑍</sub>.
>
> Note that 𝑓<sup>-1</sup>(𝐴) is [[Topological Spaces (Open Sets - Closed Sets - Clopen Sets - Closures)|open]] if and only if 𝑞<sup>−1</sup>𝑓<sup>−1</sup>(𝐴) is [[Topological Spaces (Open Sets - Closed Sets - Clopen Sets - Closures)|open]] and of course 𝑞<sup>−1</sup>𝑓<sup>−1</sup>(𝐴) = (𝑓 ◦ 𝑞)<sup>−1</sup>(𝐴), which is [[Topological Spaces (Open Sets - Closed Sets - Clopen Sets - Closures)|open]].
# Examples

The following are quotient topologies
1. \[0,1\] / (0\~1)
2. ℝ/\~ where 𝑥\~𝑦 if and only if 𝑥-𝑦 ∊ ℤ. This is often written as ℝ/ℤ. This is in fact [[Homeomorphic／Homeomorphisms - Topological Isomorphism ／ Topologically Isomorphic - Bicontinuous Function|homeomorphic]] to \[0,1\] / (0\~1) and can be thought of as a circle.
