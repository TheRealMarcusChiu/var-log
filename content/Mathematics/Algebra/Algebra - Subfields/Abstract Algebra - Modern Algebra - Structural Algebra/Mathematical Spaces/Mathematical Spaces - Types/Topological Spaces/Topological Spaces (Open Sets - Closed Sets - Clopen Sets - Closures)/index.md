# Topology (𝜏)

![[Topology#^excerpt]]

# Open Sets <sup>[[Mathematical Space (Open Sets - Closed Sets - Clopen Sets)|(general definition)]]</sup>

An <em>open set</em> is an element of a [[Topology|topology (𝜏)]] of a [[Topological Spaces|topological space (𝑋,𝜏)]]

A set 𝑆∈𝜏 is called <em>open</em>

# Closed Sets <sup>[[Mathematical Space (Open Sets - Closed Sets - Clopen Sets)|(general definition)]]</sup>

A subset 𝑆∈𝑋 is called <em>closed</em> if its complement 𝑋\𝑆∊𝜏

# Clopen Sets <sup>[[Mathematical Space (Open Sets - Closed Sets - Clopen Sets)|(general definition)]]</sup>

A subset 𝑆 of a [[Topological Spaces|topological space (𝑋,𝜏)]] is called <em>clopen</em> if 𝑆∈𝜏 and 𝑋\𝑆∈𝜏.

In any topological space, the empty set ∅ and the set 𝑋 itself are always clopen.

# Closures <sup>[[Mathematical Space (Closures - Spans - Generated Sets)|(general definition)]]</sup>

The <em>closure</em> of a subset 𝑆 of a [[Topological Spaces|topological space (𝑋,𝜏)]] consists of all points in 𝑆 together with all [[Topological Spaces (Limit Points - Accumulation Points - Cluster Points)|limit points]] of 𝑆
