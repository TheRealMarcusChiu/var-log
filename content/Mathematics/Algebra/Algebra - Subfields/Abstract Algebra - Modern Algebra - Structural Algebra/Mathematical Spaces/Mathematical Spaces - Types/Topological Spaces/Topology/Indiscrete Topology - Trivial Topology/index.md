###### Indiscrete Topology - Trivial Topology (𝜏)

```excerpt
- is a type of [[Topology|topology (𝜏)]] on a set 𝑋 where the family 𝜏 = {⦰, 𝑋}
- is the smallest topology on a set, where the only [[Topological Spaces (Open Sets - Closed Sets - Clopen Sets - Closures)|open sets]] are the empty set ⦰ and the entire set 𝑋 (these are also the [[Topological Spaces (Open Sets - Closed Sets - Clopen Sets - Closures)|closed sets]])
```

^excerpt
