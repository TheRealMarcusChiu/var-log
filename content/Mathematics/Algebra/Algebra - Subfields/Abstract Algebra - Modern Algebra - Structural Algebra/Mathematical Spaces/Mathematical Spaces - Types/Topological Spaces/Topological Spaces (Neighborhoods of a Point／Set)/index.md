###### Neighborhoods 𝑁<sub>𝜏</sub>(·) <sup>[[Mathematical Space (Neighborhoods)|(general definition)]]</sup>
````excerpt
- given a [[Topological Spaces|topological space (𝑋,𝜏)]] where:
	- 𝑋 is a set
	- 𝜏 is a [[Topology|topology]] on 𝑋
- a <em>neighborhood</em> of point 𝑎∊𝑋, denoted as 𝑁<sub>𝜏</sub>(𝑎), is a subset 𝑆 of 𝑋 that contains an [[Topological Spaces (Open Sets - Closed Sets - Clopen Sets - Closures)|open set]] 𝑂 (i.e. 𝑂∊𝜏) where 𝑂 contains 𝑎:
	- 𝑎∊𝑂⊆𝑆⊆𝑋 → 𝑁<sub>𝜏</sub>(𝑎)=𝑆
- a neighborhood of set 𝐴⊆𝑋, denoted as 𝑁<sub>𝜏</sub>(𝐴), is a subset 𝑆 of 𝑋 that contains an [[Topological Spaces (Open Sets - Closed Sets - Clopen Sets - Closures)|open set]] 𝑂 (i.e. 𝑂∊𝜏) where 𝑂 contains 𝐴:
	- 𝐴⊆𝑂⊆𝑆⊆𝑋 → 𝑁<sub>𝜏</sub>(𝐴)=𝑆
````
