###### Particular Point Topology (𝜏)

```excerpt
- is a type of [[Topology|topology (𝜏)]] on a set 𝑋 where the family 𝜏 = { 𝑆⊆𝑋 | 𝑝∊𝑆 } ∪ { ⦰ } where 𝑝 is a particular point
- a generalization of the <em>particular point topology</em> is the [closed extension topology](https://en.wikipedia.org/wiki/Closed_extension_topology)
```

^excerpt

# Particular Point Topology - Types

Given a particular point topology on set 𝑋:

- if 𝑋 has 2 points, then the particular point topology on 𝑋 is the <strong>Sierpinski space</strong>
- if 𝑋 is finite (with at least 3 points), then the particular point topology on 𝑋 is called the <strong>finite particular point topology</strong>
- if 𝑋 is countable infinite, then the particular point topology on 𝑋 is called the <strong>countable particular point topology</strong>
- if 𝑋 is uncountable infinite, then the particular point topology on 𝑋 is called the <strong>uncountable particular point topology</strong>
