A finite set 𝐵 of linearly independent vectors is said to be a [[Basis Vectors|basis vectors]] for the [[Vector Spaces - Linear Spaces|vector space]] 𝑉 if 𝐵 spans 𝑉.

A vector space having a finite basis is said to be a [[Finite-Dimensional Vector Spaces|finite-dimensional vector space]].
All other vector spaces are said to be [[Infinite-Dimensional Vector Spaces|infinite-dimensional vector spaces]].
