###### Vector Space of Bounded Linear Transformations - Set of Bounded Linear Transformations - 𝐵(𝑋,𝑌) or 𝐵(𝑋)
````excerpt
- is a type of [[Mathematical Spaces|mathematical space]]
- is a type of [[Vector Space of Linear Transformations - Set of Linear Transformations|vector space of linear transformations]]
- suppose we have a [[Bounded／Unbounded Linear Operator (Transformations／Operations／Operators／Mappings／Maps／Functions)|bounded linear transformation (𝐿: 𝑋→𝑌)]] for [[Vector Spaces - Linear Spaces|vector spaces]] 𝑋 and 𝑌. Then the space of all those transformations forms the <em>vector space of bounded linear transformations</em>
- If 𝑋 and 𝑌 are finite vector spaces then the dimension is 𝑑𝑖𝑚(𝑋) \* 𝑑𝑖𝑚(𝑌)
- The set of bounded linear transformations 𝑋→𝑌 is denoted by 𝐵(𝑋,𝑌). If 𝑋=𝑌, we may abbreviate 𝐵(𝑋,𝑋) by 𝐵(𝑋)
````
^excerpt
