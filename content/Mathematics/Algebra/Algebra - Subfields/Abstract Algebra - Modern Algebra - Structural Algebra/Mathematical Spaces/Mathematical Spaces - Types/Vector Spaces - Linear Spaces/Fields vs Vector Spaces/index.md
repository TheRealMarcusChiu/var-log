# Multiplication Operations are Fundamentally Different

It is true that vector spaces and fields both have operations we often call multiplication, but these operations are fundamentally different

The operations on a field 𝐹 are:
- +: 𝐹×𝐹→𝐹
- ×: 𝐹×𝐹→𝐹

The operations on a vector space 𝑉 over a field 𝐹 are:
- +: 𝑉×𝑉→𝑉
- ×: 𝐹×𝑉→𝑉

# Every Field is a 1D Vector Space Over Itself
- but not every vector space is a field

# Multiplicative Inverse
- every field has a multiplicative inverse
- vector space has NO multiplicative inverse defined (this is because scalar multiplication always outputs a vector, thus it will never be the identity value 1 which is a scalar field)
