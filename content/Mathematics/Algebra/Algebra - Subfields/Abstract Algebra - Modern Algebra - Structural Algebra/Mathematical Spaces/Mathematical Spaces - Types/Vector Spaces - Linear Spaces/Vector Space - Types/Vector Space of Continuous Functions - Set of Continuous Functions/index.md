###### Vector Space of Continuous Functions on Interval - Set of Continuous Functions - 𝐶\[a, b\]
- TODO, separate between:
	- continuous functions that have one input and one output
	- continuous functions that have multi inputs and multi outputs
````excerpt
- is a type of [[Infinite-Dimensional Vector Spaces|infinite dimensional vector space (𝑉,𝐹)]] of [[Continuous／Continuity [at a point - everywhere] (Transformations／Operations／Operations／Mappings／Maps／Functions／Morphisms)|continuous functions]] over the real field
- the space is defined as 𝐶\[𝑎,𝑏\] = {𝑓 | 𝑓 : \[𝑎,𝑏\] → 𝑅 is continuous}
- the space is a vector space because the sum and scalar multiples of continuous functions are also continuous (and addition is commutative)
- the space has:
	- an additive [[Identity／Neutral Element - Right／Left Identity Element|identity element]] 0 (i.e. 0 + 𝑓 = 𝑓)
	- additive [[Invertibility／Invertible／Inverse Element - Right／Left Inverse Element|inverses]] -𝑓 (i.e. 𝑓 + (-𝑓) = 0)
- the space 𝐶(𝐼) is the space of all continuous functions on 𝐼
- the space 𝐶<sub>∞</sub>(𝑅) is the set of all infinitely differentiable functions 𝑓: 𝑅 → 𝑅
- there are NO finite [[Basis Vectors|basis vectors]] that span this vector space
````
