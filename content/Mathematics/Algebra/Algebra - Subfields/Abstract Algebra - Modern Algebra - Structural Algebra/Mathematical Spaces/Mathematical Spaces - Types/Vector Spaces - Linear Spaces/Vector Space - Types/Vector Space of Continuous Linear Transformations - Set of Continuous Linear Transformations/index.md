###### Vector Space of Continuous Linear Transformations - Set of Continuous Linear Transformations - 𝐶(𝑋,𝑌) or 𝐶(𝑋)

```excerpt
- is a type of [[Mathematical Spaces|mathematical space]]
- is a type of [[Vector Space of Linear Transformations - Set of Linear Transformations|vector space of linear transformations]]
- suppose we have a [[Continuous Linear Operator (Transformations／Operations／Operators／Mappings／Maps／Functions)|continuous linear transformation (𝐿: 𝑋→𝑌)]] for [[Vector Spaces - Linear Spaces|vector spaces]] 𝑋 and 𝑌. Then the space of all those transformations forms the <em>vector space of continuous linear transformations</em>
- If 𝑋 and 𝑌 are finite vector spaces then the dimension is 𝑑𝑖𝑚(𝑋) \* 𝑑𝑖𝑚(𝑌)
- The set of continuous linear transformations 𝑋→𝑌 is denoted by 𝐶(𝑋,𝑌). If 𝑋=𝑌, we may abbreviate 𝐶(𝑋,𝑋) by 𝐶(𝑋)
```
