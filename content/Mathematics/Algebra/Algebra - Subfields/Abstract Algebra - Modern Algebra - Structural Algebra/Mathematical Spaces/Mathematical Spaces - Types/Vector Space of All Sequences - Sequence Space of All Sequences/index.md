# Vector Space of All Sequences - Definition

A [sequence](https://en.wikipedia.org/wiki/Sequence) 𝑥̅<sub></sub>= (𝑥<sub>𝑛</sub>)<sub>𝑛∈ℕ</sub> in a set 𝑋 is just an 𝑋-valued map 𝑥̅: ℕ → 𝑋 whose value at 𝑛∈ℕ is denoted by 𝑥<sub>𝑛</sub> instead of the usual parentheses notation 𝑥(𝑛).

- 𝐹 denotes the [[Fields (Algebraic Structure) - Field Theory|field]] either of [[Real Numbers System|real numbers]] or [[Complex Numbers System|complex numbers]]
- ℕ denotes the set of [[Numbers Classes／Classification of Numbers|natural numbers]]

Thus:

- 𝐹<sup>ℕ</sup>denotes the vector space of all sequences (i.e. 𝐹<sup>ℕ</sup> = 𝛱<sub>𝑛∊ℕ</sub>𝐹)

𝐹<sup>ℕ</sup> is also a [[Vector Spaces - Linear Spaces|vector space]] that satisfies:

- <strong>componentwise addition</strong> : (𝑥<sub>𝑛</sub>)<sub>𝑛∈ℕ</sub> + (𝑦<sub>𝑛</sub>)<sub>𝑛∈ℕ</sub> = (𝑥<sub>𝑛</sub> + 𝑦<sub>𝑛</sub>)<sub>𝑛∈ℕ</sub>
- <strong>componentwise </strong><strong>scalar multiplication</strong> : 𝛼(𝑥<sub>𝑛</sub>)<sub>𝑛∈ℕ</sub> = (𝛼𝑥<sub>𝑛</sub>)<sub>𝑛∈ℕ</sub>
