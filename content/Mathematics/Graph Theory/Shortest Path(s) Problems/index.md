In [[Graph Theory|graph theory]], the <strong>shortest path problem</strong> is the [[Problems|problem]] of finding a path between two vertices/nodes in a graph such that the sum of the weights of its constituent edges is minimized

# Variants of Shortest Path(s) Problem

- <strong>[[Single-Source Shortest Paths Problem|single-source shortest paths]]</strong> - given vertex 𝑣 find the shortest path TO every other node
- <strong>[[Single-Destination Shortest Paths Problem|single-destination shortest paths]]</strong> - given vertex 𝑣 find the shortest path FROM every other node
- <strong>[[Single-Pair Shortest Path Problem|single-pair shortest path]]</strong> - given 2 vertices 𝑢 and 𝑣, find the shortest path between them
- <strong>[[All-Pairs Shortest Paths Problem|all-pairs shortest paths]]</strong> - find the shortest path between all possible pairs of vertices
