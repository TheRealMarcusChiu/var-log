<strong>Degree </strong><strong>Sequences Problem (undirected graphs only)</strong>

- given a sequence of numbers (e.g. {1, 3, 2, 5}) determine whether there exists a graph such that:
  - the number of nodes equals the size of the sequence
  - each node has a degree equal to a number in the sequence

# Degree Sequence Check

- <strong>complex graphical check</strong> - a degree sequence is graphical iff there are an even number of odd degrees within a given degree sequence
