### DFS Spanning Tree (DST)

Given a graph 𝐆 = (𝐕, 𝐄) and given a node 𝑋<sub>1</sub>, a DFS spanning tree 𝐓 of 𝐆 is generated by applying a [[Depth First Search (DFS)|depth-first-search traversal]] over the graph, yielding an ordering 𝑑 = (𝑋<sub>1</sub>, ..., 𝑋<sub>𝑛</sub>). The DFS spanning tree 𝐓 of 𝐆 is defined as the tree rooted at the first node, 𝑋<sub>1</sub>, and which includes only the traversed edges of 𝐆

DFS spanning tree 𝐓 = (𝐕, 𝐄') where:

- 𝐕 = same 𝐕
- 𝐄' = {(𝑋<sub>𝑖</sub>, 𝑋<sub>𝑗</sub>) | 𝑋<sub>𝑗</sub> traversed from 𝑋<sub>𝑖</sub> by [[Depth First Search (DFS)|DFS traversal]]}
