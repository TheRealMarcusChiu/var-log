- <strong>depth</strong> of a node is the number of edges from the node to the tree's root node. A root node will have a depth of 0
- <strong>height</strong> of a node is the number of edges on the <em>longest path</em> from the node to a leaf. A leaf node will have a height of 0

- depth of a tree = height of a tree
- depth and height of a node is not always equal
