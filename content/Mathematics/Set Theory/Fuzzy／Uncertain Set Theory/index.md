<strong>fuzzy sets (a.k.a. uncertain sets)</strong> are somewhat like sets whose elements have degrees of membership

a fuzzy set is a pair (𝑈,𝑚) where:

- 𝑈 is a set
- 𝑚: 𝑈 → \[0,1] is a membership function
