- <strong>disjoint</strong>
	- [[Set／Sets|sets]] 𝐴 and 𝐵 are disjoint if their intersection is empty
	- 𝐴∩𝐵 = ∅
- <strong>pairwise disjoint</strong> (definition \#1)
	- synonymous to disjoint
- <strong>pairwise disjoint</strong> (definition \#2)
	- a finite collection of sets 𝐶 = {𝛢<sub>1</sub>, 𝛢<sub>2</sub>, 𝛢<sub>3</sub>, ...} are pairwise disjoint if all pairs of sets in 𝐶 are disjoint
	- 𝛢<sub>𝑖</sub> ∩ 𝛢<sub>𝑗</sub> = ∅ for any 𝑖 ≠ 𝑗
- <strong>mutually disjoint</strong>
	- synonymous to disjoint
- <strong>mutually exclusive</strong> (definition \#1)
	- a finite collection of sets 𝐶 = {𝛢<sub>1</sub>, 𝛢<sub>2</sub>, 𝛢<sub>3</sub>, ...} are mutually exclusive if each set in 𝐶 is disjoint to any combination of other sets in 𝐶
