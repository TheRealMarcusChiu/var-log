```excerpt
- <strong>Combinatorial Programming</strong> or <strong>Combinatorial Optimization</strong> is a subset of [[Integer／Discrete Linear Programming／Optimization (ILP)|ILP]] (similar to 0-1 Programming) that consists of finding an optimal object from a finite set of objects
- <strong>0-1 Programming</strong> or <strong>0-1 Optimization </strong>problem is a type of [[Integer／Discrete Linear Programming／Optimization (ILP)|ILP]] (similar to Combinatorial Programming) in which some or all of the variables are restricted to be 0 or 1 values
```

^excerpt
