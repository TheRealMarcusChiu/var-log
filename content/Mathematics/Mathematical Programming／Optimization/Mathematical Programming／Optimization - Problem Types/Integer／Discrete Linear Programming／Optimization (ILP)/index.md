###### Integer/Discrete Linear Programming/Optimization (ILP)
````excerpt
- is a [[Linear Programming／Optimization (LP)|linear programming (LP) problem]] where ALL variables are restricted to integer/discrete values
- is usually much more difficult to solve than [[Linear Programming／Optimization (LP)|LP]], often we have to apply heuristic or approximative approaches because finding the exact global optimum for a large problem is rarely feasible
- if some variables are mixed between integer values and real values, we call it a [[Mixed Integer／Discrete Linear Programming／Optimization (MILP)|Mixed Integer Linear Programming]] problem
````
^excerpt
