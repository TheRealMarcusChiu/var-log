### Example Linear Programming Problem
- <strong>objective function:</strong>
  maximize Z = 3x<sub>1</sub> + 2x<sub>2</sub>
- <strong>linear constraints:</strong>
  -x<sub>1</sub> + 3x<sub>2</sub> ≤ 12
  x<sub>1</sub> + x<sub>2</sub> ≤ 8
  2x<sub>1</sub> - x<sub>2</sub> ≤ 10
- <strong>non-negative variables:</strong>
  x<sub>1</sub> ≥ 0
  x<sub>2</sub> ≥ 0
