###### Mixed Integer/Discrete Linear Programming/Optimization (MILP)

```excerpt
- is a [[Linear Programming／Optimization (LP)|linear programming (LP) problem]] where SOME variables are restricted to integer/discrete values while the rest remain continuous
- when all variables are restricted to integer values, we call it an [[Integer／Discrete Linear Programming／Optimization (ILP)|Integer Linear Programming]] problem
```
