###### Jaccard Index - Jaccard Similarity Coefficient
````excerpt
- a type of [[Similarity Measures／Metrics／Semi-Metrics／Functions|similarity function]] that measures the similarity between 2 sets
- by the ratio between the size of the intersection over the size of the union
- for measuring dis-similarity see [[Jaccard Distance Metric]]
````
^excerpt

# Formula
- 𝐽(𝐴,𝐵) = |𝐴⋂𝐵| / |𝐴⋃𝐵|

where:
- 𝐽 - Jaccard Distance
- || - the size of set
- ⋂ - intersection
- ⋃ - union
- 𝐴 - set 1
- 𝐵 - set 2
