###### Squared Euclidean Distance
````excerpt
- is a [[Distance Measures／Metrics／Semi-Metrics／Functions|distance measure (𝑑)]] defined as 𝑑(𝑢̅,𝑣̅) = 𝛴<sub>1≤𝑖≤𝑛</sub>(𝑢̅<sub>𝑖</sub>-𝑣̅<sub>𝑖</sub>)<sup>2</sup> where 𝑢̅,𝑣̅<sub></sub>are [[Vectors|vectors]] in ℝ<sup>𝑛</sup>
- similar to [[L2／Euclidean Distance Metric|Euclidean Distance Metric]] but without the square-root operation
- is a simple [[Divergences|divergence]]
````
^excerpt
