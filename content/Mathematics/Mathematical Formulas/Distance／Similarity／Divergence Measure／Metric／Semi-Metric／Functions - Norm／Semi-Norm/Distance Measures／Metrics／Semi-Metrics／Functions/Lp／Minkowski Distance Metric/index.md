###### Lp/Minkowski Distance Metric

```excerpt
- is a [[Distance Measures／Metrics／Semi-Metrics／Functions|distance metric (𝑑)]] defined as 𝑑(𝑢̅,𝑣̅) = 𝐿<sub>𝑝</sub> = ||𝑢̅-𝑣̅||<sub>𝑝</sub> = \[ 𝛴<sub>1≤𝑖≤𝑛</sub>|𝑢̅<sub>𝑖</sub>-𝑣̅<sub>𝑖</sub>|<sup>𝑝</sup> \]<sup>(1/𝑝)</sup> where 𝑢̅,𝑣̅<sub></sub>are [[Vectors|vectors]] in ℝ<sup>𝑛</sup>
- can be considered as a generalization of both the [[L2／Euclidean Distance Metric|Euclidean distance]] and the [[L1／Manhattan／Rectilinear Distance Metric|Manhattan distance]]
- utilizes the [[Lp／Minkowski Norm (p-norm)|Lp Norm]] function
```

^excerpt
