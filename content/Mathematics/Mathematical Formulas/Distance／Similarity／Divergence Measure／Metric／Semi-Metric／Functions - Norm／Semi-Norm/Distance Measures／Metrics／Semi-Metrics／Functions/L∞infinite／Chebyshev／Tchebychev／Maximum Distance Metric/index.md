###### L∞infinite/Chebyshev/Tchebychev/Maximum Distance Metric

```excerpt
- is a [[Distance Measures／Metrics／Semi-Metrics／Functions|distance metric (𝑑)]] defined as 𝑑(𝑢̅,𝑣̅) = 𝐿<sub>∞</sub> = ||𝑢̅-𝑣̅||<sub>∞</sub> = \[ 𝛴<sub>1≤𝑖≤𝑛</sub>|𝑢̅<sub>𝑖</sub>-𝑣̅<sub>𝑖</sub>|<sup>∞</sup> \]<sup>(1/∞)</sup> = 𝑚𝑎𝑥(|𝑢̅<sub>𝑖</sub>-𝑣̅<sub>𝑖</sub>|) where 𝑢̅,𝑣̅<sub></sub>are [[Vectors|vectors]] in ℝ<sup>𝑛</sup>
- for instance, if we have two 2D vectors (𝑥<sub>1</sub>, 𝑦<sub>1</sub>) and (𝑥<sub>2</sub>, 𝑦<sub>2</sub>), the Chebyshev distance is 𝑚𝑎𝑥(|𝑦<sub>1</sub> - 𝑦<sub>2</sub>|, |𝑥<sub>1</sub> - 𝑥<sub>2</sub>|)
- utilizes the [[L∞infinity／Max／Chebychev／Tchebychev Norm|𝐿∞ Norm]]
```
