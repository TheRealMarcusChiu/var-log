###### L0 "Norm" (||·||<sub>0</sub>)

```excerpt
- is a [[Norms - Semi-Norms|norm (||·||)]] defined as ||𝑣̅|| = 𝐿<sub>0</sub> = ||𝑣̅||<sub>0</sub> = 𝛴<sub>1≤𝑖≤𝑛</sub>|𝑣̅<sub>𝑖</sub>|<sup>0</sup> where 𝑣̅ is a [[Vectors|vector]] in ℝ<sup>𝑛</sup>(where 0<sup>0</sup> = 0)
- is not a real norm because it fails the <em>absolute homogeneity</em> property:
	- ||𝑠𝑣|| = |𝑠|·||𝑣|| for all 𝑣∈𝑉and all scalars 𝑠
```
