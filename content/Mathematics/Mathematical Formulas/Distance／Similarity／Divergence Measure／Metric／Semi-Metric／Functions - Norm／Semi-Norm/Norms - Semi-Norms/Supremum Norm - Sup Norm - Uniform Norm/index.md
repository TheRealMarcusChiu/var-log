###### Supremum Norm - Sup Norm - Uniform Norm (||·||<sub>∞</sub>)

```excerpt
- is a [[Norms - Semi-Norms|norm (||·||)]] that assigns real-valued or complex-valued bounded functions 𝑓 defined on a set 𝑆 the non-negative number:
	- ||𝑓||<sub>∞</sub> = ||𝑓||<sub>∞,𝑆</sub> = 𝑠𝑢𝑝{|𝑓(𝑠)| : 𝑠∊𝑆}
- if 𝑥 such that 𝑥 = (𝑥<sub>1</sub>, 𝑥<sub>2</sub>, ..., 𝑥<sub>𝑛</sub>) in some finite-dimensional coordinate space it becomes the [[L∞infinity／Max／Chebychev／Tchebychev Norm|max norm]]:
	- ||𝑥||<sub>∞</sub> = ||𝑥||<sub>∞,𝑆</sub> = 𝑚𝑎𝑥(|𝑥<sub>1</sub>|, |𝑥<sub>2</sub>|, ..., |𝑥<sub>𝑛</sub>|)
```
