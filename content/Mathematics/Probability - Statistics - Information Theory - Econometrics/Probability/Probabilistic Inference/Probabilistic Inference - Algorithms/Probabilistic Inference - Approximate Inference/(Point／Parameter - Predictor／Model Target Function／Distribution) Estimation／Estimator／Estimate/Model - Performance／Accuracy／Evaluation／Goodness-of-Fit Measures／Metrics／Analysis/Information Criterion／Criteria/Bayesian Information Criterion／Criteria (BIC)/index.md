###### Bayesian Information Criterion (BIC)

- a criterion for [model selection](https://en.wikipedia.org/wiki/Model_selection) among a finite set of models; the model with the lowest BIC is preferred
- based on the [[Likelihood Function - Log-Likelihood Function - Score Function|likelihood function]]
- closely related to the [[Akaike's Information Criterion／Criteria (AIC)|Akaike information criterion (AIC)]]
