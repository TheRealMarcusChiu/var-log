###### Automatic Differentiation
````excerpt
- is a set of techniques to evaluate the [[Derivative of Scalar-Valued Function (Partial Derivative - Total Derivative - Gradient - Directional Directive - Second Order Partial Derivative)|partial derivative]] of a function specified by a computer program
````
