###### Cross-Entropy Loss Function - Log Loss Function - Negative Log Likelihood Loss Function
````excerpt
- is a [[Objective Function／Criterion - Cost／Loss／Error Function - Expected Cost／Loss／Error|loss function]] defined as: -𝑙𝑜𝑔(𝐏(𝑦)) where 𝐏(𝑦) is the probability estimate of 𝑦 (e.g. [[Logistic Function - Sigmoid Function (Standard Logistic Function)|sigmoid function]])
- is a common evaluation metric for binary classification models
- it measures the performance of a model by quantifying the difference between predicted probabilities and actual values
- related to [[Univariate Entropy (Information Content - Entropy - Cross Entropy - KL Divergence)|cross entropy]]
````
^excerpt

# Applications
- used in [[Softmax Activation Function]]
- used in [[Binomial／Binary Logistic Regression (BLR)|Binomial/Binary Logistic Regression (BLR)]] and its generalized [[Logistic (Logit) Regression Model]]s
