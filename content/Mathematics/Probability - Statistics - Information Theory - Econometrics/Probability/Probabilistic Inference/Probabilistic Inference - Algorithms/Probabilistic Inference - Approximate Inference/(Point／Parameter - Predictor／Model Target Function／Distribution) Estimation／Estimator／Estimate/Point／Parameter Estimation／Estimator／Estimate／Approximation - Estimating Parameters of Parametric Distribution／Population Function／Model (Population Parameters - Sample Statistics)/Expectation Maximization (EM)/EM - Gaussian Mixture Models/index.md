# Using EM to Learn Parameters of a GMM

Consider the case where a dataset is comprised of many points that happen to be generated by two different processes (e.g. [[Univariate Normal／Gaussian／Gauss／Laplace-Gauss﻿ Distribution／Model／Process (Bell Curve)|Gaussian Distributions]]). The data is combined and the distributions are similar enough that it is not obvious to which distribution a given point may belong. 

The processes used to generate the data point represents a latent variable (e.g. process A and process B). It influences the data but is not observable.

methods in learning with [[Data Observability Types (Complete／Fully-Observable Data ｜ Incomplete／Partially-Observable Data ｜ Latent／Hidden／Hypothetical Variables)|latent variables]]:

- [[Expectation Maximization (EM)|Expectation Maximization Algorithm]] is an appropriate approach to use to estimate the parameters of the distributions

# Code Implementation

```py
# example of a bimodal constructed from two gaussian processes
import numpy as np
from numpy import hstack
from numpy.random import normal
from matplotlib import pyplot
from sklearn.mixture import GaussianMixture

# generate a sample
X1 = normal(loc=20, scale=5, size=3000)
X2 = normal(loc=40, scale=5, size=7000)
X = hstack((X1, X2))

# # plot the histogram
pyplot.hist(X, bins=50, density=True)
pyplot.show()

# reshape into a table with one column
X = X.reshape((len(X), 1))
# fit model
model = GaussianMixture(n_components=2, init_params='kmeans')
model.fit(X)

print('means: ' + str(model.means_))
print('covariances: ' + str(model.covariances_))
print('converged: ' + str(model.converged_))

# predict latent values
test = np.arange(0,60,0.5)
test = test.reshape(test.size, 1)
yhat = model.predict(test)
# check latent values
for i in range(0,120):
    print(str(yhat[i]) + ": " + str(test[i]))
```
