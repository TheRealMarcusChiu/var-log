<strong>Two-Stage Least Squares (2SLS)</strong> regression analysis

- is a [statistical technique](http://www.statisticssolutions.com/directory-of-statistical-analyses/) that is used in the analysis of [structural equations](http://www.statisticssolutions.com/resources/directory-of-statistical-analyses/structural-equation-modeling)
- this technique is the extension of the [[Ordinary Least Squares (OLS)|OLS]] method
- it is used when the dependent variable’s error terms are correlated with the independent variables
- additionally, it is useful when there are feedback loops in the model
- in [structural equations modeling](http://www.statisticssolutions.com/resources/directory-of-statistical-analyses/structural-equation-modeling), we use the maximum likelihood method to estimate the path coefficient. This technique is an alternative in SEM modeling to estimate the path coefficient. This technique can also be applied in quasi-experimental studies
