### Strict-Sense Stationarity

Let:

- {𝑋<sub>𝑡</sub>} - a [[Stochastic／Random／Markov Models／Process|stochastic process]]
- 𝐹<sub>𝑋</sub>(𝑥<sub>𝜏+𝑡<sub>1</sub></sub>, ..., 𝑥<sub>𝜏+𝑡<sub>𝑛</sub></sub>) the [[Probability Distribution Function Variants - Probability／Mass／Density／Cumulative／Survivor／Hazard／Cumulative-Hazard／Inverse／Moment-Generating Distribution Functions|cumulative distribution function]] of the unconditional (i.e., with no reference to any particular starting value) [joint distribution](https://en.wikipedia.org/wiki/Joint_distribution) of {𝑋<sub>𝑡</sub>} at times ![](https://wikimedia.org/api/rest_v1/media/math/render/svg/7f4a768913184f6318d9928686740f056505835a)

Then {𝑋<sub>𝑡</sub>} is said to be <strong>strictly stationary</strong>, <strong>strongly stationary</strong> or <strong>strict-sense stationary</strong> if

- 𝐹<sub>𝑋</sub>(𝑥<sub>𝜏+𝑡<sub>1</sub></sub>, ..., 𝑥<sub>𝜏+𝑡<sub>𝑛</sub></sub>) = 𝐹<sub>𝑋</sub>(𝑥<sub>𝑡<sub>1</sub></sub>, ..., 𝑥<sub>𝑡<sub>𝑛</sub></sub>) for all 𝜏,𝑡<sub>1</sub>,...,𝑡<sub>𝑛</sub>∊ℝ and for all 𝑛∊ℕ

Since 𝜏 does not affect 𝐹<sub>𝑋</sub>(..), 𝐹<sub>𝑋</sub> is not a function of time
