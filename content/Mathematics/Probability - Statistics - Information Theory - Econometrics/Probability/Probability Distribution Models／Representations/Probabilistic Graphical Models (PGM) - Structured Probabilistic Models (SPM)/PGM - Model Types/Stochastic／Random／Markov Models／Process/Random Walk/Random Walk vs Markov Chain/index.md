# Random Walk vs Markov Chain

A one-dimensional <strong>[[Random Walk|random walk]]</strong> can also be looked at as a <strong>[[Markov Chains／Chain (Transition Markov／Probability／Stochastic Matrix) - Discrete Time Markov Chains (DTMC)|Markov chain]]</strong> whose state space is given by the integers 𝑖 = 0, ±1, ±2, ... For some number 𝑝 satisfying 0 < 𝑝 < 1, the transition probabilities (the probability <em>𝑃<sub>𝑖,𝑗</sub></em> of moving from state 𝑖 to state 𝑗) are given by 𝑃<sub>𝑖,𝑖+1</sub>= 𝑝 = 1 - 𝑃<sub>𝑖,𝑖-1</sub>
