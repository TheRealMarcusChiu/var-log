###### Hierarchical Hidden Markov Model (HHMM)

```excerpt
- is a statistical model derived from the [[Hidden Markov Models (HMM)|Hidden Markov Model (HMM)]]
- in an HHMM each state is considered to be a self-contained probabilistic model. More precisely each state of the HHMM is itself an HHMM
```

^excerpt

# Two kinds of Hierarchical Markov Models

- [Hierarchical Hidden Markov model](https://en.wikipedia.org/wiki/Hierarchical_hidden_Markov_model)
- Abstract Hidden Markov Model

Both have been used for behavior recognition. and certain conditional independence properties between different levels of abstraction in the model allow for faster learning and inference
