the <strong>n-step transient-state probability distribution vector</strong> (𝜋<sub>𝑛</sub>) is defined as:

- 𝜋<sub>𝑛</sub> = 𝜋<sub>0</sub>·𝑇<sup>𝑛</sup><font style="color: rgb(128,128,128);"># matrix-vector form</font>

where:

- 𝜋<sub>0</sub>- is the initial starting state probabilities
- 𝑇 - is the transition state matrix

if 𝜋<sub>𝑛</sub> = 𝜋<sub>0</sub>then 𝜋<sub>𝑛</sub> is the [[Markov Chain Property - Stationary／Stable／Invariant／Steady-State／Limiting／Marginal Probability Distribution Vector|stationary distribution]]
