# Multivariate vs Mixture

```merge-table
{
  "rows": [
    [
      {
        "content": "Multivariate",

      },

        "header": true,
        "bg": "#F4F5F7"
      }
    ],
    [
      "- a Multivariate Model is defined as a 𝑘 dimensional random vector",
      "- a Mixture Model is defined as a random variable (which you can call a 1-dimensional random vector)"
    ],
    [
      "- a Multivariate Model will strictly have a joint distribution of multiple variables",
      "- a Mixture Model can have either a distribution of one variable or joint distribution of more than one variable as the underlying distribution"

}
```
