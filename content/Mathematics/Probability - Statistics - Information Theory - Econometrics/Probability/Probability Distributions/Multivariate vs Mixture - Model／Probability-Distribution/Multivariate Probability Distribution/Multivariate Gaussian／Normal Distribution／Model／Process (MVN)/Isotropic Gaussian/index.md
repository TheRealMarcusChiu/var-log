###### Isotropic Gaussian

```excerpt
- is a [[Multivariate Gaussian／Normal Distribution／Model／Process (MVN)|multivariate normal distribution]] that looks the <em>same in every direction</em>
```

^excerpt

# Definition

A random vector 𝑥∈ℝ<sup>𝑑</sup> follows an <strong>isotropic Gaussian</strong> if:

- $x \sim \mathcal{N}(0, \sigma^2 I)$

where:

- mean = 0
- covariance = 𝜎<sup>2</sup>𝐼 (a scalar times the [[Identity Matrix|identity matrix]] = [[Covariance & Correlation Matrix|covariance matrix]])
- all dimensions have the same variance
- all dimensions are uncorrelated
