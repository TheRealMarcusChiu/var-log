<strong>Maxwell–Boltzmann Distribution</strong>

- used to describe particle speeds in idealized gases
- should not be confused with the [[Boltzmann／Gibbs Distribution|Boltzmann/Gibbs Distribution]] which gives the probability that a system will be in a certain state as a function of that state's energy
