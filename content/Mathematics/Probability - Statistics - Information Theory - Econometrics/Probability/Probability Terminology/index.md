- <strong>domain space</strong> or <strong>sample space</strong> or <strong>possibility space</strong>
	- is a set of all possible outcomes
	- denoted as 𝛺
- <strong>empty event</strong>
	- denoted as ∅
- <strong>event</strong>
	- is a set of outcomes
	- is a subset of sample space
- <strong>event space</strong> - is a set of events
- 𝐏 - is the [[Probability Measure|probability measure]]
- 𝐏(𝛢) - is the [[Probability|probability]] of the event 𝛢 happening
- <strong>[[Probability Spaces (Sample Space - Event Space - Probability Measure)|probability space]]</strong> (𝛺,𝐹,𝑃)
