###### Independent and Identically Distributed (IID)

```excerpt
- a collection of random variables is <strong>independent and identically distributed</strong> if each random variable has the same [[Probability Distributions|probability distribution]] as the others and all are mutually [[Probability Independence (Marginal Independence - Conditional Independence)|independent]]
```
