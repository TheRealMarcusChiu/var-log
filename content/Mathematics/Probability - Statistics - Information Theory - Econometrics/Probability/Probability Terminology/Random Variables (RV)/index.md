###### Random Variables (RV)
````excerpt
- is a [[Real-Valued Function (Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms)|real-valued function]] 𝑋 : 𝛺 → ℝ where 𝛺 is the [[Probability Spaces (Sample Space - Event Space - Probability Measure)|sample space]]
````
^excerpt

# RV - Examples

The value of 2 dice rolls. For example (1,4)∊𝛺, then:
- if 𝑋 = sum of 2 dice rolls, then 𝑋(1,4) = 5
- if 𝑋 = diff of 2 dice rolls, then 𝑋(1,4) = -3
- etc
