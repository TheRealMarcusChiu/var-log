# Statistical Inference Paradigms - Types
- <strong>[[Bayesian (Statistical Inference - Inferential Statistics)|Bayesian Inference]]</strong> - is a type of statistical inference based on the Bayesian interpretation of probability where probability expresses a [[Probability as Degrees of Belief|degree of belief]] in an event
- <strong>[[Frequentist (Statistical Inference - Inferential Statistics)|Frequentist Inference]]</strong> - is a type of statistical inference that draws the probability of sample data by emphasizing the [[Probability as Long-Term Frequency|long-term frequency]] of an event
- <strong>[[Classical (Statistical Inference - Inferential Statistics)|Classical Inference]]</strong> -
- <strong>Likelihood-based Inference</strong> - a subtype of Frequentist Inference. approaches statistics by using the [[Likelihood Function - Log-Likelihood Function - Score Function|likelihood function]]. Some likelihoodists reject inference, considering statistics as only computing support from evidence. Others, however, propose inference based on the likelihood function, of which the best-known is [[Maximum Likelihood Estimation (MLE)|maximum likelihood estimation]]
- <strong>AIC-Based Inference</strong> - based on [[Akaike's Information Criterion／Criteria (AIC)|Akaike's Information Criterion]]
