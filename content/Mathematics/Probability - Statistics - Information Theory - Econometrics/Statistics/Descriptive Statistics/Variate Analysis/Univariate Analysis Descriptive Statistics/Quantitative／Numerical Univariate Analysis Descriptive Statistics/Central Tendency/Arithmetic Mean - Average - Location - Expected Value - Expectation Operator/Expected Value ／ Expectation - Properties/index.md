###### Properties of Expected Value / Expectation

```excerpt
- 𝐄\[𝑋 + 𝑌\] = 𝐄\[𝑋\] + 𝐄\[𝑌\]
- 𝐄\[𝑎𝑋\] = 𝑎𝐄\[𝑋\]
- 𝐄\[𝑐\] = 𝑐
- 𝐄\[𝑎𝑋 + 𝑏𝑌 + 𝑐\] = 𝑎𝐄\[𝑋\] + 𝑏𝐄\[𝑌\] + 𝑐
- 𝐄\[𝑋𝑌\] = 𝐄\[𝑋\]𝐄\[𝑌\] <font style="color: rgb(128,128,128);">\# for </font>[[Probability Independence (Marginal Independence - Conditional Independence)|independent]]<font style="color: rgb(128,128,128);"> 𝑋 and 𝑌</font>
- 𝐄\[𝑋𝑌\] = 𝐶𝑜𝑣(𝑋,𝑌) + 𝐄\[𝑋\]𝐄\[𝑌\] <font style="color: rgb(128,128,128);">\# for non-independent 𝑋 and 𝑌, 𝐶𝑜𝑣(𝑋,𝑌) is the [[Covariation - Covariance|covariance]]</font>
- 𝐄\[𝑋<sup>2</sup>\] = 𝑉𝑎𝑟(𝑋) + 𝐄\[𝑋\]<sup>2</sup><font style="color: rgb(128,128,128);">\# derived from</font><font style="color: rgb(122,134,154);"> the</font> [[Raw Moments - Moments About the Origin／Zero - Central／Mean Moments - Moments About The Mean - (Zeroth／First／Second／Third／Fourth／kth／nth) -  of a Probability Density Function|second central moment]]
```
