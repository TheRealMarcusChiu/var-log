###### Quartiles (4-Quantile)

```excerpt
- is a type of [[p-Quantiles (Median - Quartiles - Percentiles)|quantile]] which divides the number of data points into four more or less equal parts, or quarters
- the first quartile is defined as the middle number between the smallest number and the [[Median (2-Quantile - 50th Percentile)|median]] of the data set
```
