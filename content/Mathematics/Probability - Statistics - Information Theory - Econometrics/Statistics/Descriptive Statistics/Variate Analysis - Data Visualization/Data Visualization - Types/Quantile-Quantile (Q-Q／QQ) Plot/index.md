###### Quantile-Quantile (Q-Q/QQ) Plot

- is a scatterplot created by plotting two sets of [[p-Quantiles (Median - Quartiles - Percentiles)|quantiles]] against one another. If both sets of quantiles came from the same [[Probability Distributions|distribution]] (e.g. [[Univariate Normal／Gaussian／Gauss／Laplace-Gauss﻿ Distribution／Model／Process (Bell Curve)|normal]], [[Exponential Distribution|exponential]], etc), we should see the points forming a line that’s roughly straight
