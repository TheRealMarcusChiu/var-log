# Terminology
- <strong>measure</strong> - a <em>measure</em> on a set is a systematic way to assign a number to each suitable subset of that set, intuitively interpreted as its size. In this sense, a measure is a generalization of the concepts of length, area, and volume
- <strong>measure zero</strong> - a set of measure zero occupies no volume in the space we are measuring
- <strong>almost everywhere</strong> - a property that holds almost everywhere holds throughout all of the space except for on a set of measure zero
