###### Complete Measure Spaces

```excerpt
- is a type of [[Mathematical Spaces|mathematical space]]
- is a type of [[Measure Spaces|measure space (𝑋,𝐴,𝜇)]] in which every subset of every null set is measurable (having [measure zero](https://en.wikipedia.org/wiki/Measure_zero)):
	- \[ 𝑁 ∈ 𝐴 and 𝜇(𝑁) = 0 and 𝑆 ⊆ 𝑁 \] ⇒ 𝑆 ∈ 𝐴
```
