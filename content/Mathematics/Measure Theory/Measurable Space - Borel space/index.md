###### Measurable Space - Borel space
````excerpt
- a type of [[Mathematical Spaces|mathematical space]]
- is a tuple (𝑋,𝐹) where:
	- 𝑋 is a set
	- 𝐹 is a [[σ-Algebra - σ-Field of Sets - σ-field|σ-algebra]] on 𝑋
- is a basic object in [[Measure Theory|measure theory]]
- it consists of a [[Set Theory|set]] and a [[σ-Algebra - σ-Field of Sets - σ-field|σ-algebra]], which defines the subsets that will be measured
- note that in contrast to [[Measure Spaces|measure space]], no [measure](https://en.wikipedia.org/wiki/Measure_(mathematics)) is needed for a measurable space
````
