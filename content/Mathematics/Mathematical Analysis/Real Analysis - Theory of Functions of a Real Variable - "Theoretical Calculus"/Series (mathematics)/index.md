# Series - Definition

Let (𝑎<sub>𝑘</sub>)<sub>𝑘∊ℕ</sub> be a sequence. The sequence (𝑠<sub>𝑛</sub>)<sub>𝑛∊ℕ</sub> given by:
- 𝑠<sub>𝑛</sub> = 𝛴<sub>1≤𝑘≤𝑛</sub>𝑎<sub>𝑘</sub>

is called a series.

If (𝑠<sub>𝑛</sub>)<sub>𝑛∊ℕ</sub> is convergent, we write:
- 𝛴<sub>1≤𝑘≤∞</sub>𝑎<sub>𝑘</sub> = 𝑙𝑖𝑚<sub>𝑛→∞</sub>𝑠<sub>𝑛</sub> = 𝑙𝑖𝑚<sub>𝑛→∞</sub>𝛴<sub>1≤𝑘≤𝑛</sub>𝑎<sub>𝑘</sub>

# Series - Properties

If 𝛴<sub>1≤𝑘≤𝑛</sub>𝑎<sub>𝑘</sub>and 𝛴<sub>1≤𝑘≤𝑛</sub>𝑏<sub>𝑘</sub> are both convergent, then:
- 𝛴<sub>1≤𝑘≤𝑛</sub>(𝑎<sub>𝑘</sub> + 𝑏<sub>𝑘</sub>) is also convergent with the limit 𝛴<sub>1≤𝑘≤𝑛</sub>(𝑎<sub>𝑘</sub> + 𝑏<sub>𝑘</sub>) = 𝛴<sub>1≤𝑘≤𝑛</sub>(𝑎<sub>𝑘</sub>) + 𝛴<sub>1≤𝑘≤𝑛</sub>(𝑏<sub>𝑘</sub>)
- 𝛴<sub>1≤𝑘≤𝑛</sub>(𝜆·𝑎<sub>𝑘</sub>) is also convergent with the limit 𝛴<sub>1≤𝑘≤𝑛</sub>(𝜆·𝑎<sub>𝑘</sub>) = 𝜆·𝛴<sub>1≤𝑘≤𝑛</sub>(𝑎<sub>𝑘</sub>)
