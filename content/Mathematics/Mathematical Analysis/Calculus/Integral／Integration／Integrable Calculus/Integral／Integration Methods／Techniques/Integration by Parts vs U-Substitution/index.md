# Integration by Parts vs. U-Substitution

- <strong>[[U-Substitution - Integration by Substitution - Reverse Chain Rule - Change of Variables|U-substitution]]</strong> can be done whenever you have something containing a function (we'll call this 𝑔), and that something is multiplied by the derivative of 𝑔
  - that is if you have ∫𝑓(𝑔(𝑥))𝑔′(𝑥)𝑑𝑥, use a u-sub
- <strong>[[Integration by Parts|Integration by parts]]</strong> is whenever you have two functions multiplied together--one that you can integrate, one that you can differentiate
