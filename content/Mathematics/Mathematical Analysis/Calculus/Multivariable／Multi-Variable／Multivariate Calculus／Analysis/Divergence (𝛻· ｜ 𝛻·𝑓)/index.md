###### Divergence (𝛻· | 𝛻·𝑓)

```excerpt
- is a [[Linear (Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms) - Vector Space Homomorphism|linear operator]] that takes in a [[Vector-Valued Functions - Vector Fields|vector-valued function]] 𝑓 that defines a vector field and outputs a scalar-valued function measuring the change in density of the "fluid" at each point
```

^excerpt

# Divergence - Definition

The divergence of a vector-valued function 𝑣̅:

- $div \; \overline{v} = ∇·v = \frac{𝜕v_1}{𝜕x} + \frac{𝜕v_2}{𝜕y} + ...$

where:

- $v_1, v_2, ... \text{ are the components of } \overline{v}$

# Divergence - Properties

The divergence operator 𝛻· is a [[Linear (Transformations／Operations／Operators／Mappings／Maps／Functions／Morphisms) - Vector Space Homomorphism|linear operator]] such that it follows for any vector-valued functions 𝑓<sub>1</sub> and 𝑓<sub>2</sub>, and scalar constants 𝛼 and 𝛽:

- $𝛻·(𝛼f_1 + 𝛽f_2) = 𝛼(𝛻·f_1) + 𝛽(𝛻·f_2)$
