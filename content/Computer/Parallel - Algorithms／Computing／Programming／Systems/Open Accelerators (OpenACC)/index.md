###### Open Accelerators (OpenACC)

- is a programming standard for [[Parallel - Algorithms／Computing／Programming／Systems|parallel computing]]
- developed by Cray, CAPS, Nvidia and PGI
- the standard is designed to simplify parallel programming of heterogeneous CPU/GPU systems
