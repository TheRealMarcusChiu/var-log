###### Open Multi-Processing (OpenMP)

```excerpt
- is an API standard that supports multi-platform shared-memory multiprocessing programming in [[C]], [[C++]], and [[Fortran (Formula Translator)|Fortran]], on many platforms, [[Instruction Set Architecture (ISA)|instruction-set architectures]], and [[Operating Systems (OS)|operating systems]], including [[Solaris (Oracle Solaris)|Solaris]], AIX, [[FreeBSD]], HP-UX, [[Linux]], [[Macintosh OSs (Mac OS X) - macOS|macOS]], and [[Windows]]
- consists of a set of [[Compiler／Compilers|compiler]] directives, library routines, and environment variables that influence run-time behavior
```

^excerpt

# OpenMP vs MPI

- OpenMP is used for parallelism within a (multi-core) node
- Message Passing Interface (MPI) is used for parallelism between nodes
