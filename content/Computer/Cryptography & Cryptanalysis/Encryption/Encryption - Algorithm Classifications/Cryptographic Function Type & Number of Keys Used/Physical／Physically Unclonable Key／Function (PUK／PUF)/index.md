###### Physical/Physically Unclonable Key/Function (PUK/PUF)

```excerpt
- is a physically-defined "digital fingerprint" that serves as a unique identifier for a semiconductor device such as a [[Central Processing Unit (CPU) - Central／Main Processor - Microprocessor (uP) - Microcontroller (uC) - Microcontroller Unit (MCU)|microprocessor]]
```
