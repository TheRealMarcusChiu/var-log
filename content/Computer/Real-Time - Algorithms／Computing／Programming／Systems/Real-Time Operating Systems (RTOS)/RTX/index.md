###### RTX

- is a line of [[Real-Time Operating Systems (RTOS)|real-time operating system]] extensions by the firm IntervalZero
- is a software extension or abstraction layer that converts the Microsoft [[Windows|Windows operating system]] into an [[Real-Time Operating Systems (RTOS)|RTOS]]
