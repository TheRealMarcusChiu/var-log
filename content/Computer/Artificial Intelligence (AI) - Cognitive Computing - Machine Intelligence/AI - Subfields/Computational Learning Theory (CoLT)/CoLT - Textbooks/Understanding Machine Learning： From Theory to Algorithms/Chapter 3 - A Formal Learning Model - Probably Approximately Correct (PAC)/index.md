In this chapter, we define our main formal learning model – the PAC learning model and its extensions. We will consider other notions of learnability in Chapter 7.
# PAC Learning

In the previous chapter, we have shown that for a finite hypothesis class, if the ERM rule with respect to that class is applied on a sufficiently large training sample (whose size is independent of the underlying distribution or labeling function) then the output hypothesis will be probably approximately correct. More generally, we now define Probably Approximately Correct (PAC) learning.
###### PAC Learnability - Definition
> [!info]
> A hypothesis class 𝐻 is <strong>PAC learnable</strong> if there exists both:
> - a function 𝑚<sub>𝐻</sub> : (0,1)<sup>2</sup> → ℕ
> - a learning algorithm
>
> with the following properties:
> - for every 𝜀,𝛿∊(0,1)
> - for every distribution 𝒟 over 𝒳
> - for every labeling function 𝑓 : 𝒳 → {0, 1}
>
> if:
> - the <em>[[The Realizability Assumption|realizable assumption]]</em> holds concerning 𝐻,𝒟,𝑓
>
> then:
> - when running the learning algorithm on 𝑚 ≥ 𝑚<sub>𝐻</sub>(𝜀,𝛿) i.i.d. examples generated by 𝒟 and labeled by 𝑓, the algorithm returns a hypothesis ℎ such that, with a probability of at least 1 - 𝛿 (over the choice of the examples), 𝐿<sub>(𝒟,𝑓)</sub>(ℎ) ≤ 𝜀.
