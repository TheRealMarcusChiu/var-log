###### ComfyUI-Manager
````excerpt
- is an extension designed to enhance the usability of [[ComfyUI]]
- it offers management functions to install, remove, disable, and enable various custom nodes of ComfyUI
- this extension provides a hub feature and convenience functions to access a wide range of information within ComfyUI
````
^excerpt

# Installation \[method1\] (General installation method: ComfyUI-Manager only)

To install ComfyUI-Manager in addition to an existing installation of ComfyUI, you can follow the following steps:
- Go to <code><font style="color: rgb(122,134,154);">ComfyUI/custom\_nodes</font></code> dir in terminal (cmd)
- <code><font style="color: rgb(122,134,154);">git clone https://github.com/ltdrdata/ComfyUI-Manager comfyui-manager</font></code>
- Restart ComfyUI
