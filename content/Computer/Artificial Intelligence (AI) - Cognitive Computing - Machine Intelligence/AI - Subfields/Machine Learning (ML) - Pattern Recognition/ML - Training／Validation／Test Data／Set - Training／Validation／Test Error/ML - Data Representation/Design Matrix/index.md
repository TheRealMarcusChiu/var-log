In [[Statistics|statistics]], a <strong>design matrix</strong>, also known as <strong>model matrix</strong> or <strong>regressor matrix</strong> and often denoted by 𝑋, is a matrix of values of explanatory variables of a set of objects

- each row represents an individual object
- each column corresponds to a variable value for that object
