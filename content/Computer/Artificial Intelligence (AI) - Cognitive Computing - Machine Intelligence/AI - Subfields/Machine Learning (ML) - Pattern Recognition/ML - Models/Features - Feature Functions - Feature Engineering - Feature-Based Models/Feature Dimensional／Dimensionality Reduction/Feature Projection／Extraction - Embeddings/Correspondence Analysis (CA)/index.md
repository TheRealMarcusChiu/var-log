###### Correspondence Analysis (CA)

```excerpt
- is a multivariate statistical technique for analyzing [[Data／Variable Types - Measurement Scales (Nominal／Categorical／Factor - Ordinal - Interval - Ratio)|categorical]] data
- the main objective of CA is to visualize rows and columns of a data table in a low-dimensional space, usually two-dimensional
- is conceptually similar to [[Principal Component Analysis (PCA)|principal component analysis (PCA)]], but applies to categorical rather than continuous data
```

^excerpt
