###### Isomap
````excerpt
- is a [[Non-Linear／Nonlinear Dimensionality Reduction (NLDR) - Manifold Learning|nonlinear dimensionality reduction]] method
- is one representative of isometric mapping methods
- extends [[Distance Measures／Metrics／Semi-Metrics／Functions|metric]] [[Multidimensional Scaling (MDS)|multidimensional scaling (MDS)]] by incorporating the geodesic distances imposed by a weighted graph
````
