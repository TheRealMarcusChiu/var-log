###### Non-Linear/Nonlinear Dimensionality Reduction (NLDR) - Manifold Learning
````excerpt
- refers to various related techniques that aim to project high-dimensional data onto lower-dimensional latent [[Manifolds (n-Dimensional Manifold (n-Manifold)|manifolds]] (using the assumption of the [[Manifold Hypothesis|manifold hypothesis]])
- is a technique that finds a non-linear manifold within a higher-dimensional space
- the goal of <em>manifold learning</em> is to project high-dimensional data onto lower-dimensional latent manifolds
- are generalizations of linear decomposition methods used in dimensionality reduction (e.g. [[Singular Value Decomposition／Factorization (SVD) - Reduced SVD|singular value decomposition]] and [[Principal Component Analysis (PCA)|principal component analysis]])
````
