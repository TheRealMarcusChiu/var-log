###### Neighborhood Component Analysis
````excerpt
- is a [[ML - Experience Type (Supervised／Unsupervised／Semi-Supervised／Self-Supervised／Multi-Instance／Reinforcement Learning)|supervised learning method]] for classifying multivariate data into distinct classes according to a given distance metric over the data
- functionally, it serves the same purposes as the [[k-Nearest Neighbors (k-NN) Regression|K-nearest neighbors]] algorithm and makes direct use of a related concept termed stochastic nearest neighbors
````
