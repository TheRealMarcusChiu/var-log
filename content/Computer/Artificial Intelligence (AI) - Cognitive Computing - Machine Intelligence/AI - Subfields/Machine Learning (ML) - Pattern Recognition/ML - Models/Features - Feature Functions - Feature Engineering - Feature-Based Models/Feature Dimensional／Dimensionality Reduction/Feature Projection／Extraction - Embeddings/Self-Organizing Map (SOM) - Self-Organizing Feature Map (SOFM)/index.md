###### Self-Organizing Map (SOM) - Self-Organizing Feature Map (SOFM)

- s an [[Unsupervised Learning|unsupervised]][[Machine Learning (ML) - Pattern Recognition|machine learning]] technique used to produce a low-dimensional (typically two-dimensional) representation of a higher dimensional data set while preserving the [topological structure](https://en.wikipedia.org/wiki/Topology) of the data
