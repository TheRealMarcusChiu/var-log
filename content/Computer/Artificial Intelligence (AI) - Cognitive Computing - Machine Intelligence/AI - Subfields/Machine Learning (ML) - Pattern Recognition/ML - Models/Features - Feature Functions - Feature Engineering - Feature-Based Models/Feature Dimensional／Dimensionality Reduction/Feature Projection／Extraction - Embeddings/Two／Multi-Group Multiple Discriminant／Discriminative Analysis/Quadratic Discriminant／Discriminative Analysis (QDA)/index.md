###### Quadratic Discriminant/Discriminative Analysis (QDA)

- is a variant of [[Linear／Normal Discriminant／Discriminative Analysis (LDA／NDA)|LDA]] in which an individual [[Covariance & Correlation Matrix|covariance matrix]] is estimated for every class of observations
- QDA is particularly useful if there is prior knowledge that individual classes exhibit distinct covariances
- a disadvantage of QDA is that it cannot be used as a [[Feature Dimensional／Dimensionality Reduction|dimensionality reduction]] technique
