# Principal Component Analysis (PCA) History

[[Principal Component Analysis (PCA)]] was invented in 1901 by [Karl Pearson](https://en.wikipedia.org/wiki/Karl_Pearson) as an analog of the [principal axis theorem](https://en.wikipedia.org/wiki/Principal_axis_theorem) in mechanics; it was later independently developed and named by [Harold Hotelling](https://en.wikipedia.org/wiki/Harold_Hotelling) in the 1930s

Depending on the field of application, it is also named:

- Discrete [Karhunen–Loève](https://en.wikipedia.org/wiki/Karhunen%E2%80%93Lo%C3%A8ve_theorem) Transform (KLT) in [signal processing](https://en.wikipedia.org/wiki/Signal_processing)
- [Hotelling](https://en.wikipedia.org/wiki/Harold_Hotelling) Transform in multivariate quality control
- Proper Orthogonal Decomposition (POD) in mechanical engineering
- [[Singular Value Decomposition／Factorization (SVD) - Reduced SVD|Singular Value Decomposition/Factorization (SVD) - Reduced SVD]] of <strong>X</strong> (Golub and Van Loan, 1983)
- [[Eigen Decomposition／Factorization - Diagonalization／Diagonalizing／Diagonalize - Diagonalizable／Non-Defective Matrix - Non-Diagonalizable／Defective Matrix|Eigenvalue Decomposition (EVD)]] of <strong>X</strong><sup>T</sup><strong>X</strong> in [[Linear Algebra|linear algebra]]
- [Factor Analysis](https://en.wikipedia.org/wiki/Factor_analysis) (for a discussion of the differences between PCA and factor analysis see Ch. 7 of Jolliffe's <em>Principal Component Analysis</em>)
- [Eckart–Young Theorem](https://en.wikipedia.org/wiki/Eckart%E2%80%93Young_theorem) (Harman, 1960)
- [Empirical Orthogonal Function (EOF)](https://en.wikipedia.org/wiki/Empirical_orthogonal_functions) in meteorological science
- [Empirical Eigenfunction Decomposition](https://en.wikipedia.org/w/index.php?title=Empirical_eigenfunction_decomposition\&action=edit\&redlink=1) (Sirovich, 1987)
- [Empirical Component Analysis](https://en.wikipedia.org/w/index.php?title=Empirical_component_analysis\&action=edit\&redlink=1) (Lorenz, 1956)
- [Quasiharmonic Modes](https://en.wikipedia.org/w/index.php?title=Quasiharmonic_modes\&action=edit\&redlink=1) (Brooks et al., 1988)
- [Spectral Decomposition](https://en.wikipedia.org/wiki/Spectral_theorem) in noise and vibration
- [Empirical Modal Analysis](https://en.wikipedia.org/wiki/Mode_shape) in structural dynamics
