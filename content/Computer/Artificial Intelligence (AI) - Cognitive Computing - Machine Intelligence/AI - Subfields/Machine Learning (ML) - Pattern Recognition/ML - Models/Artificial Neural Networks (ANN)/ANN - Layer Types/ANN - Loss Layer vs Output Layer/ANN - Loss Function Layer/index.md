###### Loss Function Layer

```excerpt
- used to train the model, and is discarded after training
- specifies how training penalizes the deviation between:
	- the predicted output of the network
	- the true data labels (during [[ML - Experience Type (Supervised／Unsupervised／Semi-Supervised／Self-Supervised／Multi-Instance／Reinforcement Learning)|supervised learning]])
- various [[Objective Function／Criterion - Cost／Loss／Error Function - Expected Cost／Loss／Error|loss functions]] can be used, depending on the specific task
```
