###### Sparse Autoencoders (SAE)
````excerpt
- is a type of [[Artificial Neural Networks (ANN)|neural network]] used for unsupervised [[Features - Feature Functions - Feature Engineering - Feature-Based Models|feature]] learning and [[Feature Dimensional／Dimensionality Reduction|dimensionality reduction]]
- it is a variant of a standard [[Autoencoders (AE)|autoencoder]], but with an added sparsity constraint on the hidden layer activations
````
^excerpt

# Use Cases
- can be used for [[LLM Interpretability]] to see what's going on within a neural network
