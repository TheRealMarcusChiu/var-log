###### Latent-prediction JEPA (LpJEPA)
````excerpt
- is a version of [[Joint Embedding Predictive Architecture (JEPA)|JEPA]] where a model learns by predicting hidden representations (latents) of missing or future data instead of predicting the raw data itself
- for improved version see: [[Rectified LpJEPA]]
````
