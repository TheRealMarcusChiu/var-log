###### Generative Pre-trained Transformer (GPT)
````excerpt
- a type of advanced AI [[Language Models|language model]] based on the [[Transformer Neural Networks (TNN) - Transformers|transformer neural network]] architecture
- can generate human-like text and perform various tasks by predicting the most likely next word in a sequence, after being "pre-trained" on vast amounts of text and code data
- these models are used to create content, answer questions, translate languages, and power chatbots and virtual assistants
````
