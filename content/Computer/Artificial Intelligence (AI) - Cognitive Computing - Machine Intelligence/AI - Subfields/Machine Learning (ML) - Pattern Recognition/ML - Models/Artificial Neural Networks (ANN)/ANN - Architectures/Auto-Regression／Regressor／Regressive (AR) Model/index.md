###### Auto-Regression/Regressor/Regressive (AR) Model

```excerpt
- is a representation of a type of [[Stochastic／Random／Markov Models／Process|random process]]; as such, it can be used to describe certain time-varying processes in nature, economics, behavior, etc
- are a class of [[ML - Models|machine learning (ML) models]] that automatically predict the next component in a sequence by taking measurements from previous inputs in the sequence
- can be used for [[Image Generation|image generation]]
```
