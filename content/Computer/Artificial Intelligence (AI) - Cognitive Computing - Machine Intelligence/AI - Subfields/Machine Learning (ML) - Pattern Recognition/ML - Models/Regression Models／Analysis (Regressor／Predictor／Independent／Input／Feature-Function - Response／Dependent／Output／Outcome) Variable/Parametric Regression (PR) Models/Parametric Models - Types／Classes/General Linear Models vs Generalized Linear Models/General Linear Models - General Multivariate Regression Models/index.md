###### General Linear Models - General Multivariate Regression Models

```excerpt
- is a compact way of simultaneously writing several [[Multivariate／Multiple Linear Regression Models|multiple linear regression models]] compactly written as:
	- 𝑌 = 𝑋𝛽 + 𝛴
- where:
	- 𝑌 - is a matrix with a series of multivariate measurements (each column is a set of measurements on one of the dependent variables
	- 𝑋 - is a matrix of observations on independent variables that might be a design matrix (each column being a set of observations on one of the independent variables)
	- 𝛽 - is a matrix containing parameters that are usually estimated
	- 𝛴 - is a matrix containing errors/noise
```

^excerpt
