###### Negative Binomial Regression Model (NB2)

- [[Poisson Regression Model|Poisson Regression]] assumes that the [[Variance - Standard Deviation (Root Mean Square (RMS) Deviation)|variance]] equals the [[Arithmetic Mean - Average - Location - Expected Value - Expectation Operator|mean]]. When the variance is greater than the mean, your model has overdispersion. NB2 can be more appropriate when overdispersion is present
