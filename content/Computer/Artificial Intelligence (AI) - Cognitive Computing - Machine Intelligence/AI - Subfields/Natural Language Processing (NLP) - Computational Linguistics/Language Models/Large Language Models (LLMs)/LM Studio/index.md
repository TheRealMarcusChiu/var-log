###### LM Studio

```excerpt
- <strong>Local AI platform</strong> for running LLMs on your own machine (offline)
- <strong>User-friendly GUI</strong> with model catalog and chat interface
- <strong>Supports open-source models</strong> ([[GPT-Generated Unified Format (GGUF)|GGUF]], [[LLaMA]], etc) and local inference
- <strong>API & SDK</strong> for developers, including RAG/document chat
- <strong>Private & offline</strong> - no cloud needed, all data stays on device
```

^excerpt
