###### Latent Semantic Analysis (LSA)

```excerpt
- is a [[Natural Language Processing (NLP) - Computational Linguistics|natural language processing]] method that uses [[Singular Value Decomposition／Factorization (SVD) - Reduced SVD|singular value decomposition]] to analyze relationships between documents and the terms they contain
- uses a mathematical technique to scan unstructured data to find hidden relationships between terms and concepts
- is a popular [[Feature Dimensional／Dimensionality Reduction|dimensionality-reduction]] technique that reformulates text data in terms of latent features
```
