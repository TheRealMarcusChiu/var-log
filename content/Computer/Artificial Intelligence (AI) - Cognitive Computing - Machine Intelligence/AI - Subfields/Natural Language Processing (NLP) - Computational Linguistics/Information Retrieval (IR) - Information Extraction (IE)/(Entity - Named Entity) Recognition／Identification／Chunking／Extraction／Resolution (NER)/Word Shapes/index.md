###### Word Shapes
- maps words to simplified representation that encodes attributes such as length, capitalization, numerals, Greek Letters, internal punctuation, etc

### Examples

```merge-table
{
  "rows": [
    [
      {
        "content": "Varicella-zoster",
        "header": true,
        "bg": "#F4F5F7"
      },
      "Xx-xxx"
    ],
    [
      {
        "content": "mRNA",
        "header": true,
        "bg": "#F4F5F7"
      },
      "xXXX"
    ],
    [
      {
        "content": "CPA1",
        "header": true,
        "bg": "#F4F5F7"
      },
      "XXXd"
    ]
  ]
}
```
