###### Radiance Fields

```excerpt
- is a function representing a 3D scene by mapping each point in space (and its viewing direction) to a color and density value
- is a 3D representation reconstructed from a collection of 2D images or video that creates lifelike outputs
- there are several different radiance field representations, but the two most popular ones are:
	- [[Neural Radiance Fields (NeRFs)]]
	- [[3D Gaussian Splatting (3DGS)]] -  has broken through recently as it renders in real time
	- [[Sparse Voxels Rasterization (SVRaster)]]
```

^excerpt
