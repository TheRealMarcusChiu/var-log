# Difference Between <u>Instruction Set Architecture</u> & <u>Computer Organization</u>

For example, both [[Intel® Processors／CPUs - Intel Graphics Technology|Intel]] and [[AMD Processors／CPUs|AMD]] processors have the same [[x86 x64 (x86-16, x86-32, x86-64)|x86 architecture]] (having the same ISA), but how the two companies implement that architecture (i.e. computer organization) is usually very different. The same programs run correctly on both because the architecture is the same, but they may run at different speeds because the organizations are different
