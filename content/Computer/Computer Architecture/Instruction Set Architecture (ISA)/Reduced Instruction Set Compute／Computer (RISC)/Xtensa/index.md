###### Xtensa

```excerpt
- is a configurable CPU architecture (ISA + microarchitecture)
- originally developed by Tensilica and now owned by Cadence Design Systems
- it’s widely used in embedded systems, where designers want a processor tailored exactly to their workload rather than a fixed, one-size-fits-all core
```

^excerpt
