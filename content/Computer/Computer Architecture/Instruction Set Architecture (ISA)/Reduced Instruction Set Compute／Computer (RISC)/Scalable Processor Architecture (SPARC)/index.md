###### Scalable Processor Architecture (SPARC)

- is a [[Reduced Instruction Set Compute／Computer (RISC)|reduced instruction set computer (RISC)]] instruction set architecture originally developed by Sun Microsystems
- its design was strongly influenced by the experimental Berkeley RISC system developed in the early 1980s
