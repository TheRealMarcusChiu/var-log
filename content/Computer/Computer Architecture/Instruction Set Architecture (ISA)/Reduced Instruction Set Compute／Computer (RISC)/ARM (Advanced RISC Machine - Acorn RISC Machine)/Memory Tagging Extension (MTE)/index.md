###### Memory Tagging Extension (MTE)

```excerpt
- is a hardware security feature in modern ARM processors (especially ARMv8.5-A and later) designed to detect memory safety bugs such as:
	- buffer overflows
	- use-after-free
	- out-of-bounds memory access
- it helps developers find memory errors at runtime with very low performance overhead
```

^excerpt
