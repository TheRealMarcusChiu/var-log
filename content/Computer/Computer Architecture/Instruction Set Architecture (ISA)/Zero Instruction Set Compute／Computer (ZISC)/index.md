###### Zero Instruction Set Compute/Computer (ZISC)

- refers to an [[Instruction Set Architecture (ISA)|ISA]] based solely on pattern matching and the absence of [[Opcode - Assembly Language - Machine Code／Language／Instruction|machine language]] in the classical sense
- chips built on ZISC are known for being thought of as comparable to the [[Artificial Neural Networks (ANN)|neural networks]], being marketed for the number of "synapses" and "neurons"
