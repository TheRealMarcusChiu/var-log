###### Complex Instruction Set Compute/Computer (CISC)

- is a type of [[Instruction Set Architecture (ISA)]]
- complex instructions
- requires fewer instructions and therefore less memory than RISC
- instructions take a variable number of clock cycles

# CISC - Types

- System/360
- VAX
- PDP-11
- Motorola 68000 family
