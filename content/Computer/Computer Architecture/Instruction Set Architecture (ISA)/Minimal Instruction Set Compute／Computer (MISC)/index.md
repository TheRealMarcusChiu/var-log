###### Minimal Instruction Set Compute/Computer (MISC)

- is an [[Instruction Set Architecture (ISA)|ISA]] with a very small number of basic operations and corresponding [[Opcode - Assembly Language - Machine Code／Language／Instruction|opcodes]]
- are commonly stack-based rather than register-based to reduce the size of operand specifiers
