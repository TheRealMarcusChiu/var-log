###### Instruction Set Architecture (ISA)
````excerpt
- is a [[Set Theory|set]] of [[Opcodes (Instruction) - Operation Codes - Instruction Syllable／Parcel - Opstring|opcodes]]
- an ISA specifies the behavior of [[Opcodes (Instruction) - Operation Codes - Instruction Syllable／Parcel - Opstring|opcodes]] running on implementations (i.e. [[Computer Organization - Microarchitecture (µarch - uarch)|microarchitecture]]) of that ISA in a fashion that does not depend on the characteristics of that implementation
````
^excerpt
