###### Harvard Architecture
- a stored-program computer similar to [[Von Neumann Architecture／Model - Princeton Architecture - Stored-Program Computer|Von Neumann Architecture]] but instead of having both instructions and data stored in the same memory:
	- has a dedicated set of address and data buses for reading and writing to memory
	- and another set of address and data buses to fetch instructions
