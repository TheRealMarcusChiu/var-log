- is the extraction of [[Knowledge|knowledge]] from data, using ideas from [[Statistics|statistics]], [[Machine Learning (ML) - Pattern Recognition|machine learning]], etc

# Steps to Data Analysis

- assemble ([[Data Acquisition／Integration／Preparation|Data Acquisition/Integration/Preparation]])

  - scrub/validate
  - order
  - categorize
- interpret
- simplify/clarify
- educate
- persuade
