###### portaudio

```excerpt
- is a cross-platform audio I/O library—basically a low-level plumbing layer that lets software record and play sound in real time without caring too much about the [[Operating Systems (OS)|operating system]] underneath
- think of it as: "One API to talk to audio devices on Windows, macOS, and Linux"
```

^excerpt
