###### Cassandra Query Language (CQL)
````excerpt
- is a NoSQL interface similar to [[Structured Query Language (SQL)|SQL]], but designed for [[Apache - Cassandra|Cassandra]]'s non-relational, distributed architecture
- supports basic data manipulation operations but lacks some features found in SQL, like joins, group by, and triggers
````
^excerpt
