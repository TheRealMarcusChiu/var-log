###### LLVM Intermediate Representation (IR)

- it's a common language that sits between the high-level program and the low-level backend
- is used to express high-level concepts and is specific enough that any backend can produce a fast [[Machine Language／Code／Instruction|machine-code]]
- is a low-level [[Computer／Programming Languages|programming language]] similar to [[Assembly Languages|assembly language]] (IR has all the advantages of assembly without any of its problems)
- is a strongly typed [[Reduced Instruction Set Compute／Computer (RISC)|Reduced Instruction Set Compute/Computer (RISC)]] instruction set
