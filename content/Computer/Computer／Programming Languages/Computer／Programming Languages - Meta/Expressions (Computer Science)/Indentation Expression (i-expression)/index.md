<strong>Indentation Expression (i-expression)</strong> uses indentation to indicate parentheses implicitly, and are thus in some ways intermediate between [[Symbolic Expression (s-expression - sexprs - sexps)|S-expressions]] and [[Meta Expression (m-expression)|M-expressions]]. I-expressions were introduced in [Scheme Request For Implementation](https://en.wikipedia.org/wiki/Scheme_Requests_for_Implementation) 49 as an auxiliary syntax for [Scheme](https://en.wikipedia.org/wiki/Scheme_(programming_language)), but they have not been widely adopted
