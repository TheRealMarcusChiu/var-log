- <strong>POCO</strong> - plain old class object - a normal class
  - <strong>POJO</strong> - plain old [[Java]] object
  - <strong>POPO</strong> - plain old [[PHP (PHP： Hypertext Preprocessor)|PHP]]/[[Python]] object
- <strong>DTO</strong> - data transfer object - a POCO used to represent the structure of data being transferred
- <strong>Persistence Model</strong> - models what and how data is stored, it models STORAGE STRUCTURE
  - <strong>JDO</strong> - java data object
- <strong>Domain Model</strong> - models real-life problems and solutions, it models BEHAVIOR
