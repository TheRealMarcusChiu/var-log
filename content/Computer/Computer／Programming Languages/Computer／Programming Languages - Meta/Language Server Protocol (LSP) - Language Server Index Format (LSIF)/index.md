````excerpt
- <strong>Language Server Protocol (LSP)</strong> defines the protocol used between an editor or [[Integrated Development Environments (IDEs)|IDE]] and a language server that provides language features like auto complete, go to definition, find all references etc
- <strong>Language Server Index Format (LSIF)</strong> pronounced like "else if" - it's goal is to support rich code navigation in development tools or a Web UI without needing a local copy of the source code
````
