###### Prolog ([www.swi-prolog.org](http://www.swi-prolog.org))

- has its roots in [first-order logic](http://doc.marcuschiu.com/log-page/5c609f82154cee28cd8f4060)
- is a type of [[Computer Languages - Logic／Logical Programming Languages|logic programming language]]
- is intended primarily as a declarative programming language: the program logic is expressed in terms of relations, represented as facts and rules
