###### React Native
````excerpt
- is an open-source UI software framework developed by [[Meta - Facebook|Meta]] Platforms
- it is used to develop applications for [[Android]], Android TV, [[Apple iOS|iOS]], [[Macintosh OSs (Mac OS X) - macOS|macOS]], [[Apple tvOS|tvOS]], Web, [[Windows]] and UWP
- uses the [[React - React.js - ReactJS|React]] framework along with native platform capabilities
````
