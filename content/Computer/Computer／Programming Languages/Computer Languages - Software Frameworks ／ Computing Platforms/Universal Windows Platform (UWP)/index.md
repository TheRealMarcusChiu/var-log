###### Universal Windows Platform (UWP)
````excerpt
- is a computing platform created by Microsoft and introduced in Windows 10
- the purpose of this platform is to help develop universal apps that run on Windows 10, Windows 10 Mobile (discontinued), Windows 11, Xbox One, Xbox Series X/S, and HoloLens without the need to be rewritten for each
````
^excerpt
