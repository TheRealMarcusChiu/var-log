###### LabVIEW (Laboratory Virtual Instrument Engineering Workbench)
````excerpt
- is a software development environment for creating data acquisition, instrument control, and industrial automation applications
- it uses a graphical programming language that allows users to create applications by dragging and dropping virtual instruments, such as oscilloscopes, function generators, and data acquisition devices, onto a block diagram
- is widely used in industry and academia for data acquisition, instrument control, and test and measurement applications
````
