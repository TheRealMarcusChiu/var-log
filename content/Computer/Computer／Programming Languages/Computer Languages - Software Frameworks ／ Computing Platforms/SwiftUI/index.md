###### SwiftUI
````excerpt
- is a declarative framework for building user interfaces for [iOS](https://en.wikipedia.org/wiki/IOS), [iPadOS](https://en.wikipedia.org/wiki/IPadOS), [watchOS](https://en.wikipedia.org/wiki/WatchOS), [tvOS](https://en.wikipedia.org/wiki/TvOS), [visionOS](https://en.wikipedia.org/wiki/VisionOS) and [macOS](https://en.wikipedia.org/wiki/MacOS), developed by [[Apple Inc|Apple Inc.]] for the Swift programming language
````
^excerpt
