###### Capacitor

```excerpt
- is the native runtime + bridge layer that lets your web app talk to real device APIs:
	- <code><font style="color: rgb(122,134,154);">Your app’s JavaScript ⇄ Capacitor ⇄ iOS / Android / Web APIs</font></code>
- is a framework from Ionic that:
	- wraps your web app ([[Angular]], [[React - React.js - ReactJS|React]], [[Vue - Vue.js - VueJS|Vue]], [[Svelte]], plain [[JavaScript|JS]], etc)
	- runs it inside a native container ([[Apple iOS|iOS]], [[Android]], desktop/web)
	- exposes native functionality through plugins
```

^excerpt
