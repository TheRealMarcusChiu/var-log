###### Ionic Framework
````excerpt
- is a free and open source component library for building apps that run on [[Apple iOS|iOS]], [[Android]], [[Electron (Framework)|Electron]], and the [[Computer Languages - Web Programming|Web]]
- you write your app once using familiar technologies ([[HyperText Markup Language (HTML)|HTML]], [[Cascading Style Sheets (CSS)|CSS]], [[JavaScript]]) and deploy to any platform
- integrates with popular frameworks like [[Angular]], [[React - React.js - ReactJS|React]], [[Vue - Vue.js - VueJS|Vue]], and also vanilla [[JavaScript]]
````
^excerpt
