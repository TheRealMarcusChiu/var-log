###### D3.js
````excerpt
- [https://d3js.org/](https://d3js.org/)
- is a [[JavaScript]] library for producing dynamic, interactive data visualizations in [[Web (HTTP) Clients／Browsers|web browsers]]
- it makes use of [[CSS - SVG|Scalable Vector Graphics]], [[HyperText Markup Language (HTML)|HTML5]], and [[Cascading Style Sheets (CSS)|Cascading Style Sheets]] standards
- it is the successor to the earlier Protovis framework
````
