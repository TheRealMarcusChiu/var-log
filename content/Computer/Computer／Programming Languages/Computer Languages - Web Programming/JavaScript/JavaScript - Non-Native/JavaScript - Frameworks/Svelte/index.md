###### Svelte
````excerpt
- is a [[JavaScript - Frameworks|JavaScript framework]] used for building user interfaces
- while it competes with big names like [[React - Frameworks|React]] and [[Vue - Vue.js - VueJS|Vue]], it works fundamentally differently under the hood
- the best way to think about Svelte is as a compiler, not a library
````
