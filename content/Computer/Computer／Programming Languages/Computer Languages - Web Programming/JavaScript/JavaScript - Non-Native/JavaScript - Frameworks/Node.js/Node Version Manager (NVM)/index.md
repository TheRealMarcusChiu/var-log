###### Node Version Manager (NVM)
````excerpt
- <code>nvm</code> is a version manager for [[Node.js|node.js]], designed to be installed per-user, and invoked per-shell
- <code>nvm</code> works on any [[UNIX - Portable Operating System Interface (POSIX)|POSIX]]-compliant [[UNIX - Shell|shell]] ([[sh (Bourne SHell)|sh]], [[dash (Debian Almquist SHell)|dash]], [[ksh (KornShell)|ksh]], [[zsh (Z SHell)|zsh]], [[bash (Bourne-Again SHell)|bash]]), in particular on these platforms: [[UNIX|Unix]], [[Macintosh OSs (Mac OS X) - macOS|macOS]], and [windows WSL](https://github.com/nvm-sh/nvm#important-notes)
````
^excerpt
