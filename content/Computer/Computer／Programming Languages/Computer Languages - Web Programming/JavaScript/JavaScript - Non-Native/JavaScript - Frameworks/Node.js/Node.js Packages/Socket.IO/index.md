<strong>Socket.IO</strong>
- is a [[JavaScript]] library for realtime web applications
- it enables realtime, bi-directional communication between web clients and servers (e.g. [[WebSockets]], etc)
- it has two parts:
	- a client-side library that runs in the browser
	- a server-side library for [[Node.js]]
