###### PlayCanvas WebGL Game Engine
````excerpt
- is an open-source game engine
- it uses HTML5 and [[JavaScript - WebGL 1 & 2|WebGL]] to run games and other interactive 3D content in any mobile or desktop browser
````
^excerpt

# Components
- 🦾 [<strong>PlayCanvas Engine</strong>](https://developer.playcanvas.com/user-manual/engine/) - Open source JavaScript run-time that powers all PlayCanvas applications
- ⚛️ [<strong>PlayCanvas React</strong>](https://developer.playcanvas.com/user-manual/playcanvas-react/) - Open source React-based declarative interface that wraps the PlayCanvas Engine
- 🧩 [<strong>PlayCanvas Web Components</strong>](https://developer.playcanvas.com/user-manual/web-components/) - Open source Web Components-based declarative interface that wraps the PlayCanvas Engine
- 🛠️ [[PlayCanvas Editor]] - Real-time collaborative visual editing environment for building PlayCanvas applications
