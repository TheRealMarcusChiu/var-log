###### webpack

```excerpt
- is a free and open-source [[Web Build Tools - Module Bundler|module bundler]] for JavaScript
- it is made primarily for [[JavaScript]], but it can transform front-end assets such as [[HyperText Markup Language (HTML)|HTML]], [[Cascading Style Sheets (CSS)|CSS]], and images if the corresponding loaders are included
- takes modules with dependencies and generates static assets representing those modules
```

^excerpt
