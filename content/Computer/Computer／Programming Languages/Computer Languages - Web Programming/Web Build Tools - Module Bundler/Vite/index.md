###### Vite

```excerpt
- is a [[Web Build Tools - Module Bundler|web build tool]]
- is a fast and modern JavaScript build tool and development server for web applications, especially useful for frameworks like [[Vue - Vue.js - VueJS|Vue.js]], [[React - React.js - ReactJS|React]], and Svelte
- it aims to provide a quicker and more efficient development experience by leveraging native ES modules and Hot Module Replacement (HMR)
```

^excerpt
