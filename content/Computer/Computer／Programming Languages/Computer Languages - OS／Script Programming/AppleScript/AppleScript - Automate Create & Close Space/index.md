### The Hassle

to close the current space in <strong>Mac Book Pro 13</strong> inch you need to do the following:

- swipe up with 3 fingers to display space dock
- hold option key to display close buttons
- move mouse pointer over close button and click

### The Solution

create a script that does all of that. and bind it to a hotkey

- <https://github.com/marcus-chiu-mac-personalization/macos-create-delete-space-automation>

closing script requires <code>cliclick</code>:

- <code>brew install cliclick</code>
