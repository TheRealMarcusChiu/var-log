###### 100 Gbps FPGA TCP-IP Stack
````excerpt
- This repository provides [[Transmission Control Protocol (TCP)|TCP]]/[[Internet Protocol (IP)|IP]] network support at 100 Gbit/s in [[Vitis Software Development Platform|Vitis]] and provides several examples to demonstrate the usage.
- This repository is forked from Vitis with 100 Gbps TCP/IP Network Stack and contains the following changes:
	- Build scripts for use with the TaPaSCo framework
	- Build-time configurable bit width of data-carrying interfaces
	- Support for simulation, e.g. using NetSimulator
	- Several bug fixes
````
^excerpt
