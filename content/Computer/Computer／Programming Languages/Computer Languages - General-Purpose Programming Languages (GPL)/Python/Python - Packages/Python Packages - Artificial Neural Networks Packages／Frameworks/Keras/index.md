###### Keras

- is a high-level, [[Deep Learning|deep-learning]] API developed by Google for implementing [[Artificial Neural Networks (ANN)|neural networks]]
- it can run on top of CNTK, TensorFlow, and Theano
- was adopted and integrated into TensorFlow in mid-2017. Users can access it via the <code><font style="color: rgb(122,134,154);">tf.keras</font></code> module. However, the Keras library can still operate separately and independently
