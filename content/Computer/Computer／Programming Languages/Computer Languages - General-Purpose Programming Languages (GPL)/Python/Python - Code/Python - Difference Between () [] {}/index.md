# () - tuple

A tuple is a sequence of items that can't be changed (immutable)
# \[\] - list

A list is a sequence of items that can be changed (mutable)
# {} - dictionary or set

A dictionary is a list of key-value pairs, with unique keys (mutable). From Python 2.7/3.1, <code>{}</code> can also represent a set of unique values (mutable)
