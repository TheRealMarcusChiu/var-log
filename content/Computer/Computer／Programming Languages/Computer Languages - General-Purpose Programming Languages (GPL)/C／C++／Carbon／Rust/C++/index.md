# Installation

> [!expand]- homebrew <strong>0. search for which g++ compiler you want</strong> <span style="white-space: pre-wrap"><code>  brew search g++</code></span>
>
> <strong>1. install \<g++ version></strong> <span style="white-space: pre-wrap"><code>  brew install \<g++ version></code></span>
>
> <strong>2. verify gcc was installed</strong> <span style="white-space: pre-wrap"><code>  which \<g++ version></code><br><code>  \<g++ version> —version</code><br><code>  \<g++ version> —help</code></span>

> [!expand]- apt-get <strong>0. search for which g++ compiler you want</strong> <span style="white-space: pre-wrap"><code>  apt-cache search g++</code></span>
>
> <strong>1. install \<g++ version></strong> <span style="white-space: pre-wrap"><code>  apt-get install \<g++ version></code></span>
>
> <strong>2. verify gcc was installed</strong> <span style="white-space: pre-wrap"><code>  which \<g++ version></code><br><code>  \<g++ version> —version</code><br><code>  \<g++ version> —help</code></span>
