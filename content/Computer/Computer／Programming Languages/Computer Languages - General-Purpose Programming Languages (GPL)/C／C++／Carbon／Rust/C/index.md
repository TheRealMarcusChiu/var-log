# Installation

> [!expand]- homebrew
> <strong>0. search for which gcc compiler you want</strong>
> <span style="white-space: pre-wrap"><code>  brew search gcc</code></span>
>
> <strong>1. install (gcc version)</strong>
> <span style="white-space: pre-wrap"><code>  brew install \<(gcc version)</code></span>
>
> <strong>2. verify gcc was installed</strong>
> <span style="white-space: pre-wrap"><code>  which gcc</code><br><code>  gcc —version</code><br><code>  gcc —help</code></span>
