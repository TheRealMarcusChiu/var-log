###### Erlang ([www.erlang.org](http://www.erlang.org))
- is a [[Computer Languages - General-Purpose Programming Languages (GPL)|general-purpose]], concurrent, [[Computer Languages - Function／Functional Programming Languages|functional programming]] language, and a garbage-collected runtime system

# Erlang - Layers

Erlang is a language with several layers, each of which are supersets of the smaller layers.
