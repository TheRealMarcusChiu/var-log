###### Groovy ([www.groovy-lang.org](http://www.groovy-lang.org/))

```excerpt
- used as a scripting language where it is dynamically compiled to [[Java Virtual Machine (JVM)|JVM]] bytecode
- interoperates with other Java code and libraries
```

^excerpt

# Installation

> [!expand]- homebrew <strong>1. install groovy using homebrew</strong> <span style="white-space: pre-wrap"><code>  brew install groovy</code></span>
>
> this would download groovy into /usr/local/Cellar directory
>
> <strong>2. verify groovy was installed</strong> <span style="white-space: pre-wrap"><code>  which groovy</code><br><code>  groovy -—version</code></span>
