###### Scala ([www.scala-lang.org](http://www.scala-lang.org/))
````excerpt
- runs on the [[Java Virtual Machine (JVM)|Java Virtual Machine]]
- is an [[Object-Oriented Programming|object-oriented]] [[Computer Languages - Function／Functional Programming Languages|functional]] [[Computer／Programming Languages|programming language]]
- is [[Static／Statically vs Dynamic／Dynamically Type／Typed Programming Languages|static typed]]
````
^excerpt

# Installation

> [!expand]- homebrew
> <strong>1. install scala</strong>
> <span style="white-space: pre-wrap"><code>  brew install scala</code></span>
>
> <strong>1. To use with IntelliJ, set the Scala home to:</strong>
> <span style="white-space: pre-wrap"><code>  /usr/local/opt/scala/idea</code></span>
>
> <strong>2. Bash completion has been installed to:</strong>
> <span style="white-space: pre-wrap"><code>  /usr/local/etc/bash\_completion.d</code></span>
# Features
- unlike Java, Scala has many features of [[Computer Languages - Function／Functional Programming Languages|functional programming languages]] like [[Scheme]], Standard ML, and [[Haskell]], including currying, type inference, immutability, lazy evaluation, and pattern matching
