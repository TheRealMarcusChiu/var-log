###### Object-Graph Navigation Language (OGNL)
- it is an [expression](https://marcuschiu.atlassian.net/wiki/pages/resumedraft.action?draftId=5671869) language for getting and setting properties of [[Java]] objects, plus other extras such as list projection and selection and lambda expressions
