###### Spring Data JPA (Hibernate)
- is one of many [[Java - Spring Data - Spring Data Modules (Builtin)|Spring Data Modules]] for relational databases
- is a wrapper around [[Java - Hibernate|Hibernate]]
- allows you to make queries without writing [[Structured Query Language (SQL)|SQL]]/[[Cypher Query Language (CQL)|CQL]] in [[Java - Spring Data - Spring Data Repository|Repository interfaces]]
