###### @RepositoryRestResource
- as you can see [here](http://spring.io/guides/gs/accessing-neo4j-data-rest/) adding this annotation and linking it to your [[Classes／Objects Types (POCO／POJO／POPO - Data Transfer Object DTO - Domain Model - Persistence Model - Java Data Object (JDO))|POJO]] you have a fully functional <strong>HATEOAS</strong> service without having to implement the repository method or the REST service methods
- similar to: [[Java - Spring - Web - @Controller @RestController|@Controller & @RestController]]
