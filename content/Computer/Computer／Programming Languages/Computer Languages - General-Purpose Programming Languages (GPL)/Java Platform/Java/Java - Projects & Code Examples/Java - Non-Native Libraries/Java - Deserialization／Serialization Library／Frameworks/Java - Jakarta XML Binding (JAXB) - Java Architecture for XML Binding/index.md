<strong>Jakarta XML Binding (JAXB)</strong> (formerly Java Architecture for XML Binding) is a [[Java - Deserialization／Serialization Library／Frameworks|Java Deserialization/Serialization Library/Framework]] that allows Jakarta EE developers to map Java Class Objects to [[Extensible Markup Language (XML)|XML]] representations

[JAXB is bundled in JDK 1.6 to 1.8](https://jaxb.java.net/guide/Which_JAXB_RI_is_included_in_which_JDK_.html), marked deprecated in Java 9 and 10 and removed in Java 11
