###### Jakarta Commons Logging (JCL) - Apache Commons Logging
- is conceptually very similar to [[Java - Simple Logging Facade for Java (SLF4j)|Slf4j]], as such, it can be thought of as yet another logging [[Facade Pattern|facade]]. However, Slf4j is much simpler in design and arguably more robust. In a nutshell, Slf4j avoid the class loader issues that plague JCL
