###### Apache Log4j 2
- is an upgrade to [[Java - Log4j|Log4j]] that provides significant improvements over its predecessor, Log4j 1.x
- provides many of the improvements available in [[Java - Logback|Logback]] while fixing some inherent problems in Logback's architecture
