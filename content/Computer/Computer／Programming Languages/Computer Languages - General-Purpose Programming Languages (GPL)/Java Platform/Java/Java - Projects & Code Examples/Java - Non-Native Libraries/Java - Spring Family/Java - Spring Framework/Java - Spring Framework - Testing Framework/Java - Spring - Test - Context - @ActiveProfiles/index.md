###### @ActiveProfiles

- is a class-level annotation that is used to declare which <em>active bean definition profiles</em> (defined by <code>[[Java - Spring - @Profile|@Profile]]</code>) should be used when loading an <code>[[Java - Spring - ApplicationContext|ApplicationContext]]</code> for test classes
