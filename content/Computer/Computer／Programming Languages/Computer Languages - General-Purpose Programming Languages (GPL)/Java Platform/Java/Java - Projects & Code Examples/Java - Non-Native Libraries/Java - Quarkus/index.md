###### Quarkus

```excerpt
- is a Java framework (like [[Java - Spring Family|Spring]]) but for [[Container／Image - Container Runtimes - Container Engine - Container-Based Virtualization - Container Scheduler／Orchestrator／Orchestration／Orchestrating Engine (COE)|containers]] and [[Kubernetes]]
- optimized for:
	- fast startup
	- low memory usage
	- cloud & serverless environments
- Quarkus apps is optimized to run on [[GraalVM]] instead of standard [[Java Virtual Machine (JVM)|JVM]]
```

^excerpt
