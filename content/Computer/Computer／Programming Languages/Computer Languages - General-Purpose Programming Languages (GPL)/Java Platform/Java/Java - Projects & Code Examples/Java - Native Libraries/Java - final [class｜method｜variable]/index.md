In Java, items with the <code><font style="color: rgb(122,134,154);">final</font></code> modifier cannot be changed!

This includes final classes, final variables, and final methods:
- A <code><font style="color: rgb(122,134,154);">final</font></code> class cannot be extended by any other class (e.g. <code><font style="color: rgb(122,134,154);">java.lang.Math</font></code>)
- A <code><font style="color: rgb(122,134,154);">final</font></code> method cannot be overridden
- A <code><font style="color: rgb(122,134,154);">final</font></code> variable cannot be reassigned to another value
