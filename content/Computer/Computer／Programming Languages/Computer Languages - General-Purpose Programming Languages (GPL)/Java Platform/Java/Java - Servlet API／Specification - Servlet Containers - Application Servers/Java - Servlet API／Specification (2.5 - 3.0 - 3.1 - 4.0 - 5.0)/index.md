<strong>Servlet</strong> is a Java application programming interface (API) running on the server machine which can intercept the requests made by the client and can generate/send a response accordingly
# Specifications
- Jakarta EE 9: [Jakarta Servlet 5.0 Specification](https://jakarta.ee/specifications/servlet/5.0/)
- Jakarta EE 8: [Jakarta Servlet 4.0 Specification](https://jakarta.ee/specifications/servlet/4.0/)
- Java EE 8: [Java Servlet 4.0 Specification (JSR 369)](https://jcp.org/en/jsr/detail?id=369)
- Java EE 7: [Java Servlet 3.1 Specification (JSR 340)](http://jcp.org/en/jsr/detail?id=340)
- Java EE 6: [Java Servlet 3.0 Specification (JSR 315)](http://jcp.org/en/jsr/detail?id=315)
- Java EE 5: [Java Servlet 2.5 Specification (JSR 154)](http://jcp.org/en/jsr/detail?id=154)
