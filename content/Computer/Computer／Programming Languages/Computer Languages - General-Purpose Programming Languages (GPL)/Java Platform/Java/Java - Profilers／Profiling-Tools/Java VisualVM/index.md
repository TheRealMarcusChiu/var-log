###### [Java VisualVM](https://visualvm.github.io/)
- is a simplified yet robust [[Java - Profilers／Profiling-Tools|profiling tool]] for Java applications
- by default, this tool is bundled with the [[Java - Installation (JDK - JRE - JVM - JE - JSE - JEE - JME)|Java Development Kit (JDK)]]
- its operation relies on other standalone tools provided in the JDK, such as <em>JConsole</em>, <em>jstat</em>, <em>jstack</em>, <em>jinfo</em>, and <em>jmap</em>
