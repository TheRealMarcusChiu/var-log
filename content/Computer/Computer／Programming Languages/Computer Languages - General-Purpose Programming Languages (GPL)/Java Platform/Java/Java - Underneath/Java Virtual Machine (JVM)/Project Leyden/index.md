###### Project Leyden
````excerpt
- is an initiative from the OpenJDK team
- a project to improve the standard [[Java Virtual Machine (JVM)|JVM]]
- the primary goal of this Project is to improve the startup time, time to peak performance, and footprint of Java programs, and over time may reduce the <em>need</em> for [[GraalVM]]’s native‑image tool in some cases
````
