A repository manager serves these essential purposes:

- act as dedicated proxy server for public Maven repositories (see [Maven Guide to Mirror Settings](https://maven.apache.org/guides/mini/guide-mirror-settings.html))
- provide repositories as a deployment destination for your Maven project outputs
