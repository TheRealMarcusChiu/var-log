###### Perl (Practical Extraction and Report Language)
- is a programming language developed by Larry Wall
- designed for text processing
- it runs on a variety of platforms, such as Windows, Mac OS, and the various versions of UNIX

# Installation

> [!expand]- homebrew
> <strong>0. search perl versions via home-brew</strong>
> <span style="white-space: pre-wrap"><code>  brew search perl</code></span>
>
> <strong>1. install perl using home-brew</strong>
> <span style="white-space: pre-wrap"><code>  brew install \<perl version here\></code></span>
>
> this would download into /usr/local/Cellar directory
>
> <strong>2. verify perl</strong>
> <span style="white-space: pre-wrap"><code>  which \<perl version here\></code><br><code>  \<perl version here\> -v</code></span>
