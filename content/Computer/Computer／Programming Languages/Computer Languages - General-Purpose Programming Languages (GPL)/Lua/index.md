###### Lua

- is a lightweight, high-level, [[Computer Languages - General-Purpose Programming Languages (GPL)|multi-paradigm programming language]] designed primarily for embedded use in applications
- is cross-platform, since the interpreter of compiled bytecode is written in ANSI C, and Lua has a relatively simple C API to embed it into applications
