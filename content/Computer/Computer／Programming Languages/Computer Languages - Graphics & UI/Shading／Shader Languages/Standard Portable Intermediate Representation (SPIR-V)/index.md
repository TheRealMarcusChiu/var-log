###### Standard Portable Intermediate Representation (SPIR-V)
- is catalyzing a revolution in the ecosystem for shader and kernel language compilers used for expressing [[Parallel - Algorithms／Computing／Programming／Systems|parallel computation]] and [[Graphics Processing Unit (GPU) - Graphics Card|GPU]]-based graphics
- is meant to be machine written & read instead of being human-friendly
- is simpler and more [[Regular Languages|regular language]] than [[OpenGL Shading／Shader Language (GLSL)|GLSL]]
