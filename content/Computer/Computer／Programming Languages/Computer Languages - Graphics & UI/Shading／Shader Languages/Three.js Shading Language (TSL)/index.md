###### Three.js Shading Language (TSL)
````excerpt
- is a high-level abstraction within the [[Three.js]] library designed to simplify the process of writing shader code
- TSL acts as an interpreter, converting the [[JavaScript]]/[[Typescript|TypeScript]] shader code into the appropriate low-level shader language ([[OpenGL Shading／Shader Language (GLSL)|GLSL]] for [[JavaScript - WebGL 1 & 2|WebGL2]] or [[WebGPU Shading／Shader Language (WGSL)|WGSL]] for [[JavaScript - WebGPU|WebGPU]]) based on the browser's capabilities
````
^excerpt
