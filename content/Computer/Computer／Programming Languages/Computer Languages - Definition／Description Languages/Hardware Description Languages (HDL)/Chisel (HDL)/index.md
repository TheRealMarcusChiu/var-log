###### Chisel (HDL)
````excerpt
- is an open-source [[Hardware Description Languages (HDL)|hardware description language (HDL)]] embedded in [[Scala]]
- it allows hardware designers to use modern programming language features like object-oriented programming, functional programming, and type safety to create complex, parameterized hardware designs
- is particularly useful for register-transfer level (RTL) design and hardware generation
````
^excerpt
