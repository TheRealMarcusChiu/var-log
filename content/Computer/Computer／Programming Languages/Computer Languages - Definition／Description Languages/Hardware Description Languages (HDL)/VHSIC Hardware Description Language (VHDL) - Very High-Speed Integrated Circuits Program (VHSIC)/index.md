###### VHSIC Hardware Description Language (VHDL)

```excerpt
- is a type of [[Hardware Description Languages (HDL)|hardware description language (HDL)]]
- since 1987, VHDL has been standardized as IEEE Std 1076; the latest version of which is IEEE Std 1076-2019
- is named after the United States Department of Defense program that created it, the <strong>Very High-Speed Integrated Circuits Program (VHSIC)</strong>. In the early 1980s, the VHSIC Program sought a new HDL for use in the design of the integrated circuits it aimed to develop. The product of this effort was VHDL Version 7.2, released in 1985
- based on Ada and Pascal
- popular amongst engineers in Europe
```
