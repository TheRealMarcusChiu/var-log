###### Verilog
````excerpt
- standardized as IEEE 1364
- is a type of [[Hardware Description Languages (HDL)|hardware description language (HDL)]]
- the language is based on [[C]]
- the most popular for [[Field-Programmable Gate Arrays (FPGA)|FPGA]] engineers based in the USA
- is extended by [[SystemVerilog]]
- today is used mainly for test analysis and verification of digital circuits at the register-transfer level of abstraction
````
^excerpt
