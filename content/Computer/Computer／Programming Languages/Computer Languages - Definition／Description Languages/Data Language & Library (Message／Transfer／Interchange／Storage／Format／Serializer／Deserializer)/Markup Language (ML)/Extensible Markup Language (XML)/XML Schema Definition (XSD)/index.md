###### XML Schema Definition (XSD)
- a type of [[Schema Definition／Description Language & Libraries (SDL)|Schema Definition Language (SDL)]] written in [[Extensible Markup Language (XML)|XML]]
- defines the structure and the legal elements and attributes of an [[Extensible Markup Language (XML)|XML]] document
- a more powerful alternative to [[Document Type Definition (DTD)]]
