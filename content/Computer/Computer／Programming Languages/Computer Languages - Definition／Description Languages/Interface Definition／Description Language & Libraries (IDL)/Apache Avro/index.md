###### Apache Avro

- is a row-oriented [[Remote Procedure Call (RPC) - architectural|remote procedure call]] and [[Data Language & Library (Message／Transfer／Interchange／Storage／Format／Serializer／Deserializer)|data serialization library]]
- developed within Apache's [[Apache Hadoop]] project
- uses [[JavaScript Object Notation (JSON)|JSON]] for defining data types and protocols, and serializes data in a compact binary format
