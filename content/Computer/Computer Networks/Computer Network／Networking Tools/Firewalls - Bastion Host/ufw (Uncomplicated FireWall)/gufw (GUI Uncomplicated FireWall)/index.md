###### gufw (GUI Uncomplicated FireWall) - [www.gufw.org](http://www.gufw.org/)

- is a [[Firewalls - Bastion Host|firewall]] powered by [[ufw (Uncomplicated FireWall)]]

# Installation

> [!expand]- gufw <strong>1. install gufw</strong>
>
> ```
>   apt-get install gufw
> ```
>
> <strong>2. verify gufw was installed</strong>
>
> ```
>   which gufw
> ```
>
> <strong>3. start application</strong>
>
> ```
>   gufw
> ```
