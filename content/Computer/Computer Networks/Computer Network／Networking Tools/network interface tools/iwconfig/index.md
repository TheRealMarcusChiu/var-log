<strong>iwconfig</strong> configures a wireless [[Network Interface Controller／Card (NIC) - Network Adapter - LAN Adapter - Physical Network Interface|network interface]]

related: [[ifconfig]]

### Installation

<span style="white-space: pre-wrap"><code>sudo apt install net-tools</code></span>

### Commands

<strong>1. Change NIC channel</strong> <span style="white-space: pre-wrap"><code>  iwconfig \<NIC name> channel \<channel number></code></span>

<strong>2. change mode of NIC (monitor.. )</strong> <span style="white-space: pre-wrap"><code>  iwconfig \<NIC> mode \<mode name></code></span>
