title: Host Overrides vs Domain Overrides
created: 2024-08-18T01:59:45.400-05:00
modified: 2024-08-18T02:01:13.184-05:00
---

- host overrides are used to configure a specific hostname to resolve differently than it normally would from an upstream DNS server
- domain overrides specify a different [[DNS Server (DNS Resolver - Root Server - Top-Level Server - Authoritative Server)|DNS server]] to use when resolving a specific domain
