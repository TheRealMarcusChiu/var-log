###### OPNsense

```excerpt
- is an open-source, [[FreeBSD]]-based firewall and routing software
- developed by Deciso, a company in the Netherlands that makes hardware and sells support packages for OPNsense
- it is a fork of [[pfSense]], which in turn was forked from m0n0wall built on FreeBSD
- it was launched in January 2015
```

^excerpt
