###### Stream Control Transmission Protocol (SCTP)
````excerpt
- is a [[Computer Network／Networking Communication Protocols|network protocol]] that operates at the [[4 - Transport Layer|transport layer]]
- provides some of the features of both [[Universal／User Datagram Protocol (UDP)|UDP]] and [[Transmission Control Protocol (TCP)|TCP]]:
	- it is message-oriented like UDP and ensures reliable
	- in-sequence transport of messages with congestion control like TCP
	- It differs from those protocols by providing multi-homing and redundant paths to increase resilience and reliability
````
