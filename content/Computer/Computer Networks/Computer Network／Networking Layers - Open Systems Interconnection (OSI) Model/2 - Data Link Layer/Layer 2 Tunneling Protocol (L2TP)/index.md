###### Layer 2 Tunneling Protocol (L2TP)
````excerpt
- is a [[Computer Network／Networking Communication Protocols|network protocol]] that operates at the [[2 - Data Link Layer|data link layer]]
- is a tunneling protocol used to support [[Virtual Private Network (VPN)|VPNs]] or as part of the delivery of services by ISPs
- it does not provide any encryption or confidentiality by itself. Rather, it relies on an encryption protocol (usually [[Internet Protocol Security (IPSec)|IPSec]]) that it passes within the tunnel to provide privacy
````
^excerpt
