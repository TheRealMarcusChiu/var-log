separating addresses into [[IPv4 - Address|classes A B and C]] is not sufficient enough for today's networking, therefore subnetting and supernetting were created

- <strong>[[IPv4 - Subnetting|subnetting]]</strong> - allows splitting classful addresses into multiple subnets
- <strong>[[IPv4 - Supernetting (CIDR)|supernetting]]</strong> - allows to coalesce several classful addresses into a single supernet
