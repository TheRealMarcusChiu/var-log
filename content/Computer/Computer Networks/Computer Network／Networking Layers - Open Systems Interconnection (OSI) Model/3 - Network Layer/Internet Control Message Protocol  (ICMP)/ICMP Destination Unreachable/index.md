[[Internet Control Message Protocol  (ICMP)|ICMP]] message used by gateways to state that the datagram cannot be delivered

### Subtypes

- network unreachable
- host unreachable
- protocol unreachable
- port unreachable
- fragmentation needed but don’t fragment bit set
- etc
