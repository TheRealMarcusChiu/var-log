###### Network Layer
````excerpt
- computes paths across an interconnected mesh of links and packet switches, and forwards packets over multiple links from source to destination and routing through intermediate routers
````
^excerpt

# Network Protocol Categories
- [[Routing Protocols]] - used between routers to build and maintain routing tables, so data/packets could be routed successfully
- [[Routed Protocols]] - used to routed data/packets across an internetwork

- [[Internet Control Message Protocol  (ICMP)|ICMP]] - used for error diagnosis
- <strong>[[Internet Group Management Protocol (IGMP)|IGMP]]</strong> - integral for IP multicast

# Hardware
- <strong>[[Router]]</strong> - note: there is a difference between home routers and industrial routers
- [[Autonomous System (AS) - Domain|Autonomous System]] - a collection of networks or subnets that are in the same administrative domain
