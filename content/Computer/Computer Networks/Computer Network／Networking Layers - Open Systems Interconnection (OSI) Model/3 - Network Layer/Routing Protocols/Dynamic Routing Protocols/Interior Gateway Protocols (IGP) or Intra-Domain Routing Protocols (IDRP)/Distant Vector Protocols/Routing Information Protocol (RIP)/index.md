###### Routing Information Protocol (RIP)
````excerpt
- [routers](http://confluence.marcuschiu.com/display/NOT/Router) running <strong>RIP</strong> are <em>periodic</em> and <em>triggered</em>
- does not allow subnet masks like [RIP2](http://confluence.marcuschiu.com/pages/viewpage.action?pageId=2656708), only old [classful addresses](http://confluence.marcuschiu.com/pages/viewpage.action?pageId=2656424) of [IP](http://confluence.marcuschiu.com/pages/viewpage.action?pageId=2656674)
````
^excerpt

- is a type of [[Distant Vector Protocols|distant vector protocol]]
