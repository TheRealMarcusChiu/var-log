###### Interior Gateway Routing Protocol (IGRP)
````excerpt
- is a proprietary protocol developed by Cisco
- is a [[Distant Vector Protocols|distance vector]] [[Routing Protocols|routing protocol]] used to communicate routing information within a host network or [[Autonomous System (AS) - Domain|autonomous system]]
- manages the flow of routing information within connected [[Router|routers]] in the host network or [[Autonomous System (AS) - Domain|autonomous system]]
````
^excerpt
