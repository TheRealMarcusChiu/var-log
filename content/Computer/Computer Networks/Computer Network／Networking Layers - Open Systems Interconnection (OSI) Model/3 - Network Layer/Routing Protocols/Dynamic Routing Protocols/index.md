###### Dynamic Routing Protocols

```excerpt
- are a type of [[Routing Protocols]] used by routers to DYNAMICALLY find all networks in the internetwork, update the routing table, and ensure all routers have the same routing table. Thus, allowing [[Routed Protocols]] to determine the path of an [[Internet Protocol (IP)|IP]] packet through an internetwork
```

^excerpt

# Dynamic Routing Protocol Types

- [[Interior Gateway Protocols (IGP) or Intra-Domain Routing Protocols (IDRP)]] - routes within <em>[[Autonomous System (AS) - Domain|Autonomous System]]</em>
- [[Exterior Gateway Protocols (EGPs) or Inter-Domain Routing Protocols (IDRP)]] - routes between multiple <em>[[Autonomous System (AS) - Domain|Autonomous Systems]]</em>

[[Autonomous System (AS) - Domain|Autonomous System]] is a collection of networks or [[IPv4 - Subnet & Supernet|subnets]] that are in the same administrative domain
