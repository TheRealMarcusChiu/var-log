###### Remote Management Control Protocol (RMCP)
````excerpt
- is a lightweight transport protocol built on [[Universal／User Datagram Protocol (UDP)|UDP]] that carries remote management messages
- most commonly for the Baseboard Management Controller (BMC) and [[Intelligent Platform Management Interface (IPMI)]]
````
