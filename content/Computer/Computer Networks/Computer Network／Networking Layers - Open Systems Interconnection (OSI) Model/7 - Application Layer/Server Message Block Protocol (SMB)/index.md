###### Server Message Block Protocol (SMB)
````excerpt
- is a client-server communication protocol used for sharing access to files, printers, serial ports, and other resources on a network (i.e. [[Network Attached Storage (NAS)|NAS]])
- it can also carry transaction protocols for [[Inter-Process／Interprocess Communication (IPC)|interprocess communication]]
- operates at the [[7 - Application Layer|application layer]]
````
^excerpt
