###### HAProxy
- is a free and open-source software that provides a high availability [[Load Balancers／Balancing|load balancer]] and [[Reverse Proxy|proxy server]] for [[Transmission Control Protocol (TCP)|TCP]] and [[Hypertext／Hyper Text Transfer Protocol (HTTP) - 1／1.1／2|HTTP]]-based applications that spread requests across multiple servers.
- it is written in C and has a reputation for being fast and efficient
