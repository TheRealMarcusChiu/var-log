###### Web Distributed Authoring and Versioning (WebDAV)
````excerpt
- is an extension to the [[Hypertext／Hyper Text Transfer Protocol (HTTP) - 1／1.1／2|HTTP]] protocol that allows for collaborative file management and editing on web servers
- it enables users to interact with files stored on online servers, providing features like file locking, revision tracking, and namespace management
````
