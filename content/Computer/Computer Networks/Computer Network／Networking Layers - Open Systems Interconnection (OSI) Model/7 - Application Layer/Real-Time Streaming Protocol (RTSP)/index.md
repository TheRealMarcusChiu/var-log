###### Real-Time Streaming Protocol (RTSP)
````excerpt
- is a network protocol used to control multimedia streaming, primarily for audio and video
- it allows clients to initiate, control, and manage media sessions, like playing, pausing, and recording, much like controlling a VCR
- is often used in video surveillance systems for live streaming from IP cameras
````
^excerpt
