###### WebSockets
- is an [[7 - Application Layer|application protocol]]/[[API - Network-Level|network-level API]] protocol for [[Uni-Directional／Unidirectional - Bi-Directional／Bidirectional - Half-Duplex - Full-Duplex - Unsolicited|full-duplex]] communication between server and client
- distinguishing features:
	- full-duplex
	- uses [[Hypertext／Hyper Text Transfer Protocol (HTTP) - 1／1.1／2|HTTP]] for initializing and a single [[Transmission Control Protocol (TCP)|TCP]] connection for communication
	- allows [[Uni-Directional／Unidirectional - Bi-Directional／Bidirectional - Half-Duplex - Full-Duplex - Unsolicited|unsolicited communication]] from server to client (after [[Hypertext／Hyper Text Transfer Protocol (HTTP) - 1／1.1／2|HTTP]] initialize)
