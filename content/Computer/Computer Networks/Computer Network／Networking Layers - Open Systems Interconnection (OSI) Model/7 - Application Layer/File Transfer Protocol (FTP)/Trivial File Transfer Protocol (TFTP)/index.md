###### Trivial File Transfer Protocol (TFTP)
````excerpt
- is a simple[[File Transfer Protocol (FTP)|File Transfer Protocol]] which allows a client to get a file from or put a file onto a remote host
- one of its primary uses is in the early stages of nodes booting from a [[Computer Network／Networking Sizes (NANO／NFC／BAN／PAN／SAN／LAN／CAN／MAN／RAN／WAN) Near-Field Communication Body／Personal／Storage／System／Local／Controller／Metropolitan／Radio／Wide Area／Access Network|local area network]] (i.e. [[Network Booting (NetBoot)|network boot]])
````
^excerpt
