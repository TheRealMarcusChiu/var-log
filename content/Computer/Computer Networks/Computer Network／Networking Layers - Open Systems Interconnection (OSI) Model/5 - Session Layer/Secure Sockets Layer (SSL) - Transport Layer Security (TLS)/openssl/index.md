###### OpenSSL ([www.openssl.org](http://www.openssl.org))
- is a toolkit for the [[Secure Sockets Layer (SSL) - Transport Layer Security (TLS)|TLS and SSL]] protocols

# Installation

> [!expand]- homebrew
> <strong>1. brew install openssl</strong>
> <span style="white-space: pre-wrap"><code>  brew install openssl</code></span>
>
> <strong>2. verify openssl was installed</strong>
> <span style="white-space: pre-wrap"><code>  which openssl</code><br><code>  openssl version -a</code></span>
