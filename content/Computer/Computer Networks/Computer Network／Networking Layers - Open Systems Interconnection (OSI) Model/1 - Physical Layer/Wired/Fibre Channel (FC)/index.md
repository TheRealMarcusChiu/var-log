###### Fibre Channel (FC)

```excerpt
- is a high-speed data transfer protocol providing in-order, lossless delivery of raw block data
- is both a [[1 - Physical Layer|Layer 1]] & [[2 - Data Link Layer|Layer 2]] protocol
- is primarily used to connect computer data storage to servers in [[Storage Area Network (SAN)|storage area networks (SAN)]] in commercial data centers
```

^excerpt

# FC - Upper Protocols

- [[Fibre Channel Protocol for SCISI (FCP)]] - uses Fibre Channel to send [[Small Computer Systems Interface (SCSI)|SCSI]] data over Fibre Channel networks
- [FICON](https://en.wikipedia.org/wiki/FICON) - uses Fibre Channel to send [ESCON](https://en.wikipedia.org/wiki/ESCON) commands over Fibre Channel networks (used by [IBM mainframe](https://en.wikipedia.org/wiki/IBM_mainframe) computers)
