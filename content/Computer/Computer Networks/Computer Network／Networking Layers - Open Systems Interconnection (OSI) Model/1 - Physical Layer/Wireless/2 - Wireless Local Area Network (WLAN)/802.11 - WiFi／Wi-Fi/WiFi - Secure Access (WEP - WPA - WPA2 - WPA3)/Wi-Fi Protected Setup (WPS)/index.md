<strong>Wi</strong>-<strong>Fi Protected Setup (<strong>WPS)</strong></strong> is a wireless network security standard that tries to make connections between a router and wireless devices faster and easier. WPS works only for wireless networks that use a password that is encrypted with the [[WPA Configuration Mode (WPA-Personal - WPA-Enterprise)|WPA Personal]] or [[WPA Configuration Mode (WPA-Personal - WPA-Enterprise)|WPA2 Personal]] security protocols

<font style="color: rgb(255,0,0);">WPS should be disabled</font>
