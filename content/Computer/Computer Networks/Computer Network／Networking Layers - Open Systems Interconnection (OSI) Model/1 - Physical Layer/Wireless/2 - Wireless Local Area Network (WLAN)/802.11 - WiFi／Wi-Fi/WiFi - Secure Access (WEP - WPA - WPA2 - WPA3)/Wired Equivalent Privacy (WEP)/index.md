<strong>Wired Equivalent Privacy (WEP)</strong>, the original [[WiFi - Secure Access (WEP - WPA - WPA2 - WPA3)|Wi-Fi security]] standard was designed to provide:

- [[Authentication (AuthN)|authentication]]
- [[Encryption|confidentiality]]
- integrity

no support for:

- key management
- authorization
- replay protection

[[Wired Equivalent Privacy (WEP)|WEP]] (<font style="color: rgb(255,0,0);">not used anymore, because not secure<font style="color: rgb(0,51,102);">)</font></font> has been superseded by [[WiFi Protected Access (WPA)]] which provides more sophisticated data [[Encryption|encryption]] and better user [[Authentication (AuthN)|authentication]]

### WEP Versions

- 64-bit
- 128-bit
- 256-bit
