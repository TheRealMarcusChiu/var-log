<strong>Wi-Fi Protected Access</strong> (<strong>WPA</strong>) provides more sophisticated data [[Encryption|encryption]] and better user [[Authentication (AuthN)|authentication]] than [[Wired Equivalent Privacy (WEP)]], the original [[WiFi - Secure Access (WEP - WPA - WPA2 - WPA3)|Wi-Fi security]] standard

### WPA Encryption Options

- [[Temporal Key Integrity Protocol (TKIP)|TKIP]]
- [[Advanced Encryption Standard (AES)|AES]]

### WPA Configuration Modes

[[WPA Configuration Mode (WPA-Personal - WPA-Enterprise)]]:

- <strong>WPA-Personal </strong><strong>(WPA-PSK)</strong> - commonly used for home networks
- <strong>WPA-Enterprise (WPA-[[802.1X - EAPOL|802.1X]], [[Remote Authentication Dial-In User Service (RADIUS)|RADIUS]])</strong> - commonly used in university and company networks
