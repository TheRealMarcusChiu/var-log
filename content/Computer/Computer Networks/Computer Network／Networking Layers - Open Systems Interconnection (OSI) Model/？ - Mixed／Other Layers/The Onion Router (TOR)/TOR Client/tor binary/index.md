# Installation and Setup

> [!expand]- homebrew
> <strong>1. install tor</strong>
> <span style="white-space: pre-wrap"><code>  brew install tor</code></span>
>
> <strong>2. verify tor been downloaded</strong>
> <span style="white-space: pre-wrap"><code>  which tor</code><br><code>  tor --version</code></span>
>
> <strong>3. run tor in foreground</strong>
> <span style="white-space: pre-wrap"><code>  tor</code></span>
>
> run tor in background
> <span style="white-space: pre-wrap"><code>  brew services start tor</code><br><code>  brew services stop tor</code></span>
>
> <strong>4. follow [[Proxy Chains|proxychains]] setup to use tor</strong>
