<strong>Address Port Restricted Cone NAT</strong>

- a type of [[Network Address Translation (NAT)|Dynamic NAT]]
- acts in precisely the same way as an [[Address Restricted Cone NAT]] but applies restrictions to the source port also
- where a <strong>Restricted Cone NAT</strong> will accept connections from any source port a <strong>Port Restricted Cone NAT</strong> restricts this further by only accepting connections from the IP address <strong>and </strong>Port it sent the outbound request to
