###### NetBird

```excerpt
- unlike traditional VPNs that route all traffic through a central server, NetBird establishes encrypted tunnels between devices and handles complex configurations automatically
- uses the [[WireGuard]] protocol to create secure, direct peer-to-peer connections between devices
```
