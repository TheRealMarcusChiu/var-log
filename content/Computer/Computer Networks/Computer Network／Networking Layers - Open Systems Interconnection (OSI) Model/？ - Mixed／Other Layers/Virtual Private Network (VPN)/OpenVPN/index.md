###### OpenVPN
````excerpt
- is an open-source software application that implements [[Virtual Private Network (VPN)|VPN]] techniques for creating secure point-to-point or site-to-site connections in routed or bridged configurations and remote access facilities
- uses a custom security protocol that utilizes [[Secure Sockets Layer (SSL) - Transport Layer Security (TLS)|SSL/TLS]] for key exchange
- capable of traversing [[Network Address Translation (NAT)|NATs]] and [[Firewalls - Bastion Host|Firewalls]]
````
^excerpt
