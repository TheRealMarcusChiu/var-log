###### Automatic Private IP Addressing (APIPA)

```excerpt
- with APIPA, [[DHCP Client|DHCP clients]] can automatically self-configure an [[IPv4 - Address|IP address]] and [[IPv4 - Subnet & Supernet|subnet mask]] when a [[DHCP Server|DHCP server]] isn't available
- lives in [[3 - Network Layer|Layer 3]] of the [[Computer Network／Networking Layers - Open Systems Interconnection (OSI) Model|OSI model]]
```

^excerpt

# IP Address Range

- assigns a class B IP address from 169.254. 0.0 to 169.254. 255.255
