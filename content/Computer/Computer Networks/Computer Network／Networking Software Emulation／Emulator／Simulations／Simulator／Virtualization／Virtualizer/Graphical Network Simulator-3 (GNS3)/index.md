###### Graphical Network Simulator-3 (GNS3)

- <https://gns3.com/>
- is a [[Computer Network／Networking Software Emulation／Emulator／Simulations／Simulator／Virtualization／Virtualizer|network software emulator]] first released in 2008
- it allows the combination of virtual and real devices, used to simulate complex networks
- it uses Dynamips emulation software to simulate Cisco IOS
