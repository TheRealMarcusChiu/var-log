the process of building up the forwarding tables to allow correct forwarding

[[Routing Protocols|routing protocols]] can be either static routing or [[Dynamic Routing Protocols|dynamic routing]]. dynamic routing protocols are distributed algorithms instead of centralized to allow for scalability

we don't care about static routing

2 types of <strong>Dynamic Routing Protocols</strong>:

- [[Interior Gateway Protocols (IGP) or Intra-Domain Routing Protocols (IDRP)|intradomain routing protocols]] - used <strong>within</strong> an [[Autonomous System (AS) - Domain|Autonomous System]]
- [[Exterior Gateway Protocols (EGPs) or Inter-Domain Routing Protocols (IDRP)|interdomain routing protocols]] - used <strong>between</strong> Autonomous Systems
