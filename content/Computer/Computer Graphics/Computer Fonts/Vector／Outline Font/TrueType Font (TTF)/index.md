###### TrueType Font (TTF)
````excerpt
- is a type of [[Vector／Outline Font|Vector/Outline Font]]
- developed by Apple in the late 1980s as a competitor to Adobe's Type 1 font used in PostScript
- it has become the most common format for fonts on the classic Mac OS, [[Macintosh OSs (Mac OS X) - macOS|macOS]], and Microsoft [[Windows]] operating systems
- extended by [[OpenType Font (OTF)]]
````
