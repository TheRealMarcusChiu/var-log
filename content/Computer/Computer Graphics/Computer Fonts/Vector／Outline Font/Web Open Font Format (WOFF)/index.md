###### Web Open Font Format (WOFF)

```excerpt
- is a font format for use on web pages
- WOFF files are [[OpenType Font (OTF)|OpenType]] or [[TrueType Font (TTF)|TrueType]] fonts, with format-specific compression applied and additional [[Extensible Markup Language (XML)|XML]] metadata added
```

^excerpt
