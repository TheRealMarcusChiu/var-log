###### OpenType Font (OTF)
````excerpt
- is a type of [[Vector／Outline Font|Vector/Outline Font]]
- is an extension of the [[TrueType Font (TTF)|TrueType font]] format
- allows better support for international character sets and provides broader multiplatform support
- allows both TrueType or Adobe Type 1 outline fonts to be packaged as a TrueType font
- fully supports [[Unicode]]
````
^excerpt
