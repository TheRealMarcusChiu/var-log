###### Apache Guacamole
````excerpt
- is a <strong>clientless remote desktop gateway</strong>
- supports standard protocols like [[Virtual Network Computing  (VNC)|VNC]], RDP, and [[Secure Shell (SSH)]]
- no plugins or client software are required, thanks to [[HyperText Markup Language (HTML)|HTML5]], once Guacamole is installed on a server, all you need to access your desktops is a [[Web (HTTP) Clients／Browsers|web browser]]
````
^excerpt
