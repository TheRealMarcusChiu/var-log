###### High Efficiency Image Format (HEIF) - High Efficiency Image Container (HEIC)

```excerpt
- is a [[Raster Graphics (Bitmap Image)|raster graphics]] file format for storing images that's designed to be more efficient than [[Joint Photographic Experts Group (JPG／JPEG／JFIF)|JPEG]]
- are smaller in size, but retain quality and have more features than JPEGs
```
