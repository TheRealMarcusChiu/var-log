###### WebP

```excerpt
- is a modern [[Raster Graphics (Bitmap Image)|raster-typed]] image format that provides superior lossless and lossy compression for images on the web
- using WebP, webmasters and web developers can create smaller, richer images that make the web faster
- WebP lossless images are 26% smaller in size compared to [[Portable Network Graphics (PNG)|PNGs]]
```

^excerpt
