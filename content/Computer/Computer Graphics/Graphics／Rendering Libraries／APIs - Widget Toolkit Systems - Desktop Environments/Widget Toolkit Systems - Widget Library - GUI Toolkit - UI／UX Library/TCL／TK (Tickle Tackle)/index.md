###### TCL/TK (Tickle Tackle)
````excerpt
- <strong>Tcl (Tool Command Language)</strong> is a cross-platform general-purpose multi-paradigm system dynamic programming language
- <strong>Tk</strong> is a cross-platform [[Widget Toolkit Systems - Widget Library - GUI Toolkit - UI／UX Library|widget toolkit]] used for developing desktop applications to a higher level than conventional approaches (can produce native applications that run unchanged across [[Windows]], [[Macintosh OSs (Mac OS X) - macOS|Mac OS X]], [[Linux]], and more
````
^excerpt
