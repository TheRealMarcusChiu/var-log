###### Qt Toolkit
````excerpt
- pronounced "cute" or as an initialism
- is cross-platform application development framework for creating graphical user interfaces as well as cross-platform applications that run on various software and hardware platforms such as Linux, Windows, macOS, Android or embedded systems with little or no change in the underlying codebase while still being a native application with native capabilities and speed
- most GUI programs created with Qt have a native-looking interface, in which case Qt is classified as a [[Widget Toolkit Systems - Widget Library - GUI Toolkit - UI／UX Library|widget toolkit]]
````
^excerpt

# Qt - Usages

> [!expand-ui]- Desktop UI
> Graphical user-interfaces and [[Desktop Environments (DE)|desktop environments]] that utilize [[Qt Toolkit|Qt]]/[[Qt Modeling Language (QML)|QML]] as widget toolkit:
> - [[KDE Plasma - KDE Desktop (K Desktop Environment)|KDE Plasma]] - a libre desktop environment for various computing devices
> - DDE ([Deepin](https://en.wikipedia.org/wiki/Deepin) Desktop Environment) of Linux Deepin
> - UKUI ([Ubuntu Kylin](https://en.wikipedia.org/wiki/Ubuntu_Kylin) User Interface)
> - CutefishDE, a desktop environment built with Qt Quick and essential KDE frameworks
> - [LXQt](https://en.wikipedia.org/wiki/LXQt) (Lightweight X11 Desktop Environment)
> - [Lumina](https://en.wikipedia.org/wiki/Lumina_(desktop_environment)), a desktop environment designed for [BSD](https://en.wikipedia.org/wiki/BSD)-based TrueOS
> - Lomiri (formerly [Unity8](https://en.wikipedia.org/wiki/Unity8)), a convergent desktop environment started by Canonical, maintained by Ubports
> - Maui Shell, a DE that also features their own app suite
> - [Unity 2D](https://en.wikipedia.org/wiki/Unity_(user_interface)#Unity_2D), a desktop shell written in Qt and Qml
> - [Trinity DE](https://en.wikipedia.org/wiki/Trinity_Desktop_Environment), a continuously developed fork based on KDE3
> - [SDDM](https://en.wikipedia.org/wiki/Simple_Desktop_Display_Manager), a [display manager](https://en.wikipedia.org/wiki/X_display_manager) written in [[Qt Modeling Language (QML)|QML]] that is compatible with [[(X - X11 - X Window System) Core Protocol|X11]] and [Wayland](https://en.wikipedia.org/wiki/Wayland_(protocol))
