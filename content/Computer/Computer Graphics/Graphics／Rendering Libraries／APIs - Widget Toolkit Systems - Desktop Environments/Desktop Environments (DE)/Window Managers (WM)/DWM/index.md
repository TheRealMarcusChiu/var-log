###### DWM

```excerpt
- is a minimalist dynamic [[Window Managers (WM)|window manager]] for the [[(X - X11 - X Window System) Core Protocol|X Window System]]
- developed by Suckless
- influenced the development of several other X window managers, including xmonad and awesome
- it is externally similar to wmii, but internally much simpler
- is written purely in [[C]] for performance
```

^excerpt
