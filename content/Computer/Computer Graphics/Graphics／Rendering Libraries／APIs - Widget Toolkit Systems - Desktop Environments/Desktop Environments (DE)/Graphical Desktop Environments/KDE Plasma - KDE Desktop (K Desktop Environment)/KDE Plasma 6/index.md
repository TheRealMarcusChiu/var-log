###### KDE Plasma 6
````excerpt
- is the sixth generation of the [[KDE Plasma - KDE Desktop (K Desktop Environment)|KDE Plasma]] desktop environment
- it is the successor of [[KDE Plasma 5]]
- was first released on 28 February 2024
````
^excerpt

# Plasma 6 - New Features

Compared to the latest Plasma 5 release, the sixth generation brings the following new features:
- The panel floats by default. Panels can be configured using auto-hide mode and be resized, including being turned into a dock.
- Plasma Search allows customizing the ordering of search results, supports timezones conversion and matches the text in both the current system language and English.
- Support for the [Astronomical and Umm al-Qura Calendars](https://en.wikipedia.org/wiki/Islamic_calendar#Calculated_Islamic_calendars) in the calendar applet.
- Support for use of both password and fingerprint authentication at the same time.
- Support for sound themes.
