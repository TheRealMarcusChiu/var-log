###### CDE Desktop (Common Desktop Environment)
````excerpt
- is a [[Desktop Environments (DE)|desktop environment]] for [[UNIX]] and OpenVMS, based on the Motif widget toolkit. It was part of the UNIX 98 Workstation Product Standard, and was for a long time the "classic" Unix desktop associated with commercial Unix workstations ([[GNOME Desktop (GNU Network Object Model Environment)|GNOME]] and [[KDE Plasma - KDE Desktop (K Desktop Environment)|KDE]] took over)
````
