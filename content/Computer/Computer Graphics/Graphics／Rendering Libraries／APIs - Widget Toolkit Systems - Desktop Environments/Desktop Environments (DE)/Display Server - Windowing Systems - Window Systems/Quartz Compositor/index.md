###### Quartz Compositor
- is the [display/window server](https://en.wikipedia.org/wiki/Display_server) (and at the same time the [compositing window manager](https://en.wikipedia.org/wiki/Compositing_window_manager)) for [macOS](https://en.wikipedia.org/wiki/MacOS)
- is responsible for presenting and maintaining [rasterized](https://en.wikipedia.org/wiki/Raster_Graphics), rendered graphics from the rest of the [Core Graphics](https://en.wikipedia.org/wiki/Core_Graphics) framework and other renderers in the [Quartz technologies](https://en.wikipedia.org/wiki/Quartz_(graphics_layer)) family
- is programmed in [[OpenGL Family|OpenGL]]
