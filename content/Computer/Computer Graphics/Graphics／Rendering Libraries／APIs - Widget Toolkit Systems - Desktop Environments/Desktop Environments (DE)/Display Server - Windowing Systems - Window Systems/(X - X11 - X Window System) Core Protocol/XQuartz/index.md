###### XQuartz
````excerpt
- since [[MacX]] is no longer officially supported, the XQuartz project is an open-source effort to develop a version of the [[(X - X11 - X Window System) Core Protocol|X Window System]] that runs on OS X
- together with supporting libraries and applications, it forms the X11.app that Apple shipped with OS X versions 10.5 through 10.7
````
^excerpt
