###### Anti-Grain Geometry (AGG)
````excerpt
- is a 2D [[Graphics／Rendering Libraries／APIs|rendering graphics library]]
- is written in [[C++]]
- it features anti-aliasing and sub-pixel resolution
- it is not a graphics library, per se, but rather a framework to build a graphics library upon
````
^excerpt

# AGG - Usage
- The Haiku operating system uses AGG in its [[Display Server - Windowing Systems - Window Systems|windowing system]]
- [[matplotlib|Matplotlib]] uses AGG as its canonical renderer for interactive user interfaces
