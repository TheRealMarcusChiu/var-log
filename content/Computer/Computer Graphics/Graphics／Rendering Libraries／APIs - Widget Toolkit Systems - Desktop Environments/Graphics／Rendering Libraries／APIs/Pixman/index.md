###### Pixman

```excerpt
- is a low-level [[Graphics／Rendering Libraries／APIs|rendering library]] for pixel manipulation, providing features such as image compositing and trapezoid rasterization
- important users of pixman are the cairo graphics library and the [[X.Org Server|X server]]
- is implemented as a library in the [[C|C programming language]]
- it runs on many platforms, including [[Linux]], [[Berkeley Software Distribution (BSD)|BSD]] Derivatives, [[Macintosh OSs (Mac OS X) - macOS|MacOS X]], and [[Windows]]
```

^excerpt
