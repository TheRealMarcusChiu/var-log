###### Metal
````excerpt
- is a low-level, low-overhead hardware-accelerated compute shader3D [[Graphics／Rendering Libraries／APIs|graphic API]]
- is developed by Apple
- Metal exist just because Apple didn't want to support Vulkan (however there is [[MoltenVK]] a runtime for macOS and iOS that offers an almost complete subset of the [[Vulkan|Vulkan API]] implemented using Metal)
````
^excerpt
