###### Vulkan
````excerpt
- is replacing [[OpenGL Family|OpenGL]]
- offers higher performance and more balanced [[Central Processing Unit (CPU) - Central／Main Processor - Microprocessor (uP) - Microcontroller (uC) - Microcontroller Unit (MCU)|CPU]]/[[Graphics Processing Unit (GPU) - Graphics Card|GPU]] usage than: [[OpenGL Family|OpenGL]] and [Direct3D 11](https://en.wikipedia.org/wiki/Direct3D_11)
- comparable to [[Direct3D 12]] and [[Metal]]
- managed by [[Khronos Group]]
- is derived from and built upon components of AMD's Mantle API, which was donated by AMD to Khronos with the intent of giving Khronos a foundation on which to begin developing a low-level API that they could standardize across the industry
````
^excerpt
