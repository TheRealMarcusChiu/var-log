###### Cairo

```excerpt
- is an open-source [[Graphics／Rendering Libraries／APIs|graphics library]] that provides a vector graphics-based, device-independent API for software developers
- it provides primitives for two-dimensional drawing across a number of different backends
- uses hardware acceleration when available
- written in [[C|C programming language]]
```
