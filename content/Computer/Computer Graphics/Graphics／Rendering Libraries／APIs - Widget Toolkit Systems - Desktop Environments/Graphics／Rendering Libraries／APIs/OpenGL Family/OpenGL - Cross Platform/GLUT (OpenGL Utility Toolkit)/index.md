<strong><strong>GLUT (</strong>OpenGL Utility Toolkit)</strong> is a library of utilities for [[OpenGL (Open Graphics Library)|OpenGL]] programs, which primarily perform system-level I/O with the host [[Operating Systems (OS)|operating system]]. Functions performed include window definition, window control, and monitoring of keyboard and mouse input
