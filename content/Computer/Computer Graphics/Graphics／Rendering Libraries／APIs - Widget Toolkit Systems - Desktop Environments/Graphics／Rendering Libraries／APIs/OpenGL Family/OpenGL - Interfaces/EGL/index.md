###### EGL

```excerpt
- is an interface between [Khronos](https://en.wikipedia.org/wiki/Khronos_Group)[rendering APIs](https://en.wikipedia.org/wiki/Rendering_API) (such as [[OpenGL Family|OpenGL]], [OpenGL ES](https://en.wikipedia.org/wiki/OpenGL_ES) or [OpenVG](https://en.wikipedia.org/wiki/OpenVG)) and the underlying native platform [[Display Server - Windowing Systems - Window Systems|windowing system]]
```
