###### OpenGL for Embedded Systems
- is a subset of the [[OpenGL (Open Graphics Library)|OpenGL]] computer graphics rendering API for rendering 2D and 3D computer graphics such as those used by video games, typically hardware-accelerated using a [[Graphics Processing Unit (GPU) - Graphics Card|GPU]]
