###### Hierarchical Navigable Small World (HNSW) Algorithm

```excerpt
- is a graph-based algorithm that performs [[Approximate Nearest Neighbor (ANN) Search|approximate nearest neighbor (ANN)]] searches in [[Vector Database|vector databases]]
```

^excerpt

# HNSW - Use Cases

Databases that use HNSW as search index include:

- [[Apache - Lucene|Apache Lucene]] Vector Search
- Chroma
- [[Facebook AI Similarity Search (FAISS)|FAISS]]
- Qdrant
- Vespa
- Vearch Gamma
- Weaviate
- pgvector
- [[MariaDB]]

Several of these use the hnswlib library provided by the original authors
