A [[Sorting Algorithms|sorting algorithm]] is said to be <strong>stable</strong> if two objects with equal keys appear in the same order in the <strong>sorted</strong> output as they appear in the unsorted input. Whereas a sorting algorithm is said to be <strong>unstable</strong> if there are two or more objects with equal keys which don't appear in same order before and after sorting
