- <strong>Serialization/Marshalling/Encoding</strong> is the process of transforming the memory representation of an object to a [[Data Language & Library (Message／Transfer／Interchange／Storage／Format／Serializer／Deserializer)|data format]] suitable for the storage and transmission
- <strong>Deserialization/Unmarshalling/Decoding/Parsing</strong> refers to the process of transforming a representation of an object that is used for storage or transmission to a representation of the object that is executable
