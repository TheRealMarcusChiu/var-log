- <strong>model</strong> - is representation-agnostic
- <strong>diagram</strong> - is a representation of a model

the same <strong>model</strong> can be represented by many different <strong>diagrams</strong>
# Model Types
- Entity Relation Model (ERM) - used for describing entities and their relationships (ERD)
- Software System Model - used to describe a software system (C4 Model)
