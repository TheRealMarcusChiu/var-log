<strong>Magnet</strong> is a [[Uniform Resource Locator (URL) - Uniform Resource Identifier (URI) - Uniform Resource Name (URN) - Path／Query／Anchor|URI scheme]] that defines the format of <strong>magnet links</strong>, a de facto standard for identifying files ([[Uniform Resource Locator (URL) - Uniform Resource Identifier (URI) - Uniform Resource Name (URN) - Path／Query／Anchor|URN]]) by their content, via [[Cryptographic Hash Function - Message Digest Function|cryptographic hash value]] rather than by their location
