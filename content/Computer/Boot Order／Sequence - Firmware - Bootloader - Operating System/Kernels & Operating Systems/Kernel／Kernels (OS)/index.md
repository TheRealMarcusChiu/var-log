###### Kernel/Kernels (OS)

```excerpt
- is the core part of the [[Operating Systems (OS)|operating system]]
- it is the first program of operating system that is loaded into the [[Primary／Main／Physical Computer Memory／Storage|main memory]] (i.e. [[Random-Access Memory (RAM)|RAM]]) at start and remains there until shut down
- it acts as a bridge between application software and [[Computer Hardware|hardware]] of the system
```
