###### Netfilter
````excerpt
- is a framework provided by the [[Linux Kernel|Linux kernel]] that allows various networking-related operations to be implemented in the form of customized handlers
- offers various functions and operations for [packet filtering](https://en.wikipedia.org/wiki/Packet_filter), [[Network Address Translation (NAT)|network address translation]], and [port translation](https://en.wikipedia.org/wiki/Port_translation), which provide the functionality required for directing packets through a network and prohibiting packets from reaching sensitive locations within a network
````
^excerpt
