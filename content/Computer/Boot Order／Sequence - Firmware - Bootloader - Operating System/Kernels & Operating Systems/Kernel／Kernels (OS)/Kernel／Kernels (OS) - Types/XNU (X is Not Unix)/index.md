###### XNU (X is Not Unix)

- is the computer operating system (OS) [[Kernel／Kernels (OS)|kernel]] developed at [Apple Inc.](https://en.wikipedia.org/wiki/Apple_Inc.) in December 1996
- used in the [[Macintosh OSs (Mac OS X) - macOS|Mac OS X (now macOS)]] operating system; and is the basis for the [Apple TV Software](https://en.wikipedia.org/wiki/Apple_TV_Software), [iOS](https://en.wikipedia.org/wiki/IOS), [iPadOS](https://en.wikipedia.org/wiki/IPadOS), [watchOS](https://en.wikipedia.org/wiki/WatchOS), and [tvOS](https://en.wikipedia.org/wiki/TvOS)
- released as [free and open-source software](https://en.wikipedia.org/wiki/Free_and_open-source_software) as part of the [Darwin](https://en.wikipedia.org/wiki/Darwin_\(operating_system\)) OS
