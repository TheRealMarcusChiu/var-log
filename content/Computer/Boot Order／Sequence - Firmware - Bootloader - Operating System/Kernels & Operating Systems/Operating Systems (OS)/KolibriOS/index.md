###### KolibriOS
````excerpt
- is a tiny yet incredibly powerful and fast [[Operating Systems (OS)|operating system]]
- it requires only a few megabyte disk space and 12MB of RAM to run
- features a rich set of applications that include word processor, image viewer, graphical editor, web browser and over 30 exciting games
- has full FAT12/16/32 support is implemented, as well as read-only support for NTFS, exFAT, ISO9660 and Ext2/3/4
````
^excerpt
