###### lemonbar

```excerpt
- is a stripped down clone of [[dzen]]
- written on top of [[Xcb|XCB]]
```

^excerpt

# Example

It works the same way dzen does, by piping text to it:

```
echo 'shblah' | bar -g 120x20+10+80 -p
```
