<strong>Oh My Zsh</strong> ([https://ohmyz.sh/](https://ohmyz.sh/)) is a delightful, open source, community-driven framework for managing your [[zsh (Z SHell)|Zsh]] configuration
### Plugins

> [!expand]- zsh-autocompletion
> ```bash
> git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
> vim ~/.zshrc
> ```
>
> add zsh-autosuggestions to plugins
> ```
> plugins=(git zsh-syntax-highlighting zsh-autosuggestions)
> ```
>
> restart terminal
