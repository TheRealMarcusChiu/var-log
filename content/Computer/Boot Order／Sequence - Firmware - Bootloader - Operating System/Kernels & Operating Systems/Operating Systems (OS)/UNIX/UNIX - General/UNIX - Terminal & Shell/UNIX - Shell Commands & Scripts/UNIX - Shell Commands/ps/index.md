# 2 Most Common

```
ps aux

USER PID %CPU %MEM VSZ RSS TTY STAT START TIME COMMAND
```

That corresponds to a <strong>[[Berkeley Software Distribution (BSD)|BSD]]-style process listing</strong>

```
ps -ef

UID  PID  PPID C STIME TTY  TIME CMD
```

Corresponds to a <strong>[[UNIX - Portable Operating System Interface (POSIX)|POSIX]]-style process listing</strong>
