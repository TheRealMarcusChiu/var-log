# Commands

###### Shortcut keys Options:

- <strong>Ctrl-a + c:</strong> It create a new windows.
- <strong>Ctrl-a + w:</strong> It display the list of all the windows currently opened.
- <strong>Ctrl-a + A:</strong> It rename the current windows. The name will appear when you will list the list of windows opened with Ctrl-a + w.
- <strong>Ctrl-a + n:</strong> It go to the next windows.
- <strong>Ctrl-a + p:</strong> It go to the previous windows.
- <strong>Ctrl-a + Ctrl-a:</strong> It back to the last windows used.
- <strong>Ctrl-a + k:</strong> It close the current windows (kill).
- <strong>Ctrl-a + S:</strong> It split the current windows horizontally. To switch between the windows, do Ctrl-a + Tab.
- <strong>Ctrl-a + |:</strong> It split the current windows vertically.
- <strong>Ctrl-a + X: </strong>Close active Split window
- <strong>Ctrl-a + Q: </strong>Close all Split windows
- <strong>Ctrl-a + d:</strong> It detach a screen session without stopping it.
- <strong>Ctrl-a + r:</strong> It reattach a detached screen session.
- <strong>Ctrl-a + \[:</strong> It start the copy mode.
- <strong>Ctrl-a + ]:</strong> It paste the copied text.
