```excerpt


- [fish shell](https://fishshell.com/) - easy to use, beginner-friendly shell
- [starship](https://starship.rs/) - a great prompt that requires no setup
- [z](https://github.com/rupa/z) - quickly jump around your filesystem
- [fzf](https://github.com/junegunn/fzf) - general-purpose fuzzy search
- [fd](https://github.com/sharkdp/fd) - like <code>find</code> but better
- [ripgrep](https://github.com/BurntSushi/ripgrep) - like <code>grep</code> but better
- [htop](https://hisham.hm/htop/) and [glances](https://nicolargo.github.io/glances/) - system monitoring tools
- [virtualenv](https://pypi.org/project/virtualenv/) and [virtualfish](https://github.com/justinmayer/virtualfish) - Python virtual environment management
- [pyenv](https://github.com/pyenv/pyenv), [nodenv](https://github.com/nodenv/nodenv), and [rbenv](https://github.com/rbenv/rbenv) - manage different versions of Python, Node, and Ruby
- [pipx](https://github.com/pipxproject/pipx) - install Python packages in isolated environments
- [ctop](https://github.com/bcicen/ctop) and [lazydocker](https://github.com/jesseduffield/lazydocker) - monitoring tools for Docker
- [asciinema](https://asciinema.org/) - record your terminal sessions (and let viewers copy code from those recordings)
- [colordiff](https://www.colordiff.org/) and [diff-so-fancy](https://github.com/so-fancy/diff-so-fancy) - like <code>diff</code> but with colors
- tree - for presenting the content of a folder
- [bat](https://github.com/sharkdp/bat) - like <code>cat</code> but better
- [[httpie]] - like <code>curl</code> but better
- [tldr](https://tldr.sh/) - simplified "man pages"
- [exa](https://the.exa.website/) - like <code>ls</code> but better
- [litecli](https://litecli.com/) and [pgcli](https://www.pgcli.com/) - like <code>sqlite3</code> and <code>psql</code> but better
- [mas](https://github.com/mas-cli/mas) - CLI interface for App Store
- [ncdu](https://dev.yorhel.nl/ncdu) - disk usage analyzer (replacement fo du)
```

^excerpt
