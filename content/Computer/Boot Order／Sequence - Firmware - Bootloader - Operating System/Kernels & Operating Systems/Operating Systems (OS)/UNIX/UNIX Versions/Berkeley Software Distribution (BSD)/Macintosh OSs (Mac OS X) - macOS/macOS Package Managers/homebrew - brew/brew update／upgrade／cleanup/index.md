- update homebrew itself and the package "lists": <code>brew update</code>
- upgrade all software installed with brew: <code>brew upgrade</code>
- upgrade all software installed with cask: <code>brew cask upgrade</code>
- remove old versions of installed formulae: <code>brew cleanup</code>
