<strong>MacPorts</strong> Project is an open-source community initiative to design an easy-to-use system for compiling, installing, and upgrading either command-line, [[(X - X11 - X Window System) Core Protocol|X11]] or Aqua based open-source software on the [[Macintosh OSs (Mac OS X) - macOS|Mac operating system]].
