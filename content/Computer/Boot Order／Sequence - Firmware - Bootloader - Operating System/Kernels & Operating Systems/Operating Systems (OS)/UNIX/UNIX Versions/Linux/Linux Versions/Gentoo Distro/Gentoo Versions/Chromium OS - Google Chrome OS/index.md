<strong>Google Chrome OS</strong> is to <strong>Chromium OS</strong> what Google Chrome browser is to Chromium

- <strong>Chromium OS</strong> is the open source project, used primarily by developers, with code that is available for anyone to checkout, modify, and build
- <strong>Google Chrome OS</strong> is the Google product that OEMs ship on Chromebooks for general consumer use
