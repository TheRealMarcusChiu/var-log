###### Talos Linux

```excerpt
- is actually a ground-up rewrite of the userspace, from PID 1
- runs the [[Linux Kernel|Linux kernel]], but everything downstream of that is custom code, written in [[Go-lang|Go]]
- is Linux designed for [[Kubernetes]] – secure, immutable, and minimal
- is the "Kubernetes [[Operating Systems (OS)|operating system]]"
- all system management is done via an API. No [[Secure Shell (SSH)|SSH]], [[UNIX - Shell|shell]] or console
```

^excerpt
