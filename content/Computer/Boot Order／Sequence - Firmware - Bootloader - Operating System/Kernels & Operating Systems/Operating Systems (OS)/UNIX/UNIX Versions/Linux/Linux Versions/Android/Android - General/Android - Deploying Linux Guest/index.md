This application is open source software for quick and easy installation of the [[Operating Systems (OS)|operating system (OS)]] GNU/[[Linux]] on your Android device.
1. download “Linux Deploy” from Play Store
   
   [https://play.google.com/store/apps/details?id=ru.meefik.linuxdeploy&hl=en](https://play.google.com/store/apps/details?id=ru.meefik.linuxdeploy&hl=en)
2. toggle the necessary settings
3. install, deploy and start
4. install “VNC viewer” from Play Store
   
   [https://play.google.com/store/apps/details?id=com.realvnc.viewer.android&hl=en](https://play.google.com/store/apps/details?id=com.realvnc.viewer.android&hl=en)
