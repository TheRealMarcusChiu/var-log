1. Search the market (Google Play) in your phone for:
   1. "SSHDroid" app
   2. “Servers Ultimate” app
2. open terminal on laptop

   <span style="white-space: pre-wrap"><code>ssh root@192.168.1.70</code></span>
3. Under Ubuntu, browse any folder and click File > Connect to Server:
4. Enter the address generated by your phone (sftp://root@192.168.1.70, for example)
5. Enter password
