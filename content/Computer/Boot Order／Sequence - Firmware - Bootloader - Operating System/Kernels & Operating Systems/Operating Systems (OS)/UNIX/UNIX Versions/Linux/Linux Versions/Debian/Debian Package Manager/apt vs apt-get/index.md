- <code><font style="color: rgb(122,134,154);">apt-get</font></code> was initially released in 1998 as part of Debian 2.0 (Hamm)
- <code><font style="color: rgb(122,134,154);">apt</font></code> command-line tool, a more user-friendly interface to the Advanced Package Tool (APT), was introduced with APT 1.0, which was released on April 1, 2014, and subsequently included in Debian 8 (Jessie)
