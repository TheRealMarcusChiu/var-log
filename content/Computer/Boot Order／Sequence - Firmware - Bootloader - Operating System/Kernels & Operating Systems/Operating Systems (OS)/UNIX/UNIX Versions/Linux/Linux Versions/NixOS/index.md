###### NixOS
````excerpt
- a Linux distribution built on top of the [[Nix|Nix package manager]]
- can be described as "OS as Code"
- the entire system is defined in a <code><font style="color: rgb(122,134,154);">configuration.nix</font></code> file, making it declarative and reproducible.
- NixOS upgrades are atomic and transactional, meaning a system rollback is always possible if an update fails
````
^excerpt
