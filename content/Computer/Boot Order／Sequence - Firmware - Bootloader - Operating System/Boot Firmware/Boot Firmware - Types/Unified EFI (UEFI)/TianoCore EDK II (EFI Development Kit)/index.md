###### TianoCore EDK II (EFI Development Kit)
````excerpt
- <em>TianoCore</em> is a community that supports an open-source implementation of [[Unified EFI (UEFI)|UEFI]]
- <em>TianoCore EDK II</em> is the reference implementation of UEFI by Intel started in 2004
- <em>TianoCore EDK II</em> is written in [[C]]
````
