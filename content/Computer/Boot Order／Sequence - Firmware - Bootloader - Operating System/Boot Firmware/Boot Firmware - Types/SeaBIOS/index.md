###### SeaBIOS
````excerpt
- is an open-source implementation of an x86 BIOS, serving as a freely available firmware for x86 systems
- it supports standard [[Basic Input／Output System (BIOS)|BIOS]] features and calling interfaces that are implemented by a typical proprietary x86 BIOS
````
