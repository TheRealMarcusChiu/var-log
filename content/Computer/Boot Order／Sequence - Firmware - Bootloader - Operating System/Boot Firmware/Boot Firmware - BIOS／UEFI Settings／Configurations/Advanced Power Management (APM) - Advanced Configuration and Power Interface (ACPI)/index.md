# Advanced Power Management (APM)

- is an old technology that aimed to provide power management capabilities to the computer and the user
- is centered on the [[Basic Input／Output System (BIOS)|BIOS]]
- is already considered as obsolete

# Advanced Configuration and Power Interface (ACPI)

- is a much newer technology that replaces APM among other things
- is centered on the [[Operating Systems (OS)|operating system]]
- is the preferred technology
