###### wakeonlan (Terminal Command)
````excerpt
- uses UDP packets to send out WoL packets
````
^excerpt

# Installation

> [!expand-ui]- Debian
> ```
> sudo apt install wakeonlan
> ```

> [!expand-ui]- macOS
> ```
> brew install wakeonlan
> ```
# Use Case
```
wakeonlan <mac address>
```
