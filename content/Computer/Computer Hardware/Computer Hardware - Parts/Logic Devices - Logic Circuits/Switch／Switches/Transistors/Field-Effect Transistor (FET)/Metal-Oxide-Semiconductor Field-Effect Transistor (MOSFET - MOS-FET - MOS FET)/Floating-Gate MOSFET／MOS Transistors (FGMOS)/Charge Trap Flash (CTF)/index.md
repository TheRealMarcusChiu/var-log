###### Charge Trap Flash (CTF)
- is a type of [[Floating-Gate MOSFET／MOS Transistors (FGMOS)|floating-gate MOSFET]] memory technology, but differs from the conventional floating-gate technology in that it uses a silicon nitride film to store electrons rather than the doped polycrystalline silicon typical of a floating-gate structure
- used in creating non-volatile [[NOR Flash Memory|NOR flash memory]] and [[NAND Flash Memory|NAND flash memory]]

# CTF - Manufacture Cosy Savings

Using silicon nitride instead of polycrystalline silicon allows memory manufacturers to reduce manufacturing costs in five ways:
1. Fewer process steps are required to form a charge storage node
2. Smaller process geometries can be used (therefore reducing chip size and cost)
3. Multiple bits can be stored on a single flash memory cell
4. Improved reliability
5. Higher yield since the charge trap is less susceptible to point defects in the tunnel oxide layer
