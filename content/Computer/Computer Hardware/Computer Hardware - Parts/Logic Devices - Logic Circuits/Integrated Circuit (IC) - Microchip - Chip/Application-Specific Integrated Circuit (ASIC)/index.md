###### Application-Specific Integrated Circuit (ASIC)

```excerpt
- is an [[Integrated Circuit (IC) - Microchip - Chip|integrated circuit]] chip customized for a particular use, rather than intended for general-purpose use (i.e. [[Central Processing Unit (CPU) - Central／Main Processor - Microprocessor (uP) - Microcontroller (uC) - Microcontroller Unit (MCU)|CPU]])
```

^excerpt

# ASIC - Examples

- a chip designed to run in a digital voice recorder or a high-efficiency [[Cryptocurrency - Crypto Currency|Bitcoin]] miner is an ASIC
