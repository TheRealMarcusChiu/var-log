###### MicroBlaze

```excerpt
- is a [[Soft Microprocessor - Softcore Microprocessor - Soft Processor|soft microprocessor]] core designed for Xilinx [[Field-Programmable Gate Arrays (FPGA)|field-programmable gate arrays (FPGA)]]
- is implemented entirely in the general-purpose memory and logic fabric of Xilinx FPGAs
- was introduced in 2002
```
