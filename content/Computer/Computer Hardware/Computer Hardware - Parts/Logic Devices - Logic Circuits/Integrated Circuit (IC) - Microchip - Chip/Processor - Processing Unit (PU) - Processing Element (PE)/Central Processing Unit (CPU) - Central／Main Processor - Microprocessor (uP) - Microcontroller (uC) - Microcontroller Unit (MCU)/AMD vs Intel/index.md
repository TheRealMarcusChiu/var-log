Intel and AMD share a cross-licensing agreement under which:
- Intel lets AMD make [[x86 x64 (x86-16, x86-32, x86-64)|x86 CPUs]]
- AMD lets Intel use its [[x86 x64 (x86-16, x86-32, x86-64)|x86_64 instruction set]]

Both Intel and AMD processors have the same [[x86 x64 (x86-16, x86-32, x86-64)|x86 architecture]] (having the same [[Instruction Set Architecture (ISA)|instruction set]]), but how the two companies implement that architecture (their [[Computer Organization - Microarchitecture (µarch - uarch)|computer organizations]]) is usually very different. The same programs run correctly on both because the architecture is the same, but they may run at different speeds because the organizations are different
