###### Intel Quick Sync
````excerpt
- is a hardware-based video encoding and decoding technology built into most modern Intel processors with integrated graphics, designed to accelerate video processing tasks like streaming, editing, and converting files
- by offloading these duties from the main CPU to a dedicated hardware core on the processor die, Quick Sync improves performance, reduces power consumption, and allows for faster video creation, playback, and editing
````
