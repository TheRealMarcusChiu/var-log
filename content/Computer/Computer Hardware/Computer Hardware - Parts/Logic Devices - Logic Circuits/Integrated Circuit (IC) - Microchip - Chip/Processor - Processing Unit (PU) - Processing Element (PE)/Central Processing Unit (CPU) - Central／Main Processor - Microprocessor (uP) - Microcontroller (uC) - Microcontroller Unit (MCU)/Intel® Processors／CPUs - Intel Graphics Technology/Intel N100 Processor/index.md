###### Intel N100 Processor

```excerpt
- is a low-power, mobile processor designed for entry-level computing and media consumption
- features 4 cores and 4 threads, a base clock speed of 100 MHz that can boost up to 3.4 GHz, and an integrated UHD Graphics 730
- is built on the Alder Lake-N architecture using the Intel 7 process and has a TDP of 6 W
```
