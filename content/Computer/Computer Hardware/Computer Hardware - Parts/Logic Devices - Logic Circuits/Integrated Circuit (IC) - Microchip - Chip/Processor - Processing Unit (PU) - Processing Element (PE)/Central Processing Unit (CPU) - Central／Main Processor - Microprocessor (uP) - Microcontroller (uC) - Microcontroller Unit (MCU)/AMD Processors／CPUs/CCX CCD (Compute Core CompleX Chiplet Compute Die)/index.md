- <strong>CCX (Core CompleX)</strong> - currently 4 Cores that share some L3 cache
- <strong>CCD (Core Complex Die)</strong> - a Chiplet with CPU cores on it. (Currently 2 CCX with 4 cores each, with [[Zen 1,2,3,4,5 + Microarchitecture Processors|Zen3]] probably 1 CCX with 8 Cores)
  - A Zen 2 CCD is composed of 2 CCX, each containing 4 cores each
  - A Zen 3 CCD is composed of 1 CCX, containing 8 CPU cores
