###### Accelerated Processing Unit (APU)

```excerpt
- first coined by [[Advanced Micro Devices (AMD)|AMD]]
- is a processor that combines a [[Central Processing Unit (CPU) - Central／Main Processor - Microprocessor (uP) - Microcontroller (uC) - Microcontroller Unit (MCU)|CPU]] and [[Graphics Processing Unit (GPU) - Graphics Card|GPU]] on a single chip
```
