###### Processor - Processing Unit (PU) - Processing Element (PE)
````excerpt
- is an [[Integrated Circuit (IC) - Microchip - Chip|integrated circuit]] that performs operations on some external data source, usually [[Semiconductor／Silicon／Transistor Memory／Storage|memory]] or some other data stream
````
^excerpt

- the term is frequently referred to as [[Central Processing Unit (CPU) - Central／Main Processor - Microprocessor (uP) - Microcontroller (uC) - Microcontroller Unit (MCU)|CPU]] but typical computer systems (especially [[System on Chip (SoC) - System on a Chip|SoCs]]) combine a number of specialized "processors"
- is an implementation of [[Computer Organization - Microarchitecture (µarch - uarch)|Computer Organization - Microarchitecture]]

# Processor Unit - General Purpose
- [[Central Processing Unit (CPU) - Central／Main Processor - Microprocessor (uP) - Microcontroller (uC) - Microcontroller Unit (MCU)|CPU - central processing unit]]
	- If designed conforming to the [[Von Neumann Architecture／Model - Princeton Architecture - Stored-Program Computer|von Neumann architecture]], it contains at least a [control unit](https://en.wikipedia.org/wiki/Control_unit) (CU), [arithmetic logic unit](https://en.wikipedia.org/wiki/Arithmetic_logic_unit) (ALU), and [processor registers](https://en.wikipedia.org/wiki/Processor_register).
	- In some contexts, the ALU and registers are called the <strong>processing unit</strong>
