###### System on Chip (SoC) - System on a Chip
````excerpt
- is an [[Integrated Circuit (IC) - Microchip - Chip|integrated circuit]] that integrates most or all components of a computer or other electronic system
````
^excerpt

# SoC - Components

SoC almost always includes the following on a single substrate or microchip:
- [[Central Processing Unit (CPU) - Central／Main Processor - Microprocessor (uP) - Microcontroller (uC) - Microcontroller Unit (MCU)|CPU]]
- memory interfaces
- on-chip input/output devices
- input/output interfaces
- secondary storage interfaces
- sometimes [[Graphics Processing Unit (GPU) - Graphics Card|GPU]]
