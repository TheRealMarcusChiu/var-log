###### Inter-Integrated Circuit (I²C - I2C - IIC)

```excerpt
- is a synchronous and bi-directional serial communication protocol in [[Uni-Directional／Unidirectional - Bi-Directional／Bidirectional - Half-Duplex - Full-Duplex - Unsolicited|half-duplex]] mode
- is a synchronous, multi-controller/multi-target, packet-switched, single-ended, serial communication bus
```

^excerpt

# Specifications

- Protocol: Serial, half-duplex
- Width: 1-bit (SDA) with separate clock (SCL)
- Data signal: Open-collector or open-drain
- Number of devices: 128 devices [robot-electronics.co.uk](http://robot-electronics.co.uk)
- Designed: 1982; 41 years ago
- Bitrate: 0.1, 0.4, 1.0, 3.4, or 5.0 Mbit/s depending on the mode
