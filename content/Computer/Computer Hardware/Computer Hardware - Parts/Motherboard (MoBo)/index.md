###### Motherboard (MoBo)
````excerpt
- is the main [[Printed Circuit Breadboard／Boards (PCB)|printed circuit board (PCB)]] in a computer, and is the central communications backbone connectivity point for all components and external peripherals
- contains a chip storing a [[Boot Firmware]]
- a [[Chipsets|chipset]] is a set of chips that extends the interfaces between all of the components of a motherboard
````
^excerpt
