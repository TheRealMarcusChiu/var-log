###### Optical Copper Link (OCuLink)
````excerpt
- is a connection system for EXTERNAL [[Peripheral Component Interconnect Express (PCIe - PCI-e) Specification／Interface|PCIe Interface]] for external components like eGPUs (external GPUs) and NVMe drives
- offers up to 64 Gbps with PCIe Gen 4.0 (4 lanes x 16 GT/s)
- the interface was developed by the [[Peripheral Component Interconnect Special Interest Group (PCI-SIG)|PCI-SIG]]
````
^excerpt
