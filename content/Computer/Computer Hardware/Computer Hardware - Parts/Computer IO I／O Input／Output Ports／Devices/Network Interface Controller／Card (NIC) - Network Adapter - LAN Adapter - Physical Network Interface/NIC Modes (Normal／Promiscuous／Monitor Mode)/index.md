- <strong>normal mode </strong>- capture packets only sent to local machine (including broadcast and multicast addresses)
- <strong>promiscuous mode </strong>- capture packets on a network that you have connected to
- <strong>monitor mode </strong>- capture packets regardless of connected network. No association to AP needed (and no authentication). Because it is not connected to a network, you can't process the Ethernet info.
