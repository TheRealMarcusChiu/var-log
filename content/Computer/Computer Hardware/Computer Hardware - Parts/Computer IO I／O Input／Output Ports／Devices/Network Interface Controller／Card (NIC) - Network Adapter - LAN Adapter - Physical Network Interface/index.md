###### Network Interface Controller/Card (NIC) - Network Adapter - LAN Adapter - Physical Network Interface
````excerpt
- implements the electronic circuitry required to communicate using a specific [[1 - Physical Layer|physical layer]] and [[2 - Data Link Layer|data link layer]] standard (e.g. [[802.3 - Ethernet|Ethernet]] or [[802.11 - WiFi／Wi-Fi|Wi-Fi]]). This provides a base for a full network [protocol stack](https://en.wikipedia.org/wiki/Protocol_stack), allowing communication among computers on the same network
````
