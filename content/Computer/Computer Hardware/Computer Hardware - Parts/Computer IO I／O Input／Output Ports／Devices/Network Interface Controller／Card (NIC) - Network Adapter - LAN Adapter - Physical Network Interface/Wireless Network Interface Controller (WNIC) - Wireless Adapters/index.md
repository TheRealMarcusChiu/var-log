###### Wireless Network Interface Controller (WNIC) - Wireless Adapters
- is a subtype of [[Network Interface Controller／Card (NIC) - Network Adapter - LAN Adapter - Physical Network Interface|network interface controller (NIC)]]
- are electronic devices that allow computers to connect to the Internet and to other computers without using wires
- they send data via radio waves to [[Router|routers]] that pass it on to [[Baseband v Broadband|broadband]] [[Modem|modems]] or internal networks
