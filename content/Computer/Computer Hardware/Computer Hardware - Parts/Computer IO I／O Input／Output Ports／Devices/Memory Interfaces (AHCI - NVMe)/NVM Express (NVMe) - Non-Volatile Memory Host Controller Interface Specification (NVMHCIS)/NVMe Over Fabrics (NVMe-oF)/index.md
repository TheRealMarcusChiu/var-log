###### NVMe Over Fabrics (NVMe-oF)
````excerpt
- specifications published in 2016
- is an extension of the NVMe network protocol to [[802.3 - Ethernet|Ethernet]] and [[Fibre Channel (FC)|Fibre Channel]]
- delivers faster and more efficient connectivity between storage and servers, as well as a reduction in CPU utilization of application host servers
````
