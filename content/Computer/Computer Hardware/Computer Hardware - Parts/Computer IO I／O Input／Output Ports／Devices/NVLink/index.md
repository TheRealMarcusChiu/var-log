###### NVLink

```excerpt
- offers substantially higher bandwidth compared to [[Peripheral Component Interconnect Express (PCIe - PCI-e)|PCIe]]
- NVLink 3.0 provides up to 600 GB/s total bandwidth (bidirectional) between GPUs
	- [[Peripheral Component Interconnect Express (PCIe - PCI-e) Specification／Interface|PCIe 5.0 offers 128 GB/s (x16 lanes)]], significantly lower than NVLink
	- [[Peripheral Component Interconnect Express (PCIe - PCI-e) Specification／Interface|PCIe 4.0 offers 64 GB/s (x16 lanes)]]
```

^excerpt
