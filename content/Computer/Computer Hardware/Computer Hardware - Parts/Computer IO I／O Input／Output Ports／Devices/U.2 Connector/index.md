###### U.2 Connector
````excerpt
- similar to [[U.3 Connector]]
- is physically compatible with [[Serial Attached SCSI (SAS)|SAS]]/[[Serial ATA (SATA) - Parallel ATA (PATA)|SATA]] disks but adds new pins for the [[Peripheral Component Interconnect Express (PCIe - PCI-e) Specification／Interface|PCIe]] x4 link uses for [[NVM Express (NVMe) - Non-Volatile Memory Host Controller Interface Specification (NVMHCIS)|NVMe]] drives
- is an interface standard for connecting [[Solid State Drives (SSD)|SSD]] to a computer (it covers the physical connector, electrical characteristics, and communication protocols)
- it uses up to four PCIe lanes and two SATA lanes
````
