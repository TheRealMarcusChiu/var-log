###### Converged Network Adapter (CNA) - Converged Network Interface Controller (C-NIC)
````excerpt
- is a [[Computer IO I／O Input／Output Ports／Devices|computer input/output device]] that combines the functionality of a [[Host Controller - Host Adapter - Host Bus Adapter (HBA)|host bus adapter (HBA)]] and a [[Network Interface Controller／Card (NIC) - Network Adapter - LAN Adapter - Physical Network Interface|network interface controller (NIC)]]. In other words, it "converges" access to, respectively, a [[Storage Area Network (SAN)|storage area network]] and a general-purpose computer network
````
