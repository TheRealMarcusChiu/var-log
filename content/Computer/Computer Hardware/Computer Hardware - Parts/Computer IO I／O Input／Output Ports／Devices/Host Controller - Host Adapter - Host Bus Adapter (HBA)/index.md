###### Host Controller - Host Adapter - Host Bus Adapter (HBA)
````excerpt
- connects a computer system bus which acts as the host system to a network device and/or a [[Computer Memory／Storage Devices|storage device]]
- the terms are primarily used to refer to devices for connecting [[Small Computer Systems Interface (SCSI)|SCSI]], [[Serial Attached SCSI (SAS)|SAS]], [[NVM Express (NVMe) - Non-Volatile Memory Host Controller Interface Specification (NVMHCIS)|NVMe]], [[Fibre Channel (FC)|Fibre Channel]] and [[Serial ATA (SATA) - Parallel ATA (PATA)|SATA]] devices
- the term [[Network Interface Controller／Card (NIC) - Network Adapter - LAN Adapter - Physical Network Interface|network interface controller (NIC)]] is used purely to connect to computer networks (nothing else)
- the term [[Converged Network Adapter (CNA) - Converged Network Interface Controller (C-NIC)|converged network adapter (CNA)]] combines both NIC and HBA
- can be either:
	- integrated in the [[Motherboard (MoBo)|motherboard]]
	- separate expansion card
````
^excerpt
