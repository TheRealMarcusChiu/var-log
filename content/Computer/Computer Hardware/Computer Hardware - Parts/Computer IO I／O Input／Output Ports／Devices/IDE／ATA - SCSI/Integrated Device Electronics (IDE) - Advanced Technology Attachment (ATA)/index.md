- <strong>Integrated Device Electronics (IDE)</strong> interface is based on the IBM PC Industry Standard Architecture (ISA) 16-bit bus
- <strong>Advanced Technology Attachment (ATA) </strong>- another name for IDE accepted as standard by the American National Standards Institute (ANSI) in 1990

IDE/ATA is popular for connecting [[Secondary／Virtual Memory (Drive)|drives]]
