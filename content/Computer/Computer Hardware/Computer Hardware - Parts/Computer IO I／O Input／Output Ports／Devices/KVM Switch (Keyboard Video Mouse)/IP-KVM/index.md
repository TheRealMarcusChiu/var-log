###### IP-KVM
````excerpt
- is a [[KVM Switch (Keyboard Video Mouse)|KVM]] over [[Internet Protocol (IP)|IP]]
- it allows remote access and control of a computer or server over a network using standard protocols like TCP/IP
- essentially, it enables an administrator to manage a computer as if they were physically present at the console, accessing the keyboard, video, and mouse, even from a remote location
````
