###### Pulse Width Modulation (PWM) Computer Fans
````excerpt
- PWM fans are [[Direct Current (DC) ／ 3-pin Computer Fans|DC fans]] with an extra wire
- PWM fans are 4-pin fans, where the fourth wire sends a PWM signal to the fan motor. The PWM signal is the control input of the PWM fan
- speed of the fan is controlled by PWM signal pin
````
^excerpt
