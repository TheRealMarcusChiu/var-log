# Desktop (Green, BarraCuda)

are generally [[HDD - Storage Modes (LMR - PMR - SMR - CMR - EAMR - HAMR)|SMR]]

- Seagate BarraCuda Compute - 5400 RPM consumer level disks
- Seagate BarraCuda - regulated to laptop hard drive naming, used to be desktop but no more
- Seagate BarraCuda Pro - 7200 RPM consumer level disks
- Seagate BarraCuda Pro Compute - never existed

# Desktop Gaming (Red, FireCuda)

generally is 5 times faster than BarraCuda

# NAS (Red, Ironwolf)

are generally PMR, designed to run 24/7, intended for NAS use

- Seagate Ironwolf - 180TB per year
- Seagate Ironwolf Pro - 550TB per year - includes a subscription to a data recovery company if the drive fails under warranty
  - Seagate Ironwolf Pro NAS - same as Ironwolf Pro; just a marketing gimmick

# Enterprise / Data Center (Gold/Ultrastar, EXOS)

enterprise drive, designed to run 24/7, is LOUD

Seagate EXOS

- Seagate EXOS 7e8 4kn
  - "7" stands for iteration lineup (i.e. there were EXOS 3e, 5e, etc)
  - "e" stands for exterprise
  - "8" stands for maximum capicity of the design
  - "4kn" stands for "4k native" instead of antiquated "512e" or "512 byte emulated"
- Seagate EXOS (x16, x18, x20) refers to the platter density and drive tech
  - Seagate EXOS x16 - maximum capcity 16TB
  - Seagate EXOS x18 - maximum capcity 18TB
  - Seagate EXOS x20 - maximum capcity 20TB
