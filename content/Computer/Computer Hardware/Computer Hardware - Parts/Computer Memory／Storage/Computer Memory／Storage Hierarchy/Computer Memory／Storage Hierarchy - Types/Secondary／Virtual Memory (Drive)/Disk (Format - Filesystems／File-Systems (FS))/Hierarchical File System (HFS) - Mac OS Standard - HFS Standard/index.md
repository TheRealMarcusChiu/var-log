###### Hierarchical File System (HFS) - Mac OS Standard - HFS Standard
````excerpt
- is a proprietary file system developed by Apple Inc. for use in computer systems running Mac OS
- originally designed for use on floppy and hard disks, it can also be found on read-only media such as CD-ROMs
- its successor, HFS Plus, is also called Mac OS Extended (or HFS Extended)
````
^excerpt
