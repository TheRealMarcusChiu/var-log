###### GUID Partition Table (GPT)
````excerpt
- gradually replacing [[Master Boot Record (MBR)|MBR]]
- is a part of the [[Unified EFI (UEFI)|Unified Extensible Firmware Interface (UEFI)]] standard for the layout of the partition table on a [[Computer Memory／Storage Devices|storage medium]] (e.g. [[Hard Disk Drives (HDD) - Fixed Disk|HDD]])
````
