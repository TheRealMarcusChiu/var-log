###### EFI System Partition (ESP)

- is an [[Operating Systems (OS)|OS]]-independent [[Partition／Volume - Disk Partitioning - Partition Schema|partition]] that acts as the storage place for the EFI bootloaders, applications, and drivers to be launched by the UEFI firmware. It is mandatory for [[Unified EFI (UEFI)|UEFI]] boot
- when a computer is booted, UEFI firmware loads files stored on the ESP to start operating systems and various utilities
