###### Ceph Block Devices (CBD) - RADOS Block Devices (RBD)
````excerpt
- the block storage component of [[Ceph Storage Cluster - CephFS - Ceph File System|Ceph]]
- a software instrument that orchestrates the storage of block-based data in Ceph
- splits block-based application data into “chunks”
- RADOS stores these chunks as objects
- orchestrates the storage of those objects across the storage cluster
````
