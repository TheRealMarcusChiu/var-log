###### Apple File System (APFS)
````excerpt
- is a proprietary file system developed and deployed by Apple Inc. for macOS Sierra (10.12.4) and later, iOS 10.3, tvOS 10.2, watchOS 3.2, and all versions of iPadOS
- it aims to fix core problems of HFS+ (also called Mac OS Extended)
- is optimized for solid-state drive storage and supports encryption, snapshots, and increased data integrity, among other capabilities
````
^excerpt
