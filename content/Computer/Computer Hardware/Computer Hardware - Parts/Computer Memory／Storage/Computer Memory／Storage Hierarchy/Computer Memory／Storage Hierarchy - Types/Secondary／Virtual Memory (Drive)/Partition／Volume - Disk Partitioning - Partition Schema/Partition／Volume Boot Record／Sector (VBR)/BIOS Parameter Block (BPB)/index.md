<strong>BIOS Parameter Block (BPB)</strong> is a data structure in the [[Partition／Volume Boot Record／Sector (VBR)|Volume Boot Record (VBR)]] describing the physical layout of a [[Computer Memory／Storage Devices|storage medium]] or [[Partition／Volume - Disk Partitioning - Partition Schema|partition]]:

- on partitioned devices (e.g. [[Hard Disk Drives (HDD) - Fixed Disk|HDD]]) BPB describes the volume [[Partition／Volume - Disk Partitioning - Partition Schema|partition]]
- on unpartitioned devices (e.g. [floppy disks](https://en.wikipedia.org/wiki/Floppy_disk)) BPB describes the entire [[Computer Memory／Storage Devices|storage medium]]
