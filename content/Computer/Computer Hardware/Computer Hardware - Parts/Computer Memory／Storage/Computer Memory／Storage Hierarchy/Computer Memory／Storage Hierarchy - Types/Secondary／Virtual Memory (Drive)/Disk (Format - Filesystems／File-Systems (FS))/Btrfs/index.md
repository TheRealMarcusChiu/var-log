###### Btrfs

```excerpt
- pronounced as "butter fuss", "better FS", "butter FS", "b-tree FS", or simply by spelling it out
- is a computer storage format that combines a file system based on the [copy-on-write](https://en.wikipedia.org/wiki/Copy-on-write#In_computer_storage) (COW) principle with a [logical volume manager](https://en.wikipedia.org/wiki/Logical_volume_management) (not to be confused with Linux's [[Logical Volume Manager (LVM) ＊Linux|LVM]])
```

^excerpt
