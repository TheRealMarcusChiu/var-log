###### OpenZFS
````excerpt
- is an open-source version of the [[Zettabyte File System (ZFS)]]
````
^excerpt

# History

After the Sun acquisition by Oracle in 2010, Oracle's version of ZFS became the closed source, and the development of open-source versions proceeded independently, coordinated by OpenZFS from 2013
