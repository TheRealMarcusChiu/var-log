###### FreeNAS/TrueNAS
````excerpt
- the branding for a range of free and open-source [[Network Attached Storage (NAS)|NAS]] [[Operating Systems (OS)|operating systems]]
- based on [[FreeBSD]] and [[Linux]], using the [[OpenZFS|OpenZFS file system]]
- produced by [[iXsystems, Inc.|iXsystems]]
````
