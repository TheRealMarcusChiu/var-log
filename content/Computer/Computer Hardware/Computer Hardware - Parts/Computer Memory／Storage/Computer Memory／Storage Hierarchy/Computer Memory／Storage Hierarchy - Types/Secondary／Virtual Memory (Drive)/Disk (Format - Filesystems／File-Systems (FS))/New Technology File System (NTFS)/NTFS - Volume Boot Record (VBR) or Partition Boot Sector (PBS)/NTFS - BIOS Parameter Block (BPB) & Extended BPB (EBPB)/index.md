###### BIOS Parameter Block (BPB) & Extended BPB (EBPB)

- is a specific type of [[BIOS Parameter Block (BPB)]] used for a [[New Technology File System (NTFS)|NTFS volume]]
- exist inside an NTFS [[NTFS - Volume Boot Record (VBR) or Partition Boot Sector (PBS)|Volume Boot Record (VBR)]]
- all values are [[Big Endian - Little Endian|little endian]]

see [[NTFS - Volume Boot Record (VBR) or Partition Boot Sector (PBS)]] for BPB layout
