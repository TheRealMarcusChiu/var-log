<strong>Google File System</strong> (<strong>GFS</strong> or <strong>GoogleFS</strong>)
- is a[distributed file system](https://en.wikipedia.org/wiki/Distributed_file_system) developed by [Google](https://en.wikipedia.org/wiki/Google) to provide efficient, reliable access to data using large clusters of commodity hardware
- files/data are only extremely rarely overwritten, or shrunk; files are usually appended to or read
