<strong>Unix File System (UFS)</strong> also called <strong>Berkeley Fast File System</strong>, <strong>[[Berkeley Software Distribution (BSD)|BSD]] Fast File System</strong> or <strong>FFS</strong> is a file system supported by many[[UNIX]]-like [[Operating Systems (OS)|operating systems]]. It is a distant descendant of the original filesystem used by [Version 7 Unix](https://en.wikipedia.org/wiki/Version_7_Unix)
