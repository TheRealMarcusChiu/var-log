###### \$LOGGED\_UTILITY\_STREAM
- a type of [[NTFS - File Attributes|NTFS File Attribute]]
- layout and treatment is similar to \$DATA attribute
- every EFS encrypted file will have this attribute to store the file encryption key used during encryption/decryption
