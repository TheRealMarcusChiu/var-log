###### \$VOLUME\_NAME

- a type of [[NTFS - File Attributes|NTFS File Attribute]]
- contains the [[Partition／Volume - Disk Partitioning - Partition Schema|volume]] name
- allows NTFS volumes to have long names with special characters in different languages
- prior to \$VOLUME\_NAME the only alternative was the "OEM label" located in the boot sector
