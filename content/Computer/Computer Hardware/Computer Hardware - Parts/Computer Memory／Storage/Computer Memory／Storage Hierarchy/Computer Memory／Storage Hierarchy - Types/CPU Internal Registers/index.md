###### CPU Internal Registers
````excerpt
- is the fastest in the [[Computer Memory／Storage Hierarchy|computer memory hierarchy]]
- volatile/non-persistent memory
- is used by the [[Central Processing Unit (CPU) - Central／Main Processor - Microprocessor (uP) - Microcontroller (uC) - Microcontroller Unit (MCU)|CPU]] to reduce the average cost (time or energy) to access data from the [[CPU Cache (L0／L1, L2, L3)|CPU Cache (L0/L1, L2, L3)]]
````
