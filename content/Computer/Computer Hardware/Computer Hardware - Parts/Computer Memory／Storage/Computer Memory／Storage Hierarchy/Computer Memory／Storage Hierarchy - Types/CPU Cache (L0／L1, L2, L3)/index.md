###### CPU Cache

```excerpt
- is the second fastest in the [[Computer Memory／Storage Hierarchy|computer memory hierarchy]]
- volatile/non-persistent memory
- is used by the [[Central Processing Unit (CPU) - Central／Main Processor - Microprocessor (uP) - Microcontroller (uC) - Microcontroller Unit (MCU)|CPU]] to reduce the average cost (time or energy) to access data from the [[Primary／Main／Physical Computer Memory／Storage|primary/main/physical memory]]
- typically uses [[Static RAM (SRAM)]]
```
