###### Temporary Filesystem (TMPFS)
````excerpt
- is a temporary file storage paradigm implemented in many [[UNIX|UNIX-like]] operating systems
- it is intended to appear as a mounted [[Disk (Format - Filesystems／File-Systems (FS))|file system]], but data is stored in [[Random-Access Memory (RAM)|volatile memory]] instead of a [[Computer Memory／Storage Devices|persistent storage device]]
````
^excerpt
