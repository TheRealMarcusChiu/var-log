- <strong>[[Single Inline Memory Module (SIMM)]]</strong> - transfers 32 bits in 1 clock cycle
- <strong>[[Dual Inline Memory Module (DIMM)]]</strong> - transfers 64 bits in 1 clock cycle
- <strong>[[Small Outline DIMM (SO-DIMM／SODIMM)|Small Outline DIMM (SO-DIMM/SODIMM)]]</strong> - found in laptops and other small form factor computers
- <strong>[[Rambus Inline Memory Module (RIMM)]]</strong> - uses Rambus Technology instead of traditional memory architecture
