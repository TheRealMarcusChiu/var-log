###### Dual Inline Memory Module (DIMM)

- commonly called a RAM stick
- is a module that contains one or several [[Random-Access Memory (RAM)|RAM]] chips on a small circuit board with pins that connect to a [[Motherboard (MoBo)|motherboard]]
