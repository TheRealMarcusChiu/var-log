###### Static RAM (SRAM)

```excerpt
- a type of [[Random-Access Memory (RAM)|RAM]] that is volatile
- typically uses 6 [[Transistors|transistors]] per memory cell (storing 1 bit) - some variants use 1/3/4/8/10 transistors
	- working on the same principle as an SR latch built out of two NOR gates (or two NAND gates)
- does not have to be periodically refreshed (hence static)
- more expensive than [[Dynamic RAM (DRAM)|DRAM]] because it requires MORE transistors
- faster than DRAM because TODO
- typically used in [[CPU Cache (L0／L1, L2, L3)|CPU Cache]] and [[CPU Internal Registers]]
```
