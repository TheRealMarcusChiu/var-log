###### Unbuffered Dual Inline Memory Module (UDIMM)

```excerpt
- they're called "unbuffered" because they don't have a buffer (a small amount of memory) between the memory module and the computer's memory controller. This means that the address and control signals can directly reach the [[Dynamic RAM (DRAM)|DRAM]] chip without going through a buffer and without any timing adjustment
```

^excerpt
