###### Dynamic RAM (DRAM)
````excerpt
- a type of [[Random-Access Memory (RAM)|RAM]] volatile
- uses 1 [[Transistors|transistor]] and 1 capacitor per memory cell (storing 1 bit)
- uses capacitors that need to be recharged (hence dynamic)
- operates asynchronously with the system clock
- cheaper than [[Static RAM (SRAM)|SRAM]] because it requires FEWER transistors
- slower than SRAM because TODO
- typically used as [[Primary／Main／Physical Computer Memory／Storage|primary memory]]
````
^excerpt
