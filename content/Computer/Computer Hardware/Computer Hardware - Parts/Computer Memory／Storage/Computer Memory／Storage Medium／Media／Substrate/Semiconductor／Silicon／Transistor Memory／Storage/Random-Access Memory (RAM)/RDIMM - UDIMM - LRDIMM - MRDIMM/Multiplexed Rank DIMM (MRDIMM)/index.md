###### Multiplexed Rank DIMM (MRDIMM)

```excerpt
- is a new type of server memory that boosts performance by increasing memory capacity and bandwidth compared to traditional DDR5 DIMMs
- are a newer DDR5 technology that use multiplexed buffers to double the data transfer rate, offering higher bandwidth and lower latency than [[Load-Reduced Dual Memory Module (LRDIMM ／ LR-DIMM)|LRDIMMs]]
```
