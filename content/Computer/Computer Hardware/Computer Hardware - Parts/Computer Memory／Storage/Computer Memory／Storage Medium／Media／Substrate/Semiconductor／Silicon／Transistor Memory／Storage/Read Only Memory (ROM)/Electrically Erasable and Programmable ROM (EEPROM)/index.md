###### Electrically Erasable and Programmable ROM (EEPROM)
````excerpt
- used in a [[Nonvolatile BIOS Memory (Firmware & Settings) - Boot Chip - CMOS RAM - Real-Time Clock (RTC)|BIOS Chip]] to store [[Boot Firmware|BIOS Firmware]] which enables firmware updates to be performed electronically
````
