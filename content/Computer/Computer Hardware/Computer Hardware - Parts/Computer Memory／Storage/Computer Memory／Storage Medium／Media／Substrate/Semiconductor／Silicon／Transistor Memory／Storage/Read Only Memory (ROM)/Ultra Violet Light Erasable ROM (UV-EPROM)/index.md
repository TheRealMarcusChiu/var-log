###### Ultra Violet Light Erasable ROM (UV-EPROM)
````excerpt
- used in old [[Nonvolatile BIOS Memory (Firmware & Settings) - Boot Chip - CMOS RAM - Real-Time Clock (RTC)|BIOS Chips]] to store [[Boot Firmware|BIOS Firmware]] (not settings/configurations) which enables firmware updates to be performed via ultra-violet light
- superseded by [[Electrically Erasable and Programmable ROM (EEPROM)|EEPROM]]
````
^excerpt
