[[Flash Memory]] vs [[Random-Access Memory (RAM)]]

- RAM performs calculations on the data retrieved from storage
- Flash is used primarily for storage
  <br>
- Flash memory can't replace DRAM and SRAM because the speed at which the DRAM/SRAM can access data and also their ability to address at byte level can't be matched by Flash
- DRAM and FLASH both use silicon transistors in contrast to magnetic HDD
