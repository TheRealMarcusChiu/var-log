hub is a command-line wrapper for git that makes you better at GitHub

# Installation

> [!expand]- homebrew <strong>1. install git using home-brew</strong> <span style="white-space: pre-wrap"><code>  brew install hub</code></span>
>
> this would download hub into /usr/local/Cellar directory
>
> <strong>2. verify if hub was installed</strong> <span style="white-space: pre-wrap"><code>  which hub</code><br><code>  hub —-version</code></span>
