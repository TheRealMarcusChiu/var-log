# Installation

> [!expand]- homebrew
> <strong>1. install svn using home-brew</strong>
> <span style="white-space: pre-wrap"><code>  brew install svn</code></span>
>
> this would download svn into /usr/local/Cellar directory
>
> <strong>2. verify if svn was installed</strong>
> <span style="white-space: pre-wrap"><code>  which svn</code><br><code>  svn --version</code></span>
