###### Metal as a Service (MaaS)
````excerpt
- is an open-source software solution that automates bare metal server provisioning and management, enabling a data center to function like a cloud
- it essentially allows for the provisioning, management, and release of physical servers in a similar way to virtual machines in a cloud environment
````
