###### Remote Method Invocation (RMI)
- is an architectural framework similar to [[Remote Procedure Call (RPC) - architectural|RPC (architecture)]], but takes it a step further by making it object-oriented and providing the capability to keep references to remote objects and invoke their methods
- in other words, RMI is an object-oriented RPC
