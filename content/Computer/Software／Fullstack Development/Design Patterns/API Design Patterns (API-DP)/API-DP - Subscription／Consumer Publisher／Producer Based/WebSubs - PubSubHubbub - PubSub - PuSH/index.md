<strong>WebSubs - PubSubHubbub - PubSub - PuSH</strong> is an architectural framework for subscribers and providers
# WebSubs - Components

WebSub is built upon an ecosystem of:
- <strong>Servers/Publishers</strong> (Medium, WordPress, etc.)
- <strong>Mediators/Hubs</strong> (Superfeedr, Switchboard, etc.)
- <strong>Clients/Subscribers</strong> (Feedly, Flipboard, etc.)
