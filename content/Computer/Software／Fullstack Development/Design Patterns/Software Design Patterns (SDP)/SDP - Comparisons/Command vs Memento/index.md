[[Command Pattern|Command]] and [[Memento Pattern|Memento]]
- <em>Command</em> and <em>Memento</em> can be used together when implementing “undo”. In this case, commands are responsible for performing various operations over a target object, while mementos save the state of that object just before a command gets executed.
