###### Model-View-Presenter (MVP)
````excerpt
- is a derivation of the [[Model-View-Controller (MVC)]] pattern
- the <em>presenter</em> assumes the functionality of the "middle-man" between View and Model
- all presentation logic is pushed to the presenter
````
^excerpt
