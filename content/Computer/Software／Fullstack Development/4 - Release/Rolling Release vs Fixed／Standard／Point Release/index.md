- <strong>Rolling Release</strong> - also known as rolling update or [[Continuous Delivery (CD)|continuous delivery]], is a concept in software development of frequently delivering updates to applications
- <strong>Fixed/Standard/Point Release</strong> - uses software versions that must be reinstalled over the previous version
