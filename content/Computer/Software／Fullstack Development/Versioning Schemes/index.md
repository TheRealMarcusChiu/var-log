###### Versioning Scheme
- modern computer software is often tracked using two different software versioning schemes:
	- <strong>internal version</strong> number that may be incremented many times in a single day, such as a revision control number
	- <strong>release version</strong> that typically changes far less often, such as semantic versioning or a project code name
