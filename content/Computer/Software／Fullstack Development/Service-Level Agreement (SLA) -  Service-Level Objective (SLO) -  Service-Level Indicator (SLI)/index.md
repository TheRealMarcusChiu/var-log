### SLA/SLO/SLI in Site-Reliability Engineer (SRE) Domain
- SLA - It’s more about contracts
- SLO - It says how often SLI could fail. Like SLI should be true for 99.9% of the time
- SLI - It’s some parameters that should be measured and which should be kept in some range
