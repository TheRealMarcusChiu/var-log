###### You Ain't Gonna Need It (YAGNI)
````excerpt
- avoid adding unnecessary features or functionality to our code
- don't write code for things you don't currently need or anticipate needing in the future (however one must make the code [[Open-Closed Principle (OCP)|open for extension]])
````
^excerpt
