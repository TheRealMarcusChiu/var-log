````excerpt
<strong>[[Continuous Delivery (CD)]] </strong>- an extension of CI that begins with CI and ends with the <font style="color: rgb(128,128,0);">release</font> of verified artifacts (e.g. packages, jars/wars, [[Virtual Machines (VM) - Hypervisors ／ Virtualizer ／ Virtual Machine Monitor (VMM) (Type-1 Bare Metal - Type-2 Hosted)|virtual-machine]] images or [[Container／Image - Container Runtimes - Container Engine - Container-Based Virtualization - Container Scheduler／Orchestrator／Orchestration／Orchestrating Engine (COE)|container]] images) – these artifacts are published onto a central download server (e.g. [[Binary／Artifact Repository Manager (BRM - ARM)|binary repository managers]])
````
