The Apache JMeter™ application is open source software, a 100% pure Java application designed to load test functional behavior and measure performance.
# Installation

> [!expand]- Click here to expand...
> <strong>1. install jmeter using homebrew</strong>
> <span style="white-space: pre-wrap"><code>  brew install jmeter</code></span>
>
> this would download jmeter into /usr/local/Cellar directory
>
> <strong>2. verify jmeter</strong>
> <span style="white-space: pre-wrap"><code>  which jmeter</code><br><code>  jmeter -—version</code></span>
