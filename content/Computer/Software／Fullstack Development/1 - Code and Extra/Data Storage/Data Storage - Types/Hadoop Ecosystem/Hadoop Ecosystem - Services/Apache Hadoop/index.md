###### Apache Hadoop

```excerpt
- is the core part of the [[Hadoop Ecosystem - Services|Hadoop Ecosystem]]
- is a collection of open-source software utilities that facilitate using a network of many computers to solve problems involving massive amounts of data and computation
- provides a software framework for distributed storage and processing of big data using the [[MapReduce (MR)]] programming model
```
