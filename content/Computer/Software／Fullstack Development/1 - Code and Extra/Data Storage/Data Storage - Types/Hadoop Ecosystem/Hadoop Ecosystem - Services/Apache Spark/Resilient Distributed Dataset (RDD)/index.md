###### Resilient Distributed Dataset (RDD)
````excerpt
- is a fundamental data structure in [[Apache Spark]], representing a read-only, distributed collection of data that can be partitioned across multiple machines in a cluster
- it enables parallel computation and fault tolerance by allowing for lineage reconstruction, meaning the system can reconstruct the data if a node fails
````
