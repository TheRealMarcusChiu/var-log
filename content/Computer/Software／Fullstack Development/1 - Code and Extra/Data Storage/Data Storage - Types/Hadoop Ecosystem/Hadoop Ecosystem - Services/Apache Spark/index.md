###### Apache Spark

```excerpt
- ran on top of [[Apache Hadoop]]
- an open-source distributed general-purpose cluster-computing framework
- provides an interface for programming entire clusters with implicit data parallelism and fault tolerance
- includes components such as Spark Core, Spark Programming, [[Spark SQL & Dataframes|Spark SQL]], [[Spark Streaming]], [[Spark MLlib|MLlib]], and [[Spark GraphX|GraphX]]
```

^excerpt
