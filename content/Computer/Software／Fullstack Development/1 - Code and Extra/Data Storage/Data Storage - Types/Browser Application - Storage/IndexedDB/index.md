###### IndexedDB

```excerpt
- is the successor to both [[Web Storage (sessionStorage - localStorage)]] and [[Web SQL Database|Web SQL]], designed to replace them as the “one true” browser database
- it exposes an asynchronous API that supposedly avoids blocking the DOM, but as we’ll see below, it doesn’t necessarily live up to the hype
- browser support is extremely spotty, with only Chrome and Firefox having fully usable implementations
```

^excerpt
