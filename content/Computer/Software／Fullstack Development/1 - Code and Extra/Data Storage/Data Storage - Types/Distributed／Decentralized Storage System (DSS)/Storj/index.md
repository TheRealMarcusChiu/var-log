###### Storj
````excerpt
- is an end-to-end [[Encryption|encrypted]], [[Distributed - Algorithms／Computing／Programming／Systems|distributed]] object storage, where you only have access to the data
- a type of [[Storage Virtualization|storage virtualization]] powered by blockchain payments
````
^excerpt
