###### PetaSAN

```excerpt
- is an open-source [[Software Defined Storage (SDS)|software-defined storage (SDS)]] solution
- is a [[Storage Area Network (SAN)]], specifically, it's a scale-out SAN, meaning it's designed to handle large amounts of data and can grow by adding more nodes (servers)
- uses [[Ceph Storage Cluster - CephFS - Ceph File System|Ceph]] as its storage engine and [[Internet Small Computer Systems Interface (iSCSI)|iSCSI]] to connect to servers
```

^excerpt
