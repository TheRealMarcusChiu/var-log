###### Schema Registry
````excerpt
- provides a centralized repository for [[Schema Definition／Description Language & Libraries (SDL)|schemas]] and metadata
- allowing services to flexibly interact and exchange data with each other without the challenge of managing and sharing schemas between them
- has support for multiple underlying schema representations ([[Apache Avro|Avro]], [[JavaScript Object Notation (JSON)|JSON]], etc.)
````
