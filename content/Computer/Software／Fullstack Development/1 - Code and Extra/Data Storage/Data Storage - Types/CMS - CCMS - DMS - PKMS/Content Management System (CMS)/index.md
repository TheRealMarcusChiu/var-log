###### Content Management System (CMS)
````excerpt
- is computer software used to manage the creation and modification of digital content (content management)
- is typically used for enterprise content management (ECM) and web content management (WCM)
- typically supports multiple users in a collaborative environment by integrating document management, digital asset management, and record retention
````
