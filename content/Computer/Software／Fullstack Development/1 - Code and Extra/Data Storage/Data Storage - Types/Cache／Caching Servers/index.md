###### Cache/Caching Servers
````excerpt
- is a dedicated network server or service acting as a server that saves [[HyperText Markup Language (HTML)|Web pages]] or other Internet content locally. By placing previously requested information in temporary storage, or cache, a cache server both speeds up access to data and reduces demand on an enterprise's bandwidth
- not the same as [[CPU Cache (L0／L1, L2, L3)|CPU Cache (L0/L1, L2, L3)]]
````
