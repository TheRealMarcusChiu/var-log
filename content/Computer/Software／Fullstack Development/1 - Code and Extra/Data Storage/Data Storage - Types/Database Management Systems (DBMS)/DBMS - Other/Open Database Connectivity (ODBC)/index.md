<strong>Open Database Connectivity (ODBC</strong><strong>)</strong>

- is a standard [[Application Programming Interface (API) - Software Development Kit (SDK)|application programming interface (API)]] for accessing [[Database Management Systems (DBMS)|database management systems (DBMS)]]
- is independent of [[Database Management Systems (DBMS)|database systems]] and [[Operating Systems (OS)|operating systems]]
