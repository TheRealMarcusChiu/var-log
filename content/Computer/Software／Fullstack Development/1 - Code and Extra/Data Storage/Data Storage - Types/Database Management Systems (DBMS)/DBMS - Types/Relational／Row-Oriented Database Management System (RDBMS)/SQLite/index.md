###### SQLite (<https://www.sqlite.org/>)

```excerpt
- is an embedded [[Relational／Row-Oriented Database Management System (RDBMS)|relational database management system]] contained in a [[C|C library]]
- in contrast to many other [[Database Management Systems (DBMS)|database management systems]], SQLite is not a client–server database engine. Rather, it is embedded into the end program
```
