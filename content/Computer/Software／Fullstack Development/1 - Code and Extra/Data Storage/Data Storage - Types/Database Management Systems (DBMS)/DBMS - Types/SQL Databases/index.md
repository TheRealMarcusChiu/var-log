###### SQL Databases
````excerpt
- organize data into tables with rows (records) and columns (fields), but the way data is stored on disk can be either row-oriented or column-oriented
	- row-oriented databases store data record by record, making them efficient for transactional queries where you often need to read or modify entire rows
	- [[Column-Oriented／Store Database Management System (CDBMS)|column-oriented databases]] store data by column, which can make analytical queries and aggregations faster
````
