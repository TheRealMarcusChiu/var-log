###### Graphite (Database)
````excerpt
- is a free open-source software tool that monitors and graphs numeric time-series data such as the performance of computer systems
- was developed by Orbitz Worldwide, Inc
- released as open-source software in 2008
- collects, stores, and displays time-series data in real time
````
