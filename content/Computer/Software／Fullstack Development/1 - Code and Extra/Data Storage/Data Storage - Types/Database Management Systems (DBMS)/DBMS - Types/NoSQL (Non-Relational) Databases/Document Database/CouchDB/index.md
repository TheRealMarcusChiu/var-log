<strong>CouchDB</strong>

- is a [[Document Database|document-oriented]] [[NoSQL (Non-Relational) Databases|NoSQL database]] architecture
- implemented in the concurrency-oriented language [[Erlang]]
- uses [[JavaScript Object Notation (JSON)]] to store data
- [[JavaScript]] as its query language using MapReduce
- [[Hypertext／Hyper Text Transfer Protocol (HTTP) - 1／1.1／2|HTTP]] for an [[API Design Patterns (API-DP)|API]]
