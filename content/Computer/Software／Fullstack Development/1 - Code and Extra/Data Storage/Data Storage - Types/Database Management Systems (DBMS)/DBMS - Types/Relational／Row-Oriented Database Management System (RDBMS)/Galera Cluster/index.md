###### Galera Cluster
````excerpt
- is a database clustering solution for [[MariaDB]] and [[MySQL]] that allows for a multi-primary, synchronous replication architecture
- it enables multiple database servers to form a cluster where data is consistently replicated across all nodes, providing high availability and data consistency
````
