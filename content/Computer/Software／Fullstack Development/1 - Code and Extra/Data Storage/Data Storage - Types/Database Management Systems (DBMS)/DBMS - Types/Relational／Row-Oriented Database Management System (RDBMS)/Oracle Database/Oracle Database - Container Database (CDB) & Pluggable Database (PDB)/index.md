In Oracle 12c, a new architecture or design of the databases was introduced, called “container databases” or “multitenant architecture”.
- <strong>Container Database (CDB)</strong> - is an Oracle database that functions as a “multitenant container database”
- <strong>Pluggable Database (PDB)</strong> - is a collection of schemas and objects that act like a “regular” database to applications and IDEs
