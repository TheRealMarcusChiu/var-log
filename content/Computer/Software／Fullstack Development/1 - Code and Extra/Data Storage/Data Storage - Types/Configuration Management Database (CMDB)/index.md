###### Configuration Management Database (CMDB)

```excerpt
- is a centralized system that stores information about an organization's IT assets, such as hardware, software, and their relationships
- it works by creating a logical model of the IT infrastructure, with each component, or configuration item (CI), recorded along with its attributes and connections to other CIs
- this provides IT teams with visibility to support processes like change management, incident response, and planning by showing how different components are connected and depend on one another
```

^excerpt
