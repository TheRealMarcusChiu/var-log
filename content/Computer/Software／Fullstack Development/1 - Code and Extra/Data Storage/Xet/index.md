###### Xet

```excerpt
- is a storage backend built specifically for large-scale AI and machine learning workflows
- Acquired by Hugging Face in 2024
- is replacing Git LFS as the primary storage system on the Hugging Face Hub
- its key feature is chunk-level deduplication, which significantly accelerates the transfer of large files like models and datasets
```

^excerpt
