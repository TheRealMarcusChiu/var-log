Solr is highly reliable, scalable and fault tolerant, providing distributed indexing, replication and load-balanced querying, automated failover and recovery, centralized configuration and more.

# Installation

> [!expand]- homebrew
>
> # Install
>
> <strong>1. install solr via homebrew</strong> <span style="white-space: pre-wrap"><code>  brew install solr</code></span>
>
> <strong>2. verify solr was installed</strong> <span style="white-space: pre-wrap"><code>  which solr</code></span>
>
> # Start Stop Restart
>
> <strong>1. To have launchd start solr</strong> <span style="white-space: pre-wrap"><code>  brew services start solr</code></span>
>
> Or, if you don't want/need a background service you can just run: <span style="white-space: pre-wrap"><code>  solr start</code></span>
>
> go to <http://localhost:8983/solr/#/>
>
> <strong>2. stop or restart</strong> <span style="white-space: pre-wrap"><code>  brew services stop solr</code><br><code>  brew services restart solr</code></span>
