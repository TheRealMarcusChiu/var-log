###### RabbitMQ
````excerpt
- an open source message broker software (sometimes called message-oriented middleware) that implements the [[Advanced Message Queueing Protocol (AMQP)|Advanced Message Queuing Protocol (AMQP)]].
- written in the [[Erlang]] [[Computer／Programming Languages|programming language]]
- built on the Open Telecom Platform framework for clustering and failover
````
