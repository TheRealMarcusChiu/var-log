###### Remote Authentication Dial-In User Service (RADIUS)
````excerpt
- is a networking protocol, operating on port 1812 that provides centralized Authentication, Authorization, and Accounting ([AAA](https://en.wikipedia.org/wiki/AAA_protocol) or Triple A) management for users who connect and use a network service.
````
^excerpt
