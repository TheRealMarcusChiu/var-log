###### TL;DR

- [[Light-Weight／Lightwieght Directory Access Protocol (LDAP)|LDAP]] is a protocol
- [[Active Directory (AD) - AD Directory Services (ADDS)|Active Directory]] is a server

The relationship between [[Light-Weight／Lightwieght Directory Access Protocol (LDAP)|LDAP]] and [[Active Directory (AD) - AD Directory Services (ADDS)|AD]] is much like the relationship between [[Hypertext／Hyper Text Transfer Protocol (HTTP) - 1／1.1／2|HTTP]] and [[Apache HTTP Server (Apache)|Apache]]:

- HTTP is a web protocol
- Apache is a web server that uses the HTTP protocol
- LDAP is a directory services protocol
- Active Directory is a directory server that uses the LDAP protocol
