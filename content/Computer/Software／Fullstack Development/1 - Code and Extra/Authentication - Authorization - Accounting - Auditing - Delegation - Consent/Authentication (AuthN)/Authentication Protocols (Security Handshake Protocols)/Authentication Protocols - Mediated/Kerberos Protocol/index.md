- is a [[Authentication Protocols - Mediated|mediated authentication protocol]] built from the [[Needham-Schroeder Protocol]] that provides centralized [[Symmetric／Single／Secret Key Function|symmetric-key]] third-party [[Authentication (AuthN)|authentication]] in a distributed network
