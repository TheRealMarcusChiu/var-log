###### Universal 2nd Factor (U2F)

- is an open standard that strengthens and simplifies [[2-Factor Authentication (2FA)|two-factor authentication (2FA)]] using specialized [[Universal Serial Bus (USB) 2.0／3.0／3.2／4.0 Gen 1／2 (Specification／Interface)|Universal Serial Bus (USB)]] or near-field communication (NFC) devices based on similar security technology found in smart cards
