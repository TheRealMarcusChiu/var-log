###### Reflection Attack

- is a method of attacking a challenge-response [[Authentication (AuthN)|authentication]] system that uses the same [[Authentication Protocols (Security Handshake Protocols)|authentication protocol]] in both directions. That is, the same challenge-response protocol is used by each side to authenticate the other side
