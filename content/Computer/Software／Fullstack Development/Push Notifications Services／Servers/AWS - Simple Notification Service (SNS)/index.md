###### Simple Notification Service (SNS)

```excerpt
- is a notification service provided as part of [[Amazon Web Services (AWS)|Amazon Web Services]] since 2010
- it provides a service for sending messages
- acts as a single message bus that can message to a variety of devices and platforms
```
