###### Nix

```excerpt
- used for [[Software／Server／System Configuration Management (SCM)|system configuration and automation]]
- a declarative [[UNIX - Universal Package Managers|universal package manager]]
- utilizes the [[Nix Language]]
- isolates packages and their dependencies, allowing multiple versions of the same software to coexist without conflicts
- officially used in [[NixOS]]
```

^excerpt
