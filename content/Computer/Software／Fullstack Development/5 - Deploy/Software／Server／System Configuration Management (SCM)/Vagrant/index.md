###### Vagrant

```excerpt
- is an automation tool with a [[Computer Languages - Domain Specific Languages (DSL)|DSL]] that is used to automate the creation of VMs and VM environments
- software product for building and maintaining portable virtual software development environments; e.g., for [[VirtualBox]], [[Kernel-Based Virtual Machine (KVM) Hypervisor|KVM]], [[Microsoft Hyper-V|Hyper-V]], [[Docker|Docker containers]], [[VMware]], [[Parallels]], and AWS
```

^excerpt
