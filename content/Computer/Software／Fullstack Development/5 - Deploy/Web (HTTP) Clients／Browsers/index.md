# Web Browser - History

- [[KHTML (Browser Engine)]] is a discontinued browser engine developed by the KDE project for their [[Konqueror|Konqueror browser]].
- It served as the foundation for [[WebKit (Browser Engine)]], which is now the engine used by [[Safari (web browser)|Safari]] and other browsers.
- KHTML was forked into WebKit in 2001, which in turn is the basis for Blink, used by Chrome, Chromium, Microsoft Edge, and other browsers
