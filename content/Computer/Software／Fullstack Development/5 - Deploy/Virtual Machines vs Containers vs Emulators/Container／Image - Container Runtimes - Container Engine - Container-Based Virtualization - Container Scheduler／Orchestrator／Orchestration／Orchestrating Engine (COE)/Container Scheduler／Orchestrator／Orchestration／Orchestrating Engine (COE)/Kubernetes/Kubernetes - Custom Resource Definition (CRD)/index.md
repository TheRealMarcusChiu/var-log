###### Custom Resource Definition (CRD)
````excerpt
- a type of [[Computer Languages - Definition／Description Languages|Computer Definition Language]]
- is a [[Kubernetes]] tool that extends the API to include custom resources for specific use cases
- a CRD defines Custom Resources (CR)
- a CR is an extension of the Kubernetes API that allows you to store your own API Objects and lets the API Server handle the lifecycle of a CR. On their own, CRs simply let you store and retrieve structured data
````
^excerpt
