- [[Docker Desktop]] is a local development environment that bundles the Docker Engine and a graphical user interface (GUI)
- [[Portainer]] is a separate, web-based management and operations GUI that can manage MULTIPLE Docker (and [[Kubernetes]]) environments, locally or remotely
