- <strong>[[Dockerfile]]</strong> describes how to build [[Docker]] images (used by the docker command)
- <strong>[[Docker Compose File (compose.yaml compose.yml docker-compose.yaml docker-compose.yml)|docker-compose.yaml]]</strong> is to run Docker containers (used by the [[Docker Compose|docker-compose]] command)
