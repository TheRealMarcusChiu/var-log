###### Wine - <https://www.winehq.org/>

```excerpt
- originally an acronym for "Wine Is Not an Emulator"
- is a compatibility layer capable of running Windows applications on several POSIX-compliant operating systems, such as [[Linux]], [[Macintosh OSs (Mac OS X) - macOS|macOS]], & [[Berkeley Software Distribution (BSD)|BSD]]
- instead of simulating internal Windows logic like a virtual machine or emulator, Wine translates Windows API calls into POSIX calls on the fly, eliminating the performance and memory penalties of other methods and allowing you to cleanly integrate Windows applications into your desktop
- is also more a [[Virtual Machines (VM) - Hypervisors ／ Virtualizer ／ Virtual Machine Monitor (VMM) (Type-1 Bare Metal - Type-2 Hosted)|hypervisor/virtual machine]] (albeit at a higher ABI level) than an emulator (hence "Wine Is Not an Emulator")
```

^excerpt
