# Modes
- <strong>full-system emulation</strong> - run operating systems for any machine, on any supported architecture
- <strong>user-mode emulation</strong> - run programs for another Linux/BSD target, on any supported architecture
- <strong>virtualization</strong> - run [[Kernel-Based Virtual Machine (KVM) Hypervisor|KVM]] and [[Xen Project|Xen]]virtual machines with near-native performance
