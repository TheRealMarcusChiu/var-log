###### BrowserPod
````excerpt
- runs [[Node.js]], [[Python]], [[Ruby on Rails|Rails]], and other full-stack runtimes directly in the [[Web (HTTP) Clients／Browsers|browser]]
- it does NOT virtualizes entire OS
- developed by Leaning Technologies - the same company behind [[WebVM]] and the CheerpX engine
- unlike WebVM, which uses heavy [[x86 x64 (x86-16, x86-32, x86-64)|x86]] emulation, BrowserPod is built on a WebAssembly-native kernel
````
^excerpt
