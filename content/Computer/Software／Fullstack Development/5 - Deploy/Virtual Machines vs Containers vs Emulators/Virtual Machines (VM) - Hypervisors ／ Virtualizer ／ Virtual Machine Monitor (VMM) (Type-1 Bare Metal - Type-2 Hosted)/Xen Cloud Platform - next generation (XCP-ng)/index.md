###### Xen Cloud Platform - next generation (XCP-ng)

- is based on the [[Xen Project]]
- comes pre-configured Xen Hypervisor and the Xen API project (XAPI) working out-of-the-box
- the project was born in 2018, following the fork of Citrix XenServer (now Citrix Hypervisor)
- since January 2020, it is also part of the Linux Foundation, via the Xen Project
