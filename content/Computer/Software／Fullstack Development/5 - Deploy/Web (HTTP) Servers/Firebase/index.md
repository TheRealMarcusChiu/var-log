<strong>Firebase</strong> is Google's mobile platform that helps you quickly develop high-quality apps and grow your business.

### Installation

> [!expand]- homebrew <strong>1. Install the firebase command line tools</strong> <span style="white-space: pre-wrap"><code>  npm install -g firebase-tools</code><br><code>  brew install firebase-cli</code></span>

### Projects

> [!expand]- navigate to a directory for your site <strong>1. Navigate to a directory for your site</strong> <span style="white-space: pre-wrap"><code>a. sign in to google:         \$ firebase login</code><br><code>b. initiate your project:     \$ firebase init</code><br><code>c. add static files to your deploy directory (the default is public)</code><br><code>d. deploy your website:       \$ firebase deploy</code></span>
