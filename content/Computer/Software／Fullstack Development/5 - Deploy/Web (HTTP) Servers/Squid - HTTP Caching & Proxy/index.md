<strong>Squid</strong> is a [[Cache／Caching Servers|caching]] and forwarding [[Hypertext／Hyper Text Transfer Protocol (HTTP) - 1／1.1／2|HTTP]] web [[Proxy (Forward／Reverse Proxy Chains)|proxy]].

It has a wide variety of uses including:

- speeding up a web server by caching repeated requests
- caching web
- [[Domain Name System／Service (DNS)|DNS]]
- other computer network lookups for a group of people sharing network resources
- aiding security by filtering traffic
