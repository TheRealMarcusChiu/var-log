###### AMPPS Stack
````excerpt
- is a [[WAMP]], [[MAMP]] and LAMP stack of [[Apache HTTP Server (Apache)|Apache]], [[MySQL]], [[MongoDB]], [[PHP (PHP： Hypertext Preprocessor)|PHP]], [[Perl (Practical Extraction and Report Language)|Perl]] & [[Python]]
- AMPPS enables you to focus more on using applications rather than maintaining them
- with AMPPS you can create a website by installing any of the 426 Apps, customizing it, and then simply publishing it on the internet via a wide choice of hosting service providers
````
