###### American Standard Code for Information Interchange (ASCII)

- is the first [[Character Encoding (Character Set - Encoding Scheme)|character encoding]] scheme (i.e. mapping between: characters ↔ numbers ↔ binary-sequence)
- only need 7 bits
- later superseded by [[Unicode]]+[[Unicode Transformation Format (UTF-8)]]
