###### Robot Operating System (ROS)
````excerpt
- is a set of software libraries and tools that help you build robot applications
- isn't actually an [[Operating Systems (OS)|operating system]] but more of a middleware or a software framework that runs on top of an existing operating system
````
^excerpt
