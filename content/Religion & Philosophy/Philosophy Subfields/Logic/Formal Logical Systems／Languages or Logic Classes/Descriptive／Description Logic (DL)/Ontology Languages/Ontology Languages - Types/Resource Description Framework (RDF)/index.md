###### Resource Description Framework (RDF)
- is a type of [[Ontology Languages|Ontology Language]] used to describe resources as a set of RDF statements/triples

# RDF - Terminology
- <strong>RDF Statement/Triple</strong> - consists of three components:
	- subject - is a resource being described by the triple
	- predicate - describes the relationship between the subject and the object
	- object - is a resource that is related to the subject
