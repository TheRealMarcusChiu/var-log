###### Open Digital Rights Language (ODRL)

```excerpt
- specifies the set of RDF classes, predicates, and named entities that are used by the [ODRL Information Model](https://www.w3.org/TR/odrl-vocab/#bib-odrl-model) which defines the ODRL Core Vocabulary
```
