<strong>Web Ontology Language (OWL</strong><strong>)</strong>
- is a type of [[Ontology Languages|Ontology Language]]
- builds upon the [[RDF Schema (RDFS)|RDFS]] language
- is a family of [[Knowledge Representation & Reasoning (KRR - KR² - KR&R) - Machine Reasoning (MR)|knowledge representation]] languages for authoring [[Ontology|ontologies]]
