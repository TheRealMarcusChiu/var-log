###### Fuzzy Logic

- is a form of many-valued logic in which the truth values of variables may be any real number between 0 and 1 both inclusive. It is employed to handle the concept of partial truth, where the truth value may range between completely true and completely false. By contrast, in Boolean Logic/[[First-Order Logic - Predicate Logic - Predicate Calculus - Quantification Theory|First-Order Logic]], the truth values of variables may only be the integer values 0 or 1
- an extension of the case of [multi-valued logic](https://en.wikipedia.org/wiki/Multi-valued_logic), valuations (𝑢:𝑉<sub>𝑜</sub>→𝑊) <font style="color: rgb(128,128,128);">(𝑉<sub>𝑜</sub> - </font>[<font style="color: rgb(128,128,128);">propositional variables</font>](https://en.wikipedia.org/wiki/Propositional_variable)<font style="color: rgb(128,128,128);"> & 𝑊 - set of membership degrees) </font>can be thought of as [membership functions](https://en.wikipedia.org/wiki/Membership_function_\(mathematics\)) mapping [[First-Order Logic - Predicate Logic - Predicate Calculus - Quantification Theory|predicates]] into [[Fuzzy／Uncertain Set Theory|fuzzy sets]] (or more formally, into an ordered set of fuzzy pairs, called a fuzzy relation). With these valuations, many-valued logic can be extended to allow for fuzzy [premises](https://en.wikipedia.org/wiki/Premise) from which graded conclusions may be drawn
