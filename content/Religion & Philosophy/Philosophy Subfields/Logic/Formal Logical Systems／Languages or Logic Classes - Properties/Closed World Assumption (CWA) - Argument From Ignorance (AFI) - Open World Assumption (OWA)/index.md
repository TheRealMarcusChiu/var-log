# Closed World Assumption - Argument From Ignorance - Open World Assumption

- <strong>Closed World Assumption (CWA)</strong> is the assumption that what is not known to be true must be false
- <strong>Argument From Ignorance (AFI)</strong> is the assumption that what is not known to be false must be true
- <strong>Open World Assumption (OWA)</strong> is the opposite, it is the assumptions that (either):
  - what is not known to be true is simply unknown
  - what is not know to be false is simply unknown

# Example

consider the following statement: <em>“Juan is a citizen of the USA.”</em>

Now, what if we were to ask <em>“Is Juan a citizen of Colombia?”</em>

- under CWA, the answer is <em>no</em>
- under AFI, the answer is <em>yes</em>
- under OWA, the answer is <em>unknown</em>
